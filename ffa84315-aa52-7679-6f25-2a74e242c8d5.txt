　　“那个……”叶少阳实在不知道说什么，转头看道风，“帮帮忙行吗？”
<br/>　　道风纵身飞出去，袖子一挥，一道黑气喷射出去，只几秒钟工夫，七八个和尚全都定住了，一动不能动。
<br/>　　叶少阳从人缝中刚挤出去，就听见门外传来一个诧异的声音：“法术，你是谁！”
<br/>　　呃？
<br/>　　既然能看出是法术，想必对方也是法师。叶少阳高声叫道：“茅山盟威天师道，我是茅山掌教叶少阳！”
<br/>　　门外沉默了几秒钟，问道：“你是叶少阳？”
<br/>　　“对对，是我。”
<br/>　　“胡说！叶少阳目前正在龙虎山上，怎有空闲到我天台山来？你到底是谁！”
<br/>　　“我真是叶少阳！不信身份证拿给你看！”
<br/>　　叶少阳走出去，一直走到展厅门外，这才看见外面站着十来个和尚，中间一个矮胖的老和尚，叶少阳打量了一下，确定没见过，干脆真的把身份证摸出来，丢了过去。
<br/>　　老和尚看了身份证，还是有些怀疑，这时候道风现身从房间里走出来，老和尚看到，警惕问道：“你又是谁！”
<br/>　　“他是我师兄道风。”
<br/>　　老和尚嘴角抽搐了几下，身后那十来个虎视眈眈的和尚也都露出了震惊的表情。
<br/>　　“你若是道风，那自然道法通天了！”
<br/>　　老和尚说着，双掌一翻，结了一个佛门手印，纵身朝道风打过去。
<br/>　　以他的年龄和身材，能有如此迅捷的身手，光凭这一点，叶少阳就断定他实力不弱。不过，他的对手是道风。
<br/>　　道风抬起一只手，食指往前面轻轻一点，一股黑气弥漫开来，震碎了老和尚辛苦结印聚拢起来的灵力。
<br/>　　没等老和尚从错愕中醒悟过来，道风已经站在他面前了。
<br/>　　“你……”老和尚抬头看了他半天，“你真是道风？”
<br/>　　“真的真的，他有这个实力，一个手指就能杀了你，他何必冒充道风，跟你磨叽这么多。”
<br/>　　老和尚这才醒悟，又转头来看他，激动地说道：“你是……叶少阳？”
<br/>　　叶少阳抱着胳膊，笑着点点头。
<br/>　　老和尚双手合十，微微躬身行礼，他身后那些弟子，一个个望着他们（主要是道风），眼中流露出异常激动的神色，只是碍于师父在这里，不敢越礼，不然早就上来要签名了。
<br/>　　“两位可都是传说中的人物，今日怎么会来我天台山？”
<br/>　　“这……”叶少阳低头看着手中的太阿剑，很尴尬地嘿嘿笑了两声，“那个，大师，能不能找个地方慢慢说？”
<br/>　　方丈的禅房内，叶少阳用了十几分钟时间，把事情的来龙去脉讲了一遍，没有隐瞒，毕竟太阿剑本是人家门派的，自己来偷，还被人发现，有理由给人家个交代。
<br/>　　苦行大师（老和尚的法号）听完讲述，十分感慨，坐在蒲团上半天不出声。
<br/>　　“大师，你们天台山好像没怎么参加过法术界的活动吧，咱们没见过？”
<br/>　　苦行大师摇摇头：“极少。我天台宗与五台山有法统之争，自从五台山成为佛门第一宗派，我天台山干脆就闭门不出了，再说本门弟子凋敝，维持香火已属不易，自己也觉得为法术界做不了什么，干脆就当个旁观者吧。”
<br/>　　叶少阳愣了一下，问他跟五台山有什么过节。
<br/>　　苦行大师哈哈一笑：“都是过去的事了，不提也罢，如今五台山倒是换了新任方丈，听说还是叶掌教的挚友，老的恩怨更加不要再提了。”
<br/>　　叶少阳笑道：“大师你这消息倒是挺灵通的。”
<br/>　　“哈哈，虽然不参加集体活动，但消息还是要掌握的，老衲与南方几家佛门关系不浅，法术界有什么动静，就算不打听他们也会第一时间告知……叶掌教昨日又登极龙虎山，一人兼任道门两大宗派，亘古未有，空前绝后啊。”
<br/>　　叶少阳不好意思的挠了挠头，“大师别笑话我了，这件事，我真是忏愧的很。”
<br/>　　苦行大师望着叶少阳的样子，心中感慨，叹道：“你与老衲想象中可是完全不一样。老衲久闻你的事迹，如今又身兼二门掌教、法术界魁首，以为你是个枭雄般的人物，如今看来，却也是个赤子般坦诚的人物，难得，难得。”
<br/>　　叶少阳更加尴尬，问他是咋看出来的，老和尚表示是直觉。他是和尚，没有俗世那些城府，看人就凭直觉，其实也是一种禅意的境界。
<br/>　　“叶掌教，可否把太阿剑给老衲端详一番？”
<br/>　　叶少阳毫不犹豫把宝剑奉上。
<br/>　　苦行大师手捧宝剑，观察片刻。喃喃道：“这便是太阿剑……祖师果真没有诳我。”
<br/>　　叶少阳怔了下，问他：“大师早就知道太阿剑在佛像里？”
<br/>　　苦行大师讲起了一段过往，那还是几十年前，他刚继任方丈的时候，老方丈弥留之际将药师佛中藏有太阿剑的秘密告诉了他，说是前辈祖师传下来的，只有历代方丈知道，断不可对任何人说起。
<br/>　　然后，祖师让他立誓，任何时候都不能将太阿剑据为己有，甚至不能将其取出，而要留待有缘人。身为方丈，要尽一切能力保护药师佛，也就是保护太阿剑，至死不渝。
<br/>　　“这条密令，在天台宗流传了不知道多少辈了，没人真正见过太阿剑，甚至，没人知道这太阿剑是否真的藏在佛像中，但师尊之命，不敢忘却。十年浩劫时期，是老衲事先将佛像转移到山洞里，避过劫难。九十年代，有专家组前来鉴定木雕佛像，将这尊药师佛定为最高级，当时一定要运往京城保存，是老衲率领弟子极力斡旋，才留在本寺，建了个展览馆……”
<br/>　　苦行大师抚摸着古意盎然的剑鞘，感慨道：“我本以为，这世上根本不会有人知道太阿剑的秘密，这么多年来一直相安无事，因此疏于防范了。”
<br/>　　叶少阳听他这么说，很不好意思，“那个啥，大师，我们本意也不想偷的，当时是担心你们不知情，说不清楚，才……咳，总之希望大师谅解，我来找太阿剑，绝不是为了自己，我……”
<br/>(本章完)
