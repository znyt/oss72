　　第551章
<br/>　　这一夜，程大雷睡得一点都不好。他半夜出去过一趟，回来后就听得耳边涛声阵阵。观海听涛说来颇为雅致，但在睡觉时的确可以称为噪音。
<br/>　　当初选择把书院建在山上的人，一定是脑袋秀逗了。一直捱到天亮，好不容易快睡着了，只听得外面喧哗声一片，吵得人不得安生。
<br/>　　“嘛呢，干嘛呢，要杀人嘛。”
<br/>　　刘发财从床上腾地一下跃起来，用刀将桌子拍得邦邦响。
<br/>　　“两位公子，两位公子……”伺候得小童着急走进来。
<br/>　　程大雷打了声哈欠：“外面干嘛呢，你们书院就是这样待客的么，小心我投诉你们。”
<br/>　　“昨夜……”小童顿了顿：“死人了。”
<br/>　　咦。
<br/>　　程大雷和刘发财对视一眼，都从彼此眼中看出了困惑。
<br/>　　二人简单梳洗过后，随着小童离开了院子。来到书院藏经楼前，见这里已围了不少人。在人群中，程大雷看到了楚云生。
<br/>　　楚云生忙走过来，向二人行礼。
<br/>　　“怎样，昨夜还快活么？”刘发财猥琐的问：“你却是美了，但却是苦了我们两个。”
<br/>　　楚云生脸上一红，低头道：“先生说笑了，在下能有今日，全要感谢二位。”
<br/>　　当真是居移体，养移气，只不过一夜不见，楚云生给人的感觉便和昨日不同。
<br/>　　程大雷倒是不太关心他昨夜的经历，问道：“怎样，这里发生什么事了？”
<br/>　　“钱观鱼死了。”
<br/>　　“喔，怎么死的？”
<br/>　　“好像是被人斧头砍死的，我到的时候，尸体已经被抬走了。”
<br/>　　书院年轻一代最潜力的弟子被人杀死了，而且还是在书院的腹心，这自然不是一件露脸的事情。如今钱观鱼的尸体已经被抬走，地下留着一片发干的血和昨夜战斗过得痕迹。
<br/>　　程大雷听周围人议论，大概也明白了事情经过。昨夜有一伙人潜入藏书楼盗书，杀死了守楼的钱观鱼，之后书院围追堵截，竟也没能留住对方。
<br/>　　什么人有这样的胆量，竟敢夜闯书院，而且是在伏魔大会的前夜。又是怎样的功夫，能出入书院若无人之境。
<br/>　　程大雷认真观察着周围，想从留下的战斗痕迹，判断出对方的身份。但其实也看不出来什么，书院很明显没有保护案发现场的意识。周围乱糟糟一片，仅留下的一点痕迹，当也被破坏了。
<br/>　　不过，程大雷当真还是读出了些什么，对方的出手干净利落，没有丝毫拖泥带水，似乎和帝国武学的路数不大相同。
<br/>　　当然，程大雷也没有做福尔摩斯·雷，替书院破案的愿望。他简单看过之后，便和刘发财回到住处。
<br/>　　“程当家……高啊。”刘发财竖起大拇指。
<br/>　　“高什么高？”程大雷一头雾水。
<br/>　　“夜半无声，杀人无形，这还不够高么。昨夜您偷了什么书，拿出来让我瞅瞅？”
<br/>　　“呃……”程大雷心里一万匹草泥马奔腾，他抿了口清茶，强压下去火气。
<br/>　　“那个……杀人的事不是您做的？”刘发财看出程大雷表情不太对，但他也的确知道程大雷昨夜出去过。
<br/>　　“假若是我做的，你觉得会被人碰着么？”程大雷将剑扣在桌子上。
<br/>　　“这个说得也有道理。”刘发财挠挠头：“可不是您，又会是谁呢？”
<br/>　　程大雷也猜不透，他想了想道：“有了今天的事情，咱们也莫节外生枝，今天碰到华氏一族的人，就带对方走，不掺和这里的事了。”
<br/>　　正这时候，那小童过来，冲二人道：“二位公子，夫子有请二位。”
<br/>　　刘发财心头打鼓，对方该不会已经开始怀疑二人。程大雷倒是坦然得很，挎上剑随着童子去往书院大厅。
<br/>　　大厅内坐满了人，表情都不是很愉悦。白袍夫子坐在主位，两侧立着伺候的童子。左手一字排开的太师椅，上面坐着的人奇形怪状，花里胡哨……
<br/>　　大概就是书院请来，参加今日伏魔大会的高手。程大雷集中注意力，查看各人的信息。不看不知道，一看吓一跳。大厅内二十几人，有七八个绝世人物，其中战斗职业竟也有三四位。
<br/>　　呃……如此，倒也不可太嚣张。万一不小心惹了众怒，陷入绝世高手的围攻中，那么等待自己的或许只有死路一条。
<br/>　　刘发财惊讶的发现，程大雷瞬间变得乖巧起来。步伐小了很多，走入大厅的时候，不吭不响的找个角落坐了下来。
<br/>　　没人注意到二人的出现，在他们还没到的时候，大厅内已经开始讨论。
<br/>　　“观鱼也并非弱手，身上只有一处致命伤，很显然在对方面前根本没有出手的机会。来者……不善呐。”
<br/>　　说话的是书院的宋寒之，是书院的教师，钱观鱼的剑道便是他传授的。这次心爱的弟子惨死，他的心情自然不是很好受。
<br/>　　“能一招杀掉宋先生的高徒，功夫自然不简单。大家推测一下，会是谁动得手？”
<br/>　　说话的是徐州九江城的老武师，名叫马猛，用一柄斩马刀，双臂有千斤之力。
<br/>　　众人一片沉默，天下高手是多的，能一个照面击杀钱观鱼的人也是多的。
<br/>　　“会不会是那魔头已经来了？”马猛自问自答：“钱观鱼死在斧头下，那魔头用的也是斧头？”
<br/>　　咦。
<br/>　　众人互相看着彼此，眼里目光闪烁，表达的就是同一个意思：并不是没有这个可能啊。
<br/>　　“绝无可能。”
<br/>　　一个声音突然响起，大家的目光都汇聚到程大雷身上。程大雷咬牙切齿，义愤填膺，你们不能把什么坏事都往我身上揽。
<br/>　　“这位欧公子，是昨日来到书院的，也愿意为诛杀程魔头出份力。”夫子冲众人介绍罢，道：“欧公子怎么肯定不是那魔头动的手？”
<br/>　　程大雷没有回答，而是问道：“据说昨夜那伙人的目的是盗书而来，敢问夫子，昨夜什么书丢了？”
<br/>　　这话倒是提醒了大家，目光都看向夫子。心里琢磨着，没听说那魔头是个读书人。而书院藏书万卷，究竟是那本书被偷走了。
<br/>　　夫子叹了口气，无可奈何道：“山水经没了。”
<br/>(本章完)
