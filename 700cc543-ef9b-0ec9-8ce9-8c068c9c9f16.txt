　　“这别墅，岂能是你想来就来，说走就走的地方，还是留下吧”
<br/>　　一道悠悠然的声音徒然响起。

<br/>　　什么声音，杜天易似有所感，抬头一望，电流声不绝于耳，一道比大腿还粗的雷电直接顺势而下，朝着他的头顶劈去。
<br/>　　该死。
<br/>　　杜天易大叫不好，想反应已经慢了，直接惨叫一声的被这道雷电劈了下去。
<br/>　　轰...
<br/>　　杜天易灰头土脸的从一个大坑中爬了出来，身上电流涌动，麻痹感差点让他高潮迭起。
<br/>　　好气啊，就差一点。

<br/>　　他杜天易风神秀异，仪表堂堂，乃是人中龙凤，比那些明星还有气度跟魅力，既然被一只大鸟给嫌弃了。

<br/>　　嗷...
<br/>　　黄金瞳不能口吐人言，只能嗷嗷叫的宛如猛虎出山的扑上去。
<br/>　　一时之间，两只超等生物弄的杜天易这位妖精管理局的组长好生狼狈，只能在院中不断后退，不断疲于应付。
<br/>　　整个院中风卷残云，爆响声不绝于耳，两只超等生物恐怖的气息，跟人类三阶生灵绞杀在一起，颇为的壮观。
<br/>　　阳台上，涂小安双手放在栏杆上，从容淡定的观看下方的战斗，就像是一个局外人看戏，就差拍手叫好了。

<br/>　　“主人，老牛不懂，你为什么非要跟这个杜天易过不去，对我们没好处的”牛妖王抖了抖脸上的横肉开口。

<br/>　　这是有多嫌自己命长呀，大家都修行不易，且行且珍惜。

<br/>　　别看牛妖王宛如混世魔王，但是粗中有细，考虑事情还是很全面。
<br/>　　“我说我要杀了他吗”涂小安白了他一眼：“你忘记了昨天有个剑修跟你一战，最后他怎么了”

<br/>　　“无量寿佛”张天师挥了挥手中的浮沉：“看着这位妖精管理局的组长，也要成为主人的战仆了”

<br/>　　如今一说，他猛然倒吸了一口气，阳台上空气立刻陷入真空状态。
