　　“啊？谁？”白羽满脸震惊，而后便猛然反应过来，道：“还真有这种人？这么说来小姐姐你在狼牙基地对尚九娣说的那些话，原来不是安慰乱说？！”
<br/>　　白允儿古怪的看了他一眼：“当然。”
<br/>　　这世上的确还有一个没有被神秘家族发现的精神类强化者，说来也是巧合，在白家刚发出了对精神类强化者诛杀令的时候，这个人就来到了白允儿面前。
<br/>　　当时的白允儿也不知出于什么原因，没有告知任何白家人，并让此人隐藏能力防止被追杀。
<br/>　　时间虽然已经过去了一年，此人也好似已经销声匿迹，但白允儿却知道在哪能找到。
<br/>　　最重要的是一年前的这个精神类强化者就已经是高阶新人类，当下狼牙研发出来的等阶提升剂又效果极佳，一举将此人的等阶提高到最巅峰，再唤醒不过是八阶的楚涵并不是太难的事。
<br/>　　白羽有些接受不了这些设定，晃了晃神后又担心道：“可是敌意很大是什么意思？那个人没对你怎么样吧？你没受伤吧？小姐姐你怎么不早说呢，趁他没成长起来赶紧灭掉啊！”
<br/>　　面对白羽的焦虑，白允儿却淡定自若：“不过是情敌而已。”
<br/>　　“情，情敌？！”白羽惊呆，而后猛然反应过来大喊：“又是和楚涵牵扯不清的女人？！”
<br/>　　白允儿笑而不语，脑海中却回想起了一抹红裙，以及那一张美艳精致的脸上，一双充满敌意的冰冷双目。
<br/>　　白羽可没有白允儿这么淡然，他根本咽不下这口气，愤怒道：“小姐姐我是真不懂你，为啥偏偏对楚涵这么特别？我承认楚涵这个人很厉害，是这个地球上最厉害的普通人，但是他再如何厉害与我们之间都是有鸿沟的，更何况他还这么花心！”
<br/>　　听到白羽的质问，白允儿眉头一挑，道：“楚涵普通？”
<br/>　　白羽不服气的说道：“当然！再厉害，也是普通人，永远都无法开启两把基因锁，也根本无法打败神秘家族的成员，这可是有血脉等级法则的！”
<br/>　　白允儿偏头看着他，眼中带着玩味：“你见过能进入阴阳谷白家旧址的普通人么？”
<br/>　　“恩？”白羽呆愣的眨了眨眼睛，一时间没反应过来白允儿在说什么，但却下意识的回答：“普通人怎么可能进得去，别说普通人，其余神秘家族的人也进不去啊，哪怕水火两族的人也进不去……”
<br/>　　只是说着说着，白羽的声音渐渐消失，面上不以为意的神情也开始转变，取而代之的是不可置信的惊愕。
<br/>　　白允儿面上的笑容很神秘，轻轻一点头：“楚涵进得去。”
<br/>　　白羽震惊无比，脚步再次无意识的停下，呆呆的站在原地不知所措。
<br/>　　楚涵进入过阴阳谷的白家旧址？
<br/>　　这怎么可能！
<br/>　　——————
<br/>　　狼牙基地。
<br/>　　洛冥从木屋中走出，身后紧跟他脚步的尚九娣轻轻合上门，防止任何人的靠近。
<br/>　　“已经四天，毫无醒来的迹象？”看过楚涵状态的洛冥担忧问道。
<br/>　　尚九娣与他并肩而行，眼中满是担忧的摇头：“就像是个植物人。”
<br/>　　洛冥点头，继续问道：“这件事还有谁知道？”
<br/>　　“目睹整个事发前后的蒋天庆，参谋部的几人，送楚涵回来的飞行员，不超过十人。”尚九娣一一细数着，而后又想起了什么说道：“哦对了，还有白允儿。”
<br/>　　听到这个陌生又熟悉的名字，洛冥满脸惊讶：“白允儿？她怎么会？”
<br/>　　“我也很奇怪。”尚九娣皱眉道：“她失踪了太久，更有情报说她已经回了白家，当下为白家的嫡系大小姐，还是唯一的一位。”
<br/>　　洛冥点头道：“这是事实，但白家……她过来有没有说什么？”
<br/>　　尚九娣想了想，道：“说了一些话让我等楚涵醒后转达。”
<br/>　　“恩？”洛冥表情古怪起来：“她说，等楚涵醒后，让你转达？”
<br/>　　“对。”尚九娣点头，也明白洛冥此时的惊讶是为何，继续道：“白允儿说，楚涵是在经历一场蜕变，早晚会醒的。”
<br/>　　洛冥的神情更为古怪，思绪一下子变乱：“这不对啊……”
<br/>　　尚九娣不敢出声打扰，实际上她也在这次事件中混乱无比，甚至都有些难以控制心态。
<br/>　　古怪的事件一件接一件，让人措手不及。
<br/>　　洛冥没有失神太久，冷静后说道：“去安罗城，我想见见袁熙叶，楚涵的母亲。”
<br/>　　虽然没能理清逻辑，但洛冥还是维持着大局观，循环论很有可能关系着整个末世纪元的走向，他需要亲自与袁熙叶交谈。
<br/>　　——————
<br/>　　华夏中部的某处荒无人烟地带，这里没有丧尸也没有异种，也没有人类生活的足迹，周围大·片的戈壁限制了末世里变异植物的生长，让这里成为了一片完全被废弃的地带。
<br/>　　但在这片荒地的地底，却有着一座大型地下基地！
<br/>　　这里便是当下的末世三年还无人知晓，上一世的末世十年却人尽皆知的曹氏研究院。
<br/>　　曹氏研究院的内部构造很特殊，有些像蚁巢却又有着诸多细节的不同，研究院内的研究人员数量不少，但却很多人都相互不认识，他们被分别禁锢在不同的地方，没日没夜的研究着曹春辉分配下来的研究任务。
<br/>　　而这些人也分别来自不同的地方，有些是文明时代的门萨组织成员，有些是曹春辉直接从其他基地研究部绑架而来的，更有些是不为人知的民间高手。
<br/>　　他们出不去，逃不掉，不知道这里是那里，也不知道今夕是何年，更不知道这时候外面世界是什么样子。
<br/>　　每日面对的只有研究，只有数据，和一些不知道从哪运过来的可怕原材料。
<br/>　　丧尸，异种，狂化动物，甚至活生生的新人类……
<br/>　　胡梦浩在南都基地与楚涵达成共识后，便就此在曹氏研究院内被软禁，这里的防备实在太严苛变·态，一举一动都备受监视，别说想办法逃出去，连往外面传个消息都难如登天。
<br/>　　胡梦浩很愤怒，甚至觉得楚涵在戏弄自己，直到曹氏研究院某次来了个陌生新人类。
