　　传递消息未果，苏莫便不再尝试，又飞行了一日之后，他停留在了一座城池之中。
<br/>　　他暂时不急着去天龙神朝，也不急着回神剑山，而是准备找个地方修炼。
<br/>　　荒郊野外，已经不适合他藏身了，人多的地方才是最佳之地。
<br/>　　这座城池名为明岳城，非常庞大，比之极空城还要大的多。
<br/>　　苏莫进入明岳城中，便找了一个豪华的客栈住了下来。
<br/>　　客栈中的上房，均是能隔绝气息之地，适合武者修炼。
<br/>　　房间之中，苏莫在蒲团上盘膝而坐，开始修炼了起来。
<br/>　　他并没有吞噬源石，而是参悟吞噬奥义，因为悟道神丹的作用还在。
<br/>　　悟道神丹，乃是顶级的修炼神药，药效可以持续二十年。
<br/>　　在时空之城中，苏莫服用悟道神丹之后，已经参悟了近十年。
<br/>　　所以，悟道神丹还有十年左右的时间，他自然不能浪费了。
<br/>　　意念感知到体内的吞噬元神，苏莫静静的参悟了起来。
<br/>　　时间悄然流逝，苏莫的吞噬奥义，不断的精深，不断的加强。
<br/>　　苏莫的吞噬法则，已达二阶圆满，但是，想要晋升三阶，即便有悟道神丹之助，也颇为困难。
<br/>　　两年时间，悄然流逝，苏莫参悟到了关键的地步，距离晋阶三阶，已经近在咫尺了。
<br/>　　“祝公子凯旋而归！”
<br/>　　“祝公子凯旋而归！”
<br/>　　这一日，苏莫耳中隐隐听到了山呼海啸般的喊声，与此同时，连整座客栈都在颤动。
<br/>　　“怎么回事？”苏莫立刻睁开了眼眸，闪电般的离开了客栈。
<br/>　　客栈之外，武者无数，街道上无数的武者，分立在道路两旁，几乎从城中央排到了城门口，行成了两条长龙。
<br/>　　这些武者，好似在夹道欢迎某个大人物。
<br/>　　苏莫心中好奇，左右看了一下，便看到，从城中央的方向，走来了一道身影。
<br/>　　这道身影是一名青年，身穿华服，身材极为高大，挺拔无比。
<br/>　　青年脸盘宽大，面容刚毅，背上背着一柄黑色大剑，一步百丈，威武至极。
<br/>　　“祝公子凯旋而归！”
<br/>　　“祝公子名震天池！”
<br/>　　街道两旁的武者，振臂高呼，神情振奋，同时，也有很多如同苏莫一般的外来人，显得惊奇不已。
<br/>　　“这位兄弟，此人是谁？这是要干什么？”
<br/>　　苏莫心中好奇，转头向身边一位矮小的青年问道。
<br/>　　“你居然不知道古剑公子？”矮小青年闻言，顿时诧异的望着苏莫，一脸的无语。
<br/>　　“呃……我见识浅薄，还请告知！“苏莫无语的说道，他难道就一定要认识对方！
<br/>　　“这是古剑山，人称古剑公子，乃是万古天骄，明岳城的骄傲，也是我们万化界域的骄傲！”矮小青年面带崇敬的说道，他的修为只有一品，对于古剑山自然是崇敬不已。
<br/>　　“哦，原来是万古天骄啊！”苏莫无语的点了点头，难怪这么大阵仗。
<br/>　　不过，那慕容雷利和公孙嘉，貌似也是万古天骄，也没有此人的排场大啊。
<br/>　　“你这是什么意思？”
<br/>　　矮小青年见苏莫的神情，顿时就有些不高兴了，因为苏莫的神态，仿佛万古天骄不值得惊讶一样。
<br/>　　“没什么意思，只是没料到而已！”苏莫轻笑一声，随着那古剑山渐渐走进，苏莫也看清了对方的修为，三品高级。
<br/>　　修为比公孙嘉和慕容雷利，还要高一个小级别。
<br/>　　此人虽然气息内敛，但是，依旧能让人感觉到一股无形的锋芒，凌厉而刺目。
<br/>　　“古剑公子剑道奇才，不仅拥有太古武魂榜排名第十四的万化神剑元神，还拥有太古异体榜排名第十五的先天神源体，天赋比一般的万古天骄还要厉害！”矮小青年得意说道，仿佛他自己拥有这种天赋一般。
<br/>　　“万化神剑元神？”苏莫闻言愕然，这人居然和他拥有同样的元神，这倒是有意思了。
<br/>　　“那是当然，古剑公子这一次去万道剑冢，定然让终极剑域那些以剑道称雄的剑修黯然失色。”矮小青年道。
<br/>　　“万道剑冢是什么地方？”苏莫疑惑的问道，上次就听公孙嘉说了万道剑冢，他并未在意。
<br/>　　现在看来，这个万道剑冢貌似不一般啊。
<br/>　　“剑修圣地，亦是死亡禁地，非顶级妖孽不可入！”
<br/>　　矮小青年面色郑重，道：“万道剑冢据说是终极剑尊留下的，为了磨炼天下剑修，据说与终极剑尊的至宝有关。”
<br/>　　“至宝，什么至宝？”苏莫心中好奇。
<br/>　　矮小青年闻言一笑，道：“这个至宝可不简单，足以让所有剑修疯狂，不过，没有人能拿到。
<br/>　　“到底是什么？”苏莫又问道。
<br/>　　“神剑山！”矮小青年道。
<br/>　　“神剑山？”苏莫闻言一怔，这至宝是终极神王殿山门所在的神剑山？
<br/>　　“嘿嘿，神剑山可是终极神王的终极神剑所化，一件至尊神器，拥有斩碎神域之威，这还不是至宝吗？”矮小青年奸笑着说道，这也不是什么秘密，所以他也没有什么避讳。
<br/>　　“这么厉害？”苏莫心中一惊，斩碎神域之威，这未免太夸张了吧。
<br/>　　“不夸张，一点也不夸张，当然，一般人就算有此神剑，也没有实力使用！”
<br/>　　矮小青年摇了摇头，此神剑固然强大，也也要看是谁使用。
<br/>　　若是传说中的终极剑尊使用，自然是无可匹敌，拥有毁灭世界之威能。
<br/>　　但是，一般的真神境武者得到了，根本发挥不出半点威力。
<br/>　　苏莫闻言，面色惊奇不已，那化为神剑山的终极神剑，这么厉害吗？
<br/>　　随着两人交谈，古剑山大步走过了身前的街道，来到了城门之前。
<br/>　　这时，只见古剑山穆然停了下来，转身望着城内的无数武者。
<br/>　　古剑山面色郑重，目光如电，一股凌厉霸道的气势，从他身上冲天而起。
<br/>　　“诸位，终极剑域一直是剑道界域，剑修闻名于世，但我万化界域，乃是万化老祖开创，同样是剑道至尊，万化老祖不比终极剑尊差！”
<br/>　　说到此地，古剑山微微顿了顿，继续道：“这一次去万道剑冢，我将代表万化神殿、代表万化界域，领教终极神王殿和终极剑域那些剑道奇才，向所有人证明，万化神殿的剑道，不比终极神王殿的剑道差分毫。”
<br/>　　古剑山的话音，如同雷霆长鸣，响彻在整个明岳城，震人心魄。
<br/>　　“公子威武！”
<br/>　　“公子威武！”
<br/>　　“公子威武！”
<br/>　　霎时之间，呼声震天，无数的武者为之呐喊。
<br/>　　古剑山出自明岳城，是他们明岳城的传奇，这一次，定然还会再创奇迹。
<br/>　　面色郑重的古剑山，再次环视全场，而后，带着无敌的自信，转身出城，破空而去。
<br/>　　苏莫见此，也转身回到了客栈的房间之中。
<br/>　　“万道剑冢真的如此不凡？”盘膝坐在房间中，苏莫面带惊奇之色。
<br/>　　连其它界域的万古天骄，都去万道剑冢，便可见此地的不凡。
<br/>　　苏莫心中沉吟，他想去看看，但是又担心有危险。
<br/>　　毕竟，万道剑冢在终极剑域，他可是杀许焱等人。
<br/>　　“不入虎穴焉得虎子！”沉默了良久之后，苏莫还是决定去看看。
<br/>　　正因为危险，别人或许猜不到他会冒险回去。
<br/>　　再者，他吞噬的银色大剑元神，还准备去给予绫裳呢。
<br/>　　正好，此行见机行事，将元神给予绫裳，他在顺便去万道剑冢看看。
<br/>　　随即，苏莫也没有再继续参悟吞噬奥义，离开了明岳城，向终极剑域而去。
