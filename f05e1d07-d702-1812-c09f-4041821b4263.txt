　　李浩然哈哈大笑，“所以，这就是你入侵人间的理由？人间之事，岂用你等轩辕氏来操心？”
<br/>　　星月奴道：“我却不跟你争，今日你杀我弟子，罪当不赦……青牛，你若一意孤行，我轩辕氏便只唯有一战额灭之……”
<br/>　　李浩然微笑不语，法术公会，对普通人间法师来说，是一股绝对恐怖的力量，不过，他还真不怕。
<br/>　　李浩然转身目视山下众人，说道：“你错了。”
<br/>　　“哦。”
<br/>　　“那个张晓寒，是你选中的应劫之人？”
<br/>　　“不错，人类不比我轩辕氏，灵根太差，这孩子却是其中的佼佼者，又福源广深，必然是应劫而生。”
<br/>　　“因此，你想扶持他掌控法术界？”
<br/>　　“非如此，天劫到来，无人听从号令。”星月奴道，“因此我又找了一批人间天赋绝顶者，来辅佐他，共同经营法术界。”
<br/>　　“错了。”李浩然淡淡说道。
<br/>　　“何错之有。”
<br/>　　“这少年，根本就不是应劫之人！”
<br/>　　星月奴冷笑，“我公会对他考量多年，也培养多年，确定他便是应劫之人，怎会不是？”
<br/>　　李浩然道：“我说了你也不会信。若你愿意等，我可以证明给你看。”
<br/>　　“等多久？”
<br/>　　“将近百年。”
<br/>　　星月奴哈哈大笑，“一百年，青牛祖师……你让我说什么好？”
<br/>　　“随你。我走之前，你要来便来，随时奉陪。”
<br/>　　李浩然说完，人影后退，看也不看身后，重新回到了小亭子里。
<br/>　　碧清也化成人形，冲星月奴冷笑了一声，跟着李浩然离开。
<br/>　　“我会来找你的……”星月奴丢下这句话，再度凝聚出人形，朝一直悬在空中的画卷飞去……
<br/>　　黑气缓缓散去。山下的众人立刻睁大眼睛，怀着激动的心情等待着战斗的结果——但是所看到的一幕，却让所有人都摸不到头绪：
<br/>　　那个蓝衣人重新又坐在了亭子里。身边多了那个碧清，立在他身后，两人都背朝着山顶，目视着远处的山峦，一动不动。
<br/>　　星月奴呢？
<br/>　　众人急忙移开目光，寻找星月奴的下落，这才看到那副画卷，正在缓缓落下来，画卷上又恢复了星月奴的形象。
<br/>　　这是……打平了？
<br/>　　众人大眼瞪小眼，连鬼婆和张晓寒等人也是满心困惑，不知道发生了什么事。
<br/>　　“今日暂且守兵，不日之后，我法术公会将全面进军人间，前来斩杀妖孽……”星月奴的声音，从画卷中传来，随后画卷自动收起，落在鬼婆的手中。
<br/>　　众人无言。
<br/>　　“师兄，她为什么就这么走了，虽然她只是一缕神念，但是他们这么多人，真打起来，就我们两个，占不到便宜。”碧清站在李浩然身边，不解地问道。
<br/>　　李浩然冷冷一笑，“这是星月奴的计谋而已，她杀不了我，何苦赔上众人的性命。她正瞅着找不到理由从法术公会调来更多的人，这次，算是找到借口了。”
<br/>　　碧清想起星月奴刚说的话，似有所悟，道：“师兄的意思是，她其实醉翁之意不在酒……”
<br/>　　“自然。不管最终能不能杀死你我，她的法术公会，都可以名正言顺地留在人间……这是打的一手好算盘。”
<br/>　　“原来如此……”碧清又想了一想，道：“师兄，那她之前说的话，是真是假，他们真的是一心为了帮助人间应劫？”
<br/>　　“是又如何？非我族类其心必异，就算他们帮人间渡劫，恐怕人间法术界，也成了他们的傀儡了，断不可如此。”
<br/>　　李浩然站起来，转身看着山下，目光锁定在张晓寒身上，“他不是应劫之人，若法术公会确定这一点，他们自会减缓入侵人间的步伐……”
<br/>　　“师兄的意思是？”
<br/>　　“而今你我在这里，势单力薄，不可能抵挡得住法术公会的入侵，这不是我的世界，而今之际，唯有拖延法术公会入侵人间的时间，拖延到半年之后，那里，才是我的世界……”
<br/>　　碧清皱起了眉头，道：“如何能拖延那么久？”
<br/>　　“第一步，先证明张晓寒不是应劫之人。”
<br/>　　“杀了他？”
<br/>　　“不，你我若动手，反而会背上杀害应劫之人的罪名。”
<br/>　　“那……”
<br/>　　李浩然目光移到了李浩然边上那人身上，冷冷道：“叶少阳。”
<br/>　　“师兄，我刚要说，叶少阳就在山下，要不要把他抓来？”
<br/>　　“不着急，你去跟着他便可，先看他跟徐福有没有往来，将来我再找他，却还有别的事。”
<br/>　　“什么事？”
<br/>　　“证明张晓寒不是应劫之人。”
<br/>　　碧清一听，更加不解了，问道：“这件事跟叶少阳有什么关系？”
<br/>　　“有。因为，他才是真正的应劫之人。”
<br/>　　碧清嘴巴张的老大。
<br/>　　对于李浩然和碧清的对话，叶少阳自然没听见，他甚至连李浩然的脸庞都看不清，不过他已经开始怀疑了，不光是李浩然的身形打扮和他作法的特点，还有，叶少阳想到了另外一层：之前跟碧清打交道的时候，碧清一个劲地打听李浩然的下落，说明李浩然对她来说是个极为重要的人。
<br/>　　而且，她在炼尸缸里呆了那么多年，应该不会突然跟一个陌生人合作，并且甘心做他的副手。结合这种种线索，叶少阳不得不怀疑，这个蓝衣人就是李浩然，唯一让他感到不解的是，如果这个蓝衣人是李浩然，那么，他是这个时代的人，还是跟自己一起从未来过来的？
<br/>　　“叶少阳！这是请帖，我送你的！”
<br/>　　陈晓宇的一句话，打断了叶少阳的思绪，定睛看着陈晓宇，心里十分纳闷。
<br/>　　“明天是我张晓寒师兄和梧桐师姐大婚的日子，你可一定要来光临，我，等你！”
<br/>　　叶少阳心中冷笑，他就知道，陈晓宇不可能有这么好心，给自己送什么喜帖，敢情是死性不改，想要在魂力上，当着众多法师的面羞辱自己……
<br/>　　叶少阳看了梧桐一眼，梧桐一脸的担心，但是一旁的张晓寒等人，却是一脸的笑意。他们自然跟陈晓宇一条心，想要让自己出丑，在张晓寒，还有一层，那便是因为梧桐。
