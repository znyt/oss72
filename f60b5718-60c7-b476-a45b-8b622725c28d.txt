　　第488章
<br/>　　面对程大雷期待又困惑的眼神，李婉儿轻启朱唇，自然而然道：“刘悲啊，还能是谁。”
<br/>　　“啊！”程大雷发出一声惊呼：“这不太可能么。”
<br/>　　“怎么不可能，刘悲随你去了并州，奥黛琳娜日日祈祷，生怕刘悲出了意外。这不，刘悲刚回来，她便急着去了。”
<br/>　　“这个这个……事情的发展，还真是出人预料呐。”
<br/>　　程大雷发出一声长叹，今天受到的冲击实在太多了，他当真需要好好缓缓。
<br/>　　奥黛琳娜有传教的信念，拉到一个人就要和人谈人生、谈信仰，山寨一帮大老粗也没人愿与她聊这个。而刘悲这个负能量满满的人物，平时也很少有人愿意和他接触，主要是不想被毁掉活下去的希望。
<br/>　　最后奥黛琳娜找到刘悲，与他大谈万物有灵，信则生灵。至于刘悲，则与奥黛琳娜谈些目光放远，万事皆悲的道理。一开始，谁也说服不了谁，但根据现在的情况判断，当是信仰没敌得过爱情的萌动，奥黛琳娜彻底沦陷了。
<br/>　　“这……”程大雷干着嗓子道：“有些不太地道呐。这个圣女是不能成亲的，她想着双宿双飞，就把这圣女推给你，你是被忽悠了呀。”
<br/>　　“我知道这个，我已不打算成亲。”李婉儿平静道：“我意已决，要用终身侍灵，希望能赎轻李家给天下带来的罪孽。”
<br/>　　说这话时，李婉儿戴上了脸上的面纱，黑纱遮面，只有一双眼睛看着程大雷。从她的双眸中，程大雷看到了某种叫做虔诚的东西，这是程大雷最讨厌的眼神之一，因为这种眼神，他也在正义教的信徒身上看到过。
<br/>　　“何苦如此？”程大雷不知该说什么。
<br/>　　“程大雷，我已经不恨林少羽，我也不恨你了。”李婉儿认真道：“在这段日子，我想了很多，天下沦落到今日，我李氏一族难辞其咎，一切，都是我李家应得的。做错事总是要偿还的，我父王用他的死，偿还了他的罪，如果他还有没还清的，便由我来还。”
<br/>　　“你再想想，好好想想，事情本和你无关。这里虽然不是世外桃源，但总能安稳度过一日。”程大雷顿了顿：“你六哥临走前，将你托付了给我。”
<br/>　　李婉儿不是傻瓜，已读懂了程大雷的言外之意。她似乎笑了笑，随之摇了摇头：“我已想清楚了。”
<br/>　　程大雷顿时有些怒不可遏，怒的不是李婉儿，而是刘悲。你为了自己双宿双飞，就不顾本当家的幸福生活了。李婉儿遭遇此巨变，脑袋有些不清楚，也是理所应当，但你不能让自己趁这机会，让自己的女人来忽悠他呀。
<br/>　　程大雷拂袖离去，立刻就要去找刘悲麻烦。心里其实有些好奇，这刘悲是怎么和奥黛琳娜捣鼓在一块儿的。
<br/>　　其实奥黛琳娜来到蛤蟆城后，山寨很多人都对她动过歪脑筋，不过在得知圣女不能成亲后，大家也就偃旗息鼓了。程大雷不得不佩服刘悲，看看人家，忽悠得对方把圣女的职务都不要了，完成了一项几乎不可能完成的任务。
<br/>　　程大雷路上碰到了和珅，和珅此时要去找刘悲对账。挖到宝藏后，刘悲曾经登记过一份数目，现在和珅要找刘悲对一下账目。
<br/>　　和珅简单向程大雷汇报了一下工作进度，所有财物已快完成，和珅的工作效率不可谓不快。令程大雷没有想到的是，所有竟没出现一件卷宝潜逃的情况。
<br/>　　要知道那些宝物每一件都价值万斤，卷走一件就可以一辈子吃喝无忧。但上万人的队伍，没有出现一个动歪心思的，也着实令人吃惊。
<br/>　　这证明，程大雷手下的蛤蟆军已经是不可撼动的铁板一块。
<br/>　　他将首领光环开启，再帮自己耍一下好感。
<br/>　　“大当家，你也找刘先生？”
<br/>　　“啊，找他有些事。”
<br/>　　程大雷随口应着，心里八卦之魂熊熊燃烧，想要知道刘悲这个时候和奥黛琳娜干嘛呢。小别胜新婚，二人会不会久别重逢，干柴烈火。
<br/>　　刘悲的院门关着，里面上着门栓。程大雷放轻了脚步，冲和珅做出噤声的手势。和珅也不知道程大雷要干嘛，就见他鬼鬼祟祟的，爬到门上，竖起一只眼睛透过门缝往里面瞅。
<br/>　　蛤蟆城的建筑，除了程大雷住的城主府稍微奢侈些外，大部分都是单独的四合院，李婉儿如此，刘悲也是如此。站在门口往里望，里面的情形就一目了然。
<br/>　　幸运的是，或不幸的是，刘悲并没有和奥黛琳娜干柴烈火，二人就站在院子里，随意聊着天。
<br/>　　二人你一言我一语，程大雷侧耳倾听，听得有些乏味，刘悲和奥黛琳娜扯起了万物归寂灭，一切都是空的道理。程大雷听得有些乏味，但大概有些明白，这奥黛琳娜双目脉脉含情，浓得令人化不开，但刘悲好像没悟到，仍然有一句每一句的扯着淡。
<br/>　　程大雷听得有些不耐烦了，重重的咳嗽了一声，拍响了木门。
<br/>　　刘悲将门打开，看到程大雷和和珅站在门口。
<br/>　　“干嘛呢，干嘛呢，大白天栓门做什么？”程大雷大马金刀的走进院子。
<br/>　　奥黛琳娜脸上飞红，刘悲仍一脸木讷，道：“大当家，你们怎么来了？”
<br/>　　和珅轻咳一声：“我找先生对一下账目，看先生何时有时间？”
<br/>　　“这件事不着急。”程大雷一摆手，道：“今天我来找你是谈一件更重要的事情。”
<br/>　　“什么是更重要的事？”
<br/>　　程大雷沉吟片刻，道：“春化夏长，秋收冬藏，万物生养才是道理。咱们山寨还有许多兄弟为成亲，这样下去总不是个法子。刘悲你要给弟兄们做个表率，我想了想，决定给你指门亲。”
<br/>　　“呃……大当家，这太突然了吧。”
<br/>　　“不突然，你想一想，你要是死在了草原上，你刘家连个后都没有，怎么给祖上交待。”程大雷看向奥黛琳娜：“圣女阁下，可有没有能推荐的人选？觉得谁可以配得上我们刘军师。”
<br/>(本章完)
