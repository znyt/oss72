　　“太上皇，突围吧，再这么下去，我们的物资基本上就要被消耗干净了。”秦琼秦二哥已经不知道第几次向老李提出建议，整个人显的很不淡定。
<br/>　　“朕绝不会向那些异族番邦投降，也不会舍弃任何人独自离开！”李世民站在战舰的船舷之上，透过望远镜看着远处密密麻麻、大大小小的战船，冷静的说道。
<br/>　　“太上皇，这不是投降的问题，眼下我们被围在海上，陛下那边又收不到消息，无法赶来支援我们，这样下去迟早有一天我们会把物资消耗干净，到最后全军覆没！”秦琼上前一步，沉声说道。
<br/>　　“那又如何？朕这一生遇到的危险比这严重的不知凡几，最后不也是活下来了。”李世民放下手中的望远镜，回身笑着在老秦的肩膀的拍了拍：“所以，朕绝不离开，朕一定会与那些大食人死战到底。”
<br/>　　“太上皇……”秦琼再次开口，但却不知应该说些什么。
<br/>　　跟着李世民一同出海的老长孙见秦琼劝说失败没有露出任何意外的表情，只是对他使了个眼色，然后主动上次说道：“太上皇，我们的淡水存量不多了，如果五天之内还是无法得到补给，那就只能等老天下雨来储存雨水了。”
<br/>　　“五天？”李世民微微皱眉，转头对身边老的连牙都已经快要掉光的方老太监说道：“老方，去，把海图拿来！”
<br/>　　“喏！”方老太监微一躬身便去了，时间不大一张桌子已经支好，李世民所需要的海图也被铺到了上面。
<br/>　　李世民走到海图的边上，盯着上面看了半天，找到了自己舰队所在的位置之后，指着距离最近的一个黑色小点说道：“这是什么地方？距离我们有多远？”
<br/>　　“无名岛，距离我们大概有两天的距离！”独孤青云扫了一眼海图，瞬间做出回答。
<br/>　　到底是在大海上混了十几二十年的老家伙，几乎已经成了半个活地图，基本上南海所有的岛独孤青云都上去看过，李世民所指的那个岛他同样也上去过，只是没有过多的停留罢了。
<br/>　　李世民在听了独孤青云的回答之后，低头想了想，忽然又抬起头：“给尉迟小子发信号，让他的舰队打头，破开敌人的防线，我们……上岛！”
<br/>　　有岛就一定有淡水，有淡水就一定有生存下去的机会，李世民不是轻易就认输的人，当年身边只有两万余人，武器只有简陋的刀剑时，他都敢算计自家老头儿举旗造反，更不要说他现在已经贵为太上皇，手下能臣猛将无数，常备军力二百万上下，武器装备精良到让敌人随便打都打不到的程度。
<br/>　　当然，这些指的是大唐的力量和他现在的处境没有什么太大的关系。
<br/>　　就眼下来说，他有十二艘钢铁战舰，其余战船四十余艘，兵力加在一起一万五千左右，这还包括一部分的宫女和太监。
<br/>　　而他的敌人，据观察战船就不下千艘，兵力十万往上，而且近期还不断有新的敌人加入进来，如此明显的对比在没有后援的情况下，放在一般人身上早就崩溃了。
<br/>　　可是李世民没有因此而产生任何困惑，大唐太上皇的身份让他根本没有办法与任何敌人妥协，在他看来自己可以死，但是绝对不可以投降。
<br/>　　背后的大唐有无数人在看着他，要李承乾在看着，满朝文武在看着，大唐的百姓在看着，如果他投降了，就算是不死也没有脸面再回大唐去见列祖列宗了。
<br/>　　另外还有一点就是，他有些时候甚至在怀疑眼下的局面是不是李承乾那个小子在搞鬼，毕竟以大唐现有的实力，如果那些大食人成功了还好说，如果没有成功，他们将要面对的是大唐最惨烈的报复。
<br/>　　没有哪个敌人会在这个时候来招惹大唐，能够打遍这片大陆没有敌手的大唐绝不是那么容易对付的，或许他们会嫉妒大唐，恨大唐，但如果说挑衅大唐，实在非智者所为。
<br/>　　只是这些都是李世民暗中的猜测，并没有对任何人说出来，甚至就连长孙皇后都不知道他心里有着这样的想法。
<br/>　　秦琼、尉迟恭对李世民的命令不置可否，他们是马上将军，对于海战可以说是一窍不通。
<br/>　　倒是独孤青云父子二人略微有些犹豫，迟疑了一下说道：“陛下，若是突围臣以为还是由补给舰为先导的好，这样就算是突围的时候损失一下也不会影响舰队的战力。”
<br/>　　“那就按你们说的办，总之一会儿通知小尉迟，让他做好准备，不要回头跟不上舰队的节奏。”李世民只管做决定，至于说具体的战术是什么他根本不会去管。
<br/>　　专业的事情交给专业的人去办是李二的一贯作风，疑人不用，用人不疑也正是他的手下能拥有那么多拥趸的原因之一。
<br/>　　“是！臣等遵旨！”独孤青云与儿子对望一眼，语气坚定无比的应道。
<br/>　　……
<br/>　　尉迟宝琪所率领的南海舰队其实原本并没有想到会参与到这样一场战事中来，在马六甲他们杀的尸山血海，将大食人彻底打懵逼了。
<br/>　　但奇怪的是，大食人虽然被打的苦不堪言，但就是没有退怯的意思，反而是每天滋扰不断，虽然打不过，但却一直在纠缠着他们。
<br/>　　这样的情况开始时尉迟宝琪还没有发现，直到打了近一个多月他才察觉到情况有些不对头，于是便舍弃了那些纠缠着自己的大食人，准备调头回去。
<br/>　　而就在这个时候，两具顺着洋流漂来的尸体引起了他的注意。
<br/>　　那是在马六甲海域守岛的大唐军士，按说他们绝对不应该离开自己所驻守的岛屿，更不会死的死在大海里面。
<br/>　　于是在这份好奇心的驱使下，尉迟宝琪带上舰队，逆着洋流而上，一路查了下去。
<br/>　　结果这一查他发现，所有留有大唐驻军的岛基本上都已经被人屠杀了一遍，岛上再也没有任何大唐的军人在驻守，反而成了土著的天下。
<br/>　　如此情况让尉迟宝琪心生警惕，并没有直接对岛上的那些土著展开报复，反而是继续向下查，这一查便又是近一个来月，直到他遇到了正在与大食人交战的李世民。
