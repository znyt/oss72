　　萧逸打量着这些尸体。
<br/>　　尸体上，无论出现什么痕迹，萧逸都不会感到意外。
<br/>　　毕竟这些尸体，必然都是从坑洞之上掉下来的。
<br/>　　掉下来前，本就遭遇过战斗；或是与妖兽战斗，或是武者间的生死厮杀。
<br/>　　而有的，则是像他一样，想来这金光禁地查探一番，下来之后再身死的。
<br/>　　情况有许多，伤势痕迹等等也有许多。
<br/>　　但，唯独千刀万剐这等痕迹，他却感到疑惑。
<br/>　　若是在坑洞之上，遭遇仇家，被杀，随后被千刀万剐，再扔下这金光禁地之底。
<br/>　　这是有可能的。
<br/>　　但这些尸体中，出现一具两具还好说。
<br/>　　这遍地尸骸里，有为数极多的尸体都带有这等痕迹，这就说不过去了。
<br/>　　总不能是这些武者生前都遇到仇家这么巧吧。
<br/>　　嗖…
<br/>　　萧逸指尖一道星光打出。
<br/>　　星光，照耀了周遭数百米范围。
<br/>　　萧逸的目光，定格在山壁上。
<br/>　　周遭山壁，插满了一把把武器，或刀或剑，或棍或枪…
<br/>　　观其品质，大多是亚圣器，也有不少下品圣器，不过都是破损的。
<br/>　　周遭狂风吹袭，气流激烈。
<br/>　　应该是这些武器被狂风吹起，在风的助力下，插到了山壁上。
<br/>　　难道说，这些尸体，便是在狂风乱舞中，武器四处激荡时，被弄得千刀万剐？
<br/>　　萧逸皱着眉。
<br/>　　武者的身躯，即便不是体修武者，若修为足够深厚，也是极其强悍的。
<br/>　　如北隐宗大长老，武道大能的修为下，即便身体被洞穿，也死不了。
<br/>　　若萧逸那时不杀他，他顶多是重伤的下场。
<br/>　　这周遭的武者尸体，虽然生前不是武道大能，但有那胆量跳下这深不可测的金光禁地，起码也是圣王境以上的人物。
<br/>　　这也是周遭尸体中，有不少已经死了数十年以上，但仍旧半腐烂，甚至尸体保存完好的原因。
<br/>　　而那些插在山壁上的武器，已然悉数破损，加上无数岁月的流逝，里头力量早已散尽。
<br/>　　它们充其量不过是比较锋利的武器。
<br/>　　按理说，不可能造成千刀万剐这般恐怖的痕迹。
<br/>　　“呼。”萧逸深呼吸一口气。
<br/>　　这金光禁地之下，到底有何危险？
<br/>　　萧逸握紧了剑，朝洞口而出。
<br/>　　罡风，从洞口那里吹袭而出。
<br/>　　金光力量，亦出现于洞口那个方向。
<br/>　　很显然，洞口才是这金光禁地的‘正主’。
<br/>　　要想知道这金光禁地的秘密，只有进入那洞口。
<br/>　　来到洞口处，萧逸望了一眼，脸色陡然一惊。
<br/>　　洞口之内，竟是别有洞天。
<br/>　　就现在一眼看去，洞口里头的范围，绝对有数十里长宽以上。
<br/>　　范围中央，一个洞府赫然伫立其中。
<br/>　　“洞府？”萧逸皱了皱眉。
<br/>　　环视四周，里头除了这洞府之外，空无一物。
<br/>　　萧逸没有打出星光。
<br/>　　因为，这周遭范围，完全被金光所照耀。
<br/>　　整个洞府，本就在浓郁金光的包裹下，显得金光熠熠。
<br/>　　萧逸感知了一下。
<br/>　　围绕洞府的金光力量，远比别的地方浓郁得多，也强得多。
<br/>　　同一时间，洞府之内，一丝丝金光力量，飘荡而出。
<br/>　　飘荡而出的金光，只有一丝丝，并不多浓郁。
<br/>　　但渐渐靠近周遭金光时，这一丝丝金光，却在不断变得浑厚。
<br/>　　“原来如此。”萧逸脸色恍然。
<br/>　　金光险地，以及金光禁地内的金光；真正的出处，应该是面前这个古老而巨大的洞府。
<br/>　　洞府之内，飘荡而出的金光并不多。
<br/>　　但无数岁月来，一丝丝金光不断累积下，便有了如今的浑厚金光。
<br/>　　嗖…
<br/>　　萧逸身影一闪，靠近了洞府，打算更清楚地打量这洞府。
<br/>　　洞府上，除了这金光熠熠的外表，以及古老气息外，再无特殊之处。
<br/>　　洞府上甚至连半个牌匾都没有，无名无姓。
<br/>　　倒是洞府前，伫立着18把利剑。
<br/>　　剑上锋芒毕露，同样散发着古老气息。
<br/>　　证明，这18把利剑，已存在了无数年时间。
<br/>　　但无数年时间下，却未让这18把利剑有半分损伤。
<br/>　　剑身锃亮，剑锋锐利。
<br/>　　“上品圣器？”萧逸感知了一下后，脸色大惊。
<br/>　　对，上品圣器。
<br/>　　18把利剑，皆是上品圣器。
<br/>　　每一把利剑的气息，都远远超过了冷焱剑。
<br/>　　上品圣器是什么概念？那是放眼中域，人人位之眼热觊觎的重宝。
<br/>　　哪怕是中域十八府这样的霸主势力，都未必有上品圣器。
<br/>　　即便有，也绝对不会有两把以上。
<br/>　　而这里，这个无名无姓的洞府前，却伫立着18把上品圣器。
<br/>　　“咕。”萧逸咽了口口水，面露惊喜之色。
<br/>　　想必没人会不对上品圣器眼馋，萧逸也不例外。
<br/>　　特别这18把上品圣器还是剑类圣器，对剑修的吸引力可想而知。
<br/>　　萧逸缓步靠近着，脸色肃然。
<br/>　　18把上品圣器，显然放在这里很久了。
<br/>　　但发现金光禁地的，他萧逸可不是第一人。
<br/>　　这金光禁地，早便被无数强者查探过了。
<br/>　　可这18把上品圣器，仍旧存在，仍旧牢牢伫立于洞府之前。
<br/>　　这代表着，无数强者，哪怕是万金府那等霸主势力，都无法拿走这18把利剑。
<br/>　　萧逸也不认为自己能轻松拿走。
<br/>　　甚至于，剑的周遭，有什么危险也不一定。
<br/>　　故他现今异常谨慎，亦步亦趋，缓缓靠近着。
<br/>　　一手握着冷焱剑，一手则扣紧了乾坤戒。
<br/>　　乾坤戒内，有不少重宝，那些都是他的底牌。
<br/>　　果然，萧逸刚走没几步，距离那18把利剑还有十米距离左右时…
<br/>　　锵…
<br/>　　一把利剑，锵鸣而起。
<br/>　　一道金光剑气，疾速劈来。
<br/>　　萧逸反应极快，一剑劈出，瞬间挡下了金光剑气。
<br/>　　不过，冷焱剑剑身上，明显一阵颤动。
<br/>　　刚才那道金光剑气，锋锐程度异常惊人。
<br/>　　若非冷焱剑本就极强的话，换了别的武器，哪怕是下品圣器，怕也瞬间被劈断了。
<br/>　　“原来如此，洞府之外有禁制和阵法。”
<br/>　　“18把利剑间，更自成剑阵。”
<br/>　　萧逸瞬间发现了洞府前的奥妙。
<br/>　　“剑阵？哼。”萧逸冷哼一声。
<br/>　　手中冷焱剑瞬间收起，取而代之的，是一道赤红色光芒覆盖于双手。
<br/>　　他的双手，瞬间化作了两只狰狞兽手。
<br/>　　嗖…身影一闪，直取18把利剑而去。
