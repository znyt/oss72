　　似乎看出小医仙有些拘束，慕清霄轻轻握住她的玉手，在她手心中挠了挠，顿时惹来她一个白眼。
<br/>　　气氛得到缓解，小医仙端起桌上的茶杯，轻轻抿了一口，惊讶道：“好香，似乎蕴含着一股极强的能量……”
<br/>　　“你这小女娃倒是识货。”
<br/>　　火云老祖捋了捋白花花的胡子，挂满皱纹的老脸上露出一抹笑意。
<br/>　　“茶是用八阶灵树上的树叶泡制，其中蕴含着庞大温和的火能量，也只有炙火山脉才有这种灵树，一杯就能让你受益无穷。”
<br/>　　“多谢前辈。”
<br/>　　小医仙道谢一声，随后道：“彩鳞姐姐，你要来一杯吗？”
<br/>　　“嘶嘶……”
<br/>　　旋即，慕清霄长袖中响起细微嘶鸣，只见彩鳞七彩身躯飘在空中，金色蛇瞳孔瞥了眼老者，随后落在茶壶上。
<br/>　　火云老祖目光一凝，脸上露出一抹笑意。
<br/>　　“这是……七彩吞天蟒，当初老夫也见过一次，只是后者乃是九彩，修为已至斗圣级别，没想到道友还饲养着如此有天赋的魔兽。”
<br/>　　火云老祖乃是斗圣，见多识广，哪怕斗尊都不一定能认出七彩吞天蟒，但他却能一眼辨别出来。
<br/>　　顷刻间，一股阴冷的杀意自彩鳞体内席卷而出。
<br/>　　慕清霄手一抖，差点连茶杯都丢在地上，故作平静道：“彩鳞并非饲养的魔兽，乃是家妻。”
<br/>　　话语落下，彩鳞才将杀意收敛，瞪了一眼火云老祖，可爱的尾巴一卷，直接将茶壶卷走，踏空回到慕清霄的手腕上。
<br/>　　火云老祖脸上的笑意凝固，尴尬道：“倒是老夫失礼了，道友莫怪。”
<br/>　　说罢，不经意的看了眼慕清霄，没想到后者有如此特殊嗜好，居然与魔兽结为夫妇，恐怕整个大陆上这种列子也不多吧。
<br/>　　就在这时，门外响起一道颇为沉稳的声音。
<br/>　　“老祖，唐震求见。”
<br/>　　“进来吧。”
<br/>　　慕清霄微微抬头，映入眼帘的是一名身形瘦小，一头红发并且身着朴素麻衣的老者，老者脸上从始至终都挂着一抹笑容。
<br/>　　唐震，五星斗尊。
<br/>　　进入竹屋内，唐震微微有些惊讶，没想到这里已经有客人，能让老祖接待的客人，身份地位恐怕极高，难道是丹塔？
<br/>　　落座后，唐震打量了一眼慕清霄与小医仙，目光看向火云老祖，道：“老祖，不知这两位是？”
<br/>　　火云老祖叹息一声，道：“这位道友乃是九品炼药师。”
<br/>　　“九品！”
<br/>　　唐震闻言，眼中满是不敢置信之色，九品炼药师，那可是大陆上最顶端的人物，就面前年轻的有些不像话的青年？
<br/>　　九品炼丹师，也就意味着后者实力恐怕比老祖还要可怕，否则老祖也不会如此客气。
<br/>　　回过神来，唐震压下心中的震惊，拱了拱手道：“不知可是丹塔前辈？”
<br/>　　“丹塔？”
<br/>　　慕清霄微微摇头，道：“我与丹塔没有任何关系，只是听闻唐谷主正在寻找炼药师，于是不远千里的来到焚炎谷。”
<br/>　　“呃……”
<br/>　　唐震一时间不敢接话茬，他可不是白痴，更不认为自己有让九品炼丹师，斗圣专门赶来焚炎谷为他炼丹的资格。
<br/>　　暂且不说他，恐怕火云老祖想要请动九品炼药师，都要付出不可估计的代价吧。
<br/>　　“不知前辈有何要求？”
<br/>　　既然九品炼药师特地敢来焚炎谷，后者定有所求。
<br/>　　只是让唐震想不通的是，焚炎谷有什么东西值得九品炼药师特地赶来。
<br/>　　慕清霄微微一笑，语气平淡道：“我亲自替唐谷主炼药，不知唐谷主可否忍痛割爱，将九龙雷罡火赠于我？”
<br/>　　闻言，唐震面色不变，下意识的看了一眼火云老祖，心中顿时涌上一股无奈。
<br/>　　他已经知道，焚炎谷惹不起后者，哪怕强如斗圣的火云老祖都已经低头，若是他说一个不字，恐怕会惹来无止尽的打击。
<br/>　　焚炎谷能够在中州中域屹立至今，都是因为焚炎谷对外界纷争并不干涉。
<br/>　　只是他没想到，如此强者会对九龙雷罡火感兴趣，虽是异火，但在异火榜上并非名列前茅，要说威力，比排名前几的异火更是无法比较。
<br/>　　“前辈，九龙雷罡火区区排名十二，如果前辈想要异火，估计有更好的选择吧？”
<br/>　　虽然唐震觉得希望有些渺茫，但依旧是想争取一下，九龙雷罡火是他们焚炎谷的传承火焰，若非无奈，他并不想交出去。
<br/>　　慕清霄闻言，也不在意，道：“其他异火并非没有，但九龙雷罡火也是目标其一而已。”
<br/>　　话语刚落，不仅唐震，就连火云老祖也是怔了怔，已经猜出后者估计是在收集异火，至于目的就不得而知了。
<br/>　　沉吟片刻，唐震慎重道：“既然如此，在下也不好拒绝前辈的要求，若是前辈能收服九龙雷罡炎，这枚异火就是前辈的了。”
<br/>　　“也好。”
<br/>　　慕清霄微微点头，区区一枚九龙雷罡火而已。
<br/>　　闻言，唐震站起身来，道：“九龙雷罡火威力极强，此处乃是老祖修身养性之地，并且也是焚炎谷的武技阁，请前辈移位。”
<br/>　　不多时，跟在唐震身后来到焚炎谷后山的一块空地上。
<br/>　　唐震也不再犹豫，双手快速结印，随后一团银色火焰就自他的心口漂浮而出。
<br/>　　火焰呈现银色，升腾间隐隐能够见到九条银色火龙，在火焰之内穿梭而行。
<br/>　　异火之内，有着龙威凝聚，因此有着震慑灵魂之神效银色火焰袅袅燃烧，九条细小的火龙在其中四下穿梭，犹如具备着灵智一般。
<br/>　　许些龙威从中弥漫而出，令得人灵魂力量感到有些压抑。
<br/>　　九龙雷罡火。
<br/>　　站在远处，小医仙就能感觉到异火中毁灭性的力量，慕清霄依旧保持着风轻云淡之色。
<br/>　　唐震祭出九龙雷罡火，慕清霄随手一挥，异火就朝他手心飘来，似乎是想逃离，一声声龙吟自异火中传出。
<br/>　　但慕清霄只是随手一挥就充斥着令它无法抗拒的力量，九龙雷罡火顿时有心无力。
<br/>　　望着手中的银色异火，慕清霄失望的摇了摇头，估计也只有陀舍古帝洞府中的帝炎才能让他突破渡劫了。
