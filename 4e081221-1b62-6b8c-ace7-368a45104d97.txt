　　随着这少年的话，站在他身旁的几位北邙学院弟子一齐开口道。
<br/>　　邓灵跟武皇，见到这少年，都是不由的眉头一皱。
<br/>　　刚想说什么，楚风眠却是率先开口了。
<br/>　　“他是谁？”
<br/>　　楚风眠倒是不记得，在北邙学院之中有着这样一个人，看来应该是楚风眠离开之后，才崛起的。
<br/>　　这少年看起来也就二十岁出头的模样，一身实力已经是达到了帝尊境二重，比起北邙学院一些稍弱的老祖，都毫不逊色了。
<br/>　　怪不得足以在弟子之中，有着如此威望。
<br/>　　如此年轻的帝尊，就算是大罗天星域之中，也算是也给小小的天才了。
<br/>　　“回禀门主，此人名为蔡徐，是院长的弟子。”
<br/>　　武皇跟邓灵听到楚风眠开口，都是可怜的看了蔡徐一眼。
<br/>　　这蔡徐算是北邙学院年轻一代弟子之中的佼佼者了，所以他们二人只是以劝解为主，想要让他也加入到剑道门之中。
<br/>　　可今日这蔡徐不识时务，一旦楚风眠出手，可就没有任何商量的余地了。
<br/>　　楚风眠出手的狠辣，他们可是领教过的，铁血手段，顺者昌逆者亡。
<br/>　　“门主，我去在劝一下他……”
<br/>　　邓灵看着楚风眠，小声的开口道。
<br/>　　“不用了，这样的人，留着也是害群之马，剑道门不需要这样有二心的弟子。”
<br/>　　楚风眠猛然一挥手。
<br/>　　听到这话，邓灵便是乖乖闭嘴，急忙的向着旁边退去，也是感叹一声。
<br/>　　这蔡徐的确是不识好歹，既然如此，邓灵也救不了他。
<br/>　　“小子，你不用嚣张！我虽然不知道你到底是用了什么手段，将院长弄走了，但是你想要让北邙学院，加入所谓的剑道门，你是想也不要想！”
<br/>　　这蔡徐看着楚风眠，发出尖锐的声音道。
<br/>　　“我们走！”
<br/>　　说着他便是转过头去，打算是带着一群北邙学院弟子离开宫殿。
<br/>　　“想走？”
<br/>　　楚风眠大手一挥，整个宫殿都是被封锁起来。
<br/>　　于此同时，在楚风眠的头顶之上，一道龙爪顿时向着那蔡徐轰击了过去。
<br/>　　“你敢杀我？”
<br/>　　看着楚风眠出手，那蔡徐的脸色才突然惊慌了起来。
<br/>　　他可是院长的弟子，北邙学院年轻一代的第一人，自恃清高，就算是许多老祖，都不敢出手对付他。
<br/>　　所以在面对楚风眠的时候，他也还是一如既往的狂傲。
<br/>　　但是别人在乎这所谓的天才，楚风眠可不在乎，楚风眠杀的天才多了。
<br/>　　至于院长弟子的身份，就连北邙学院的院长楚风眠都杀了，还在乎他？
<br/>　　蔡徐这还没有来得及凝聚灵力抵挡。
<br/>　　龙爪已经是无情的落下，轰隆！
<br/>　　一爪横扫过去，这蔡徐的身躯被凌空打爆，顿时化为了一团血雾，飘散在空中。
<br/>　　“死了？”
<br/>　　“蔡徐被人杀了！”
<br/>　　一群北邙学院弟子，还没有反应过来，就见到了蔡徐已经是在楚风眠的龙爪之下，被当场轰杀。
<br/>　　这可是一名帝尊，北邙学院年轻一代的第一人，甚至是据说，这蔡徐可都是未来北邙学院的院长。
<br/>　　可在楚风眠的手中，却是被当即轰杀，弹指一挥间。
<br/>　　接下来，不止是蔡徐。
<br/>　　楚风眠头顶之上的战龙之爪再次落下轰击，顿时之前那些附和这蔡徐，反抗楚风眠的北邙学院弟子，一个个都是从人群之中被当即拖了出来。
<br/>　　“不！”
<br/>　　“饶了我，门主饶了我！”
<br/>　　“我们也是被蔡徐蛊惑的，没有想要反抗过门主！”
<br/>　　那一群被从人群之中抓出来的北邙学院弟子，一个个瑟瑟发抖，高声的喊道，想要撇清关系。
<br/>　　可楚风眠却是丝毫没有理会，随着龙爪的横扫，这些北邙学院弟子也都被凌空打爆，化为了一团团的血雾。
<br/>　　嘶！
<br/>　　整个宫殿之内，鸦雀无声。
<br/>　　看着那些刚刚还嚣张无比，想要反抗楚风眠的那些北邙学院弟子，包括蔡徐，如今已经是都化为了一团血雾。
<br/>　　每一名北邙学院弟子的脸色都是变化，心中各有心思，可再也没有任何人，还敢反抗楚风眠。
<br/>　　一个个跪在地上，低着头，都不敢抬起头。
<br/>　　显然是楚风眠铁血手段，也彻底的震慑住了这一群人。
<br/>　　楚风眠心中也明白，想要掌控北邙学院可不是一件轻松的事。
<br/>　　所以他出手的时候，就没有丝毫一点点想要手下留情的打算，只有铁血，才足以镇压一切。
<br/>　　不然的话，纵然是楚风眠建立了剑道门，内部也是一片散沙，没有任何的意义。
<br/>　　楚风眠看着空中一团团的血雾，手心一点，顿时开始凝聚，化为了几百枚血丹。
<br/>　　随即楚风眠手一挥，这些血丹便是向着下方一群北邙学院弟子飞去。
<br/>　　“从此之后，你们就是剑道门的弟子，北邙学院，本就是剑道门的分支，现在是重塑本名，只要是未来对剑道门忠心，不会亏待你们。”
<br/>　　楚风眠一边开口，一边突然打开空戒。
<br/>　　只见就在宫殿之中，飞出了无数灵石，大量的皇品，圣品灵石，都飞了出来。
<br/>　　堆积成山。
<br/>　　看到这些皇品，圣品灵石的一刻，下方无数弟子的眼神都是变的炙热无比。
<br/>　　对帝尊境之下的武者来说，灵石，便是最为需求之物，大量的灵石，就足以让他们晋升实力。
<br/>　　而楚风眠最不缺少的，就是灵石。
<br/>　　而且不止是灵石，大量的灵药，灵器，材料，记载武技的书籍玉简，纷纷飞了出来，堆积成山。
<br/>　　想要彻底建立起剑道门，光凭楚风眠一个人是没用的，他也要培养出真正的剑道门弟子出来。
<br/>　　这些北邙学院的弟子，虽然是被楚风眠的手段震慑住了，可对楚风眠也是害怕为主。
<br/>　　内心并不觉得自己是剑道门弟子，而在震慑住他们之后，第二件事，楚风眠就是要让他们，真正的认可剑道门。
<br/>　　而做到这一点，最简单的，便是资源培养。
<br/>　　北邙学院弟子，愿意加入北邙学院，为的也是北邙学院的资源。
<br/>　　楚风眠的资源，比起北邙学院要多出数千倍，数万倍，大量的资源的拿得出
<br/>　　只要是长时间培养下去，也足以是这些人，彻底的认可剑道门，没有异心。
