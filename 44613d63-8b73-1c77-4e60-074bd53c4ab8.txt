　　洪荒之内，太清老子一朝三尸融，阴阳证道成圣。
<br/>　　圣威震动九重天，紫气纵横三万里！
<br/>　　霞光漫天，瑞气浮空，灵光纵横，异象惊天。一道道巨大的阴阳太极图宛若实质般的显现在洪荒各处，大道铺路，法则显露，无数修士在这一刻顿悟，更有许多修士借此得以开启灵智，更甚者化形而出。
<br/>　　“这……这是昆仑山方向？谁成圣了？太清成圣了？”
<br/>　　“太清老子已经成圣！那么，通天也快了。”
<br/>　　西方灵山，准提与接引正在论道，突然被这股强横的圣威惊醒，他们看了一眼昆仑山方向，眼中惊色良久都没有退去，两人顿时沉默了，许久后，他们相互对视一眼，接引道：
<br/>　　“善哉！老子已经成圣了，那么通天也快了。师弟，我们也该努力了。”
<br/>　　“是啊，师兄，不过，我还没有感受到我的圣路，怎么办？”准提嘴角略带苦涩地说道。
<br/>　　“师弟，不必刻意强求，我等只需静心参悟大道，机缘迟早会到。”接引双手合起放于胸前，默默道。
<br/>　　“善哉！”
<br/>　　……
<br/>　　天庭太阳神宫中，帝俊与东皇太一正在商量妖族事宜，突然被老子成圣的惊天异象惊动，他们急忙看向昆仑山方向，默默无语良久，脸上露出苦笑，道：
<br/>　　“老子竟然成圣了？！看来，我们的计划得提前了。”
<br/>　　“可恶！老子竟然成圣了，那岂不是说，通天亦不远了，到时三清都成圣了，那么，我们怎么办？”太一在一旁骂道。
<br/>　　“二弟，慎言，圣人手段深不可测，小心此话被他人听去。”帝俊眉头一皱，道。
<br/>　　太一顿时没有说话了，但他的脸色依旧不是很好看。
<br/>　　娲皇宫中，女娲看向昆仑山方向，神色惊讶，随即脸色却变得十分难看，恨声道：
<br/>　　“三清，竟然是三尸成圣，没有功德成圣？！”
<br/>　　万寿山，五庄观，镇元子脸色一动，瞬间便恢复了平静，若仔细看，便会发现他心中远没有他外面表现出来那般平静，他看了看手中的地书天地胎膜，眼神变得更加坚定了。
<br/>　　“圣人，圣人啊！不成圣，终为蝼蚁！哎——”
<br/>　　北冥宫，鲲鹏神色一惊。幽冥血海，冥河老祖眼中血光爆闪，手中两柄先天杀剑杀气纵横，切割空间。火云洞，红云老祖神色一动，眼中闪过一丝烦恼，抓起身边的葫芦喝酒。
<br/>　　巫族之中，十二祖巫虽然也被惊到了，但他们并没有太过将此事放在心上，因为只要不是妖族成圣，其他人成圣与他们无关，如上次女娲成圣，他们就被惊扰了好久。
<br/>　　……
<br/>　　这一刻，洪荒都变得十分不平静。暗潮涌动，风云渐起，沉寂许久的洪荒将要被打破。
<br/>　　昆仑山中，在老子成圣后，元始与老子便将目光投向通天，道：
<br/>　　“三弟，该你了。”
<br/>　　“嗯。”
<br/>　　通天也不废话，他剑眉一翘，神色一定，来到虚空之中，盘腿坐下。调理好心态后，通天便准备突破了。
<br/>　　虚空之中一阵波动，走出三道身影，正是通天的三尸。
<br/>　　一道身穿白衣的青年男子横踏一步，手持十一品净世白莲，道：
<br/>　　“贫道阵道之主灵宝天尊！”
<br/>　　一道身穿黑衣的青年男子来到通天左边，手持昆吾剑，霸声道：
<br/>　　“本尊玉宸道君！”
<br/>　　一身青衣，一头青丝的青年满脸孤傲，他手持造化青莲所化的极品先天灵宝青萍剑，来到通天身前，淡声道：
<br/>　　“贫道青莲剑尊！”
<br/>　　“无极生太极，太极成五行，定天地之万法，使万灵归期宗！吾乃上清通天，三尸，给吾融！”
<br/>　　通天大叫一声，三尸顿时化作黑、白、青三色的清流，融入到通天头顶的三朵道花之上。
<br/>　　通天并不像老子那般小心，他一上来，便是三尸同化。在三道清流的滋润之下，三花齐放，三道虚幻的身影盘坐在其中诵读天地道经，大道之音飘荡出来，化为一头头洪荒异兽，在空中飞舞。
<br/>　　“一剑开世界！”
<br/>　　通天眼神一亮，手持青萍剑，一剑向着虚空劈去，顿时，空间破碎，演化混沌，混沌分裂，演化天地四象，在三人的眼前出现了一道充满剑气的世界，世界中满天星辰是剑气组成的，大山湖泊一样是凌厉的剑气组成，世界中的一切，都被化作漫天剑气。
<br/>　　这是一个由剑组成的世界。
<br/>　　“二剑灭世间！”
<br/>　　轰！
<br/>　　世界倒塌，星辰破碎，山河倒流，大海干枯，化为剑刃风暴，漫天剑气化作一道狂裂的剑刃风暴，瞬间便席卷整片剑界。
<br/>　　世界化为一道惊天界剑，虚化的界剑之上，充满天地万物的道韵，万灵皆在其中，万法自有它定。
<br/>　　它，就是一个世界，一个剑形的世界。
<br/>　　“三剑截天机！”
<br/>　　哗啦！
<br/>　　一件化出，宛若虚空生莲，光华流转，极尽辉煌，璀璨无比。剑光划开天际，没入虚空，切断时空，天机混乱，宛若截断天道，破开天道牢笼，一股大自在，大逍遥的意味，自在其中。
<br/>　　“给我，破——”
<br/>　　通天大喝一声，他头顶的三花早已经凝聚成一颗晶莹剔透的道果了。在通天的道果之中，布满凌厉的剑气，他的道果就仿佛是一个缩小了无数倍的剑界。
<br/>　　鸿蒙紫气从他元神之中飞出，没入道果之重，将道果染成紫金色，闪烁着紫金光芒，一股莫名的道韵从通天身上散发出来，通天全身的气势在不断拔高，宛若一道屹立绝巅的剑山在不断被拔高，剑刃之气切割空间，空间变得破碎不堪。
<br/>　　若不是旁边还有元始与老子出手护住了昆仑山，否则昆仑山就要不存在了。
<br/>　　通天的气势不停，依旧在不停增长，终于，他来到一个屏障之前，他突破不进去，好似被禁锢了，若是不能突破，他此次便不能进入圣人境界。
<br/>　　“一鼓作气，再而衰，三而竭！万古道成空，剑道我独尊，破！”
<br/>　　通天大喝一声，剑光大涨，剑气飞扬，在他头顶，好似出现了一柄巨大的虚幻之剑，一无反顾地向着天际劈去。
<br/>　　咔擦——
<br/>　　陡然，通天身上好似传来一声清脆的响声。紧接着，喀嚓声不断响起，那道阻拦通天j晋级的屏障瞬间破碎，化为无数碎片，融入到通天身体之中。
<br/>　　轰！
<br/>　　一股更为强大的气息从通天身上升起，通天的元神随着那气息不断升高，如之前老子那般，他的元神穿过无数空间，被天道改造成了圣魂，魂寄天道。
<br/>　　从此，天道不灭，圣人不死！
<br/>　　此时，一道大道歌声从虚空传来，声音宏大，玄妙无比，传遍洪荒大地，令洪荒震动：
<br/>　　“鸿蒙判世吾为尊，玄黄衍化炼真神！
<br/>　　开天辟地为剑主，混元一念尽道生！”
<br/>　　“洪荒之内，吾为通天剑主！”
<br/>　　“吾，上清灵宝天尊玉宸大道君通天，今日成圣！”
<br/>　　轰！
