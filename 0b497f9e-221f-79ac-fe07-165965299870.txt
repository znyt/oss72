　　秋镇长早就把他当活神仙看待了，听他这么说，感激的不行，各种道谢。
<br/>　　叶少阳慢悠悠地喝了一口茶，道：“但是我也不能白白帮你。”
<br/>　　秋镇长一听，立刻就明白了，赔笑说道：“这是自然，不能让先生白白出手，不知先生要多少报酬？”
<br/>　　这句话可让叶少阳犯难了，一个镇长，官说大不大，说小也不小，钱肯定是有一些的，如果在当代，叶少阳开口要个几十万，人家应该能给的起，但是这是民国，叶少阳来了有几天，也大致知道了这时候的货币价值，但还是缺乏一些具体概念，简单说，他并不知道一个镇长家里应该有多少钱。
<br/>　　于是叶少阳求助地朝翠云看去。
<br/>　　翠云正在吃一个水果糖，嚼得津津有味，见叶少阳看她，不知道什么意思，不好意思地笑了笑，剥了一个给他，说道：“好吃，少阳你也吃一个。”
<br/>　　叶少阳无语，想了想，对秋镇长道：“我看你也是实在人，我跟你说实话吧，我要去一趟江西，但是现在身无分文，你给我一笔钱，够我能够走到那里就行，当然路上不至于风餐露宿。”
<br/>　　秋镇长本来已经做好了他狮子大开口的准备，听叶少阳这么一说，暗暗松了口气，感激地看了叶少阳一眼，道：“叶先生这般豪情，我秋某人绝不小气，我给你四条小黄鱼……”说罢就叫来一个管家，让他去拿过来。
<br/>　　叶少阳听他前面那么说，觉得他一定不会亏待自己，最后听到“四条小黄鱼”，当场就蒙了，甚至以为自己听错了，望着秋镇长，喃喃说道：“四条小黄鱼，我说大哥，拿东西来当钱我是听说过，您要是真没钱，给我点粮食去换钱也行，您这就给四条鱼，你逗我呢？”
<br/>　　秋镇长比他还懵，一时间竟不知道怎么回答，恰好这时候管家来了，将所谓的四条小黄鱼放在桌子上。
<br/>　　叶少阳看了一眼，怔住了，这哪里是鱼，分明是金条啊！顿时无师自通的明白了：小黄鱼，说的就是金条！因为是黄颜色长长的？
<br/>　　当下讪讪地笑起来，对秋镇长说道：“镇长大人真风趣，把金条说成是鱼，我还以为是真鱼来着。”
<br/>　　秋镇长咧了咧嘴，一肚子的郁闷，心想把金条叫“黄鱼”的说法，又不是自己发明的。人人都知道，不知道叶少阳为什么这么说。
<br/>　　翠云看见四条小黄鱼，忍不住激动起来，冲叶少阳抿嘴笑着，说道：“少阳，这次别说是江西，就是广西都去得了。”
<br/>　　叶少阳不知道金条的价格，她是知道的，按照当时的货币，一条小黄鱼能换差不多三百块大洋，而像秋镇长居住的这么大的宅子，在这个小地方最多也只要不到一百块大洋。
<br/>　　也就是说，如果叶少阳不走的话，完全可以买一套大房子，用剩下的钱买几十亩地，再做个小买卖，只要时局不变，基本上可以衣食无忧一辈子了。
<br/>　　叶少阳还是第一次接触金条，在手里把玩了一会，将一根装进兜里，剩下三根递给翠云。
<br/>　　翠云愣了一下，摆手说道：“这么贵重，你自己放着，我怕给你丢了。”
<br/>　　叶少阳道：“不是让你放着，是给你的。”
<br/>　　翠云当成就懵了。秋镇长也懵了。
<br/>　　半晌，翠云回过神来，猛地站起来，摆着两手道：“使不得使不得！这是你赚的，你要是真心疼姐，走之前给姐丢几十块大洋就行，这么多钱……少阳你脑袋不好了！”
<br/>　　叶少阳道：“我脑袋很好，这些钱，就当是饭钱好了，行了，你先拿着，别推来推去的，让人笑话。”
<br/>　　将三条小黄鱼硬塞进翠云口袋里，翠云看了秋镇长一眼，只好暂时忍着了，看向叶少阳的眼神，充满了一种复杂的神情。
<br/>　　叶少阳却觉得没什么，并不是因为不知道金条的价值，只有一点，滴水之恩，涌泉相报。翠云等于救了自己的命，又照顾自己这些天，别说这几条小黄鱼，就算是再多十倍，叶少阳也眼睛不眨。
<br/>　　秋镇长用力一拍桌子，对叶少阳竖起了大拇指。“叶先生真是好汉子！秋某人佩服之极，叶先生放心，你这表姐是我治下居民，将来自当竭力照顾，没得说！”
<br/>　　叶少阳道了谢，道：“我还想要一件东西。”
<br/>　　翠云一听，急忙说道：“少阳，够多了！”
<br/>　　秋镇长也觉得有些纳闷。
<br/>　　叶少阳从桌上拿起了那个石英钟的八音盒，道：“这个东西，能不能送我？”
<br/>　　秋镇长先是愣了一下，随即笑起来：“这是我当初从北京回来，买来的新奇玩意，并不值钱，只是我们小地方没有，叶先生喜欢，拿去就是了。”
<br/>　　“那就谢了。”叶少阳将八音盒递到翠云手中，笑道：“我说过送你一个的。”
<br/>　　翠云呆呆地看着他，心里头拥塞着各种情绪，也是从这一刻起，他对叶少阳的印象完全改观，相信他不是一般人，与所有她过去认识的人都不一样！
<br/>　　喝完茶，叶少阳开始跟秋镇长制定捉鬼的计划。
<br/>　　“她走之后，这三天之内，必然回来探听消息，或者改变计划，过来杀死阿兰，免得夜长梦多……你别担心，我布置一下，擒住她肯定没问题，你先给我准备一些东西，最重要的是，找七盏灯，最好是大号的那种手电，能瞬间很亮的……”
<br/>　　叶少阳只顾在那说，等说完了，抬头看去，秋镇长和翠云一脸懵逼，顿时皱起眉头，“我说的东西，都不是很难找吧，有问题？”
<br/>　　秋镇长嗫嚅道：“别的都好办，就是你说的那种灯……不是我说，世上有这种灯？”
<br/>　　叶少阳怔住。“大号的射灯，可以买到啊。”
<br/>　　“哪里去吗？”秋镇长比他还震惊，“那个，射灯是什么灯？”
<br/>　　叶少阳瞬间就明白了，用力拍了一下脑门，妈个蛋的，自己一时间错乱了，这里是民国！这个小镇估计连电都没通，哪来的手电？
