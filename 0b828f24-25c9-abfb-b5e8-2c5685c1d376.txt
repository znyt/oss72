　　“海阁主，多谢了！”
<br/>　　苏莫向海明江微微抱拳，如此多的财富资源，对方分文不取，他确实要感谢对方。
<br/>　　“哈哈！苏宫主见外了，你我现在是盟友，不必客气！”海明江哈哈大笑道，他对于这些财富倒是没有什么概念。
<br/>　　天涯海阁是天下间最富有的势力，财富对于他来说已经没有任何用处。
<br/>　　而且，现在天涯海阁的总部被灭，天涯海阁陷在了灭亡的边缘，一切他还要指望苏莫，自然不会和苏莫争抢。
<br/>　　“苏宫主，接下来我们怎么办？”海明江问道。
<br/>　　苏莫闻言，心中沉思了片刻，问道：“海阁主，你是不是想去天海城？”
<br/>　　海明江闻言一怔，随即笑道：“不错，老朽的确是想回天海城看看！”
<br/>　　天海城是天涯海阁总部所在之地，也是阴阳神宫所在之地，海明江早就想去天海城了，只是一直没说而已。
<br/>　　“好，我们就去天海城！”苏莫点了点头。
<br/>　　随即，苏莫便和海明江一切，向天海城而去。
<br/>　　阴阳神宫，也是和黄泉魔宗结盟的势力，若是有机会诛灭阴阳神宫，苏莫自然不会放过。
<br/>　　……
<br/>　　如今的天海城显得有些萧条，高.耸的城墙，很多位置都进行了重新修缮。
<br/>　　前段时间的天涯海阁总部遭到攻击，天海城也遭到了很大的破坏，虽然如今已经全部修缮完毕，但因为太涯海阁总部被灭，城中人气大减。
<br/>　　阴阳神宫，坐落在天海城的城东，同样是一座城中之城。
<br/>　　一间偏殿之中，端坐着五位身影。
<br/>　　这五人中，有两人身穿黑袍，其中一人身体上黑雾缭绕，正是黄泉魔宗二魔老。
<br/>　　而另外一人则是身形枯瘦无比，头上戴着斗篷，全身都笼罩在黑袍之中，看不清其面容。
<br/>　　至于另外三人，则全部都是四、五十岁左右的中年人，个个身上气息深邃，实力深不可测。
<br/>　　其中一人身穿锦袍，五官英挺、气度非凡，此人在年轻之时，必定是一位美男子。
<br/>　　而另外两人，一人身形高壮，一人身形矮小，均是面容肃穆，面带威严。
<br/>　　“二魔老，以你和北堂兄、殷兄三人之力，居然杀不死帝释，看来这帝释的实力果然不简单。”那名身穿锦袍、气度非凡的中年人开口了，声音中略有些凝重。
<br/>　　这名气度不凡的中年人，正是阴阳神宫的宫主，阳鼎天。
<br/>　　“他一心逃跑，我等想杀他谈何容易！”二魔老声音低沉无比，他们追杀了帝释整整五日，最终还是让帝释给逃了。
<br/>　　作为准帝强者，几人的实力都相差不多，帝释一心想逃，他们也是无计可施。
<br/>　　“帝释已经不足为患，天涯海阁也是垂死挣扎，我等目前要考虑的是如何对付苍穹神宫！”那名身形矮小的中年说道。
<br/>　　此人，便是殷家家主，殷棕。
<br/>　　“不错！对付苍穹神宫和苏莫，才是重中之重，只要灭了苍穹神宫，整个苍穹世界便垂手可得！”那名身形高壮的中年人点了点头。
<br/>　　此人，乃是北堂世家家主，北堂平华。
<br/>　　“苍穹神宫之外的守护大阵，乃是远古奇阵‘浑天河洛阵’，虽然威力已经大减，但想硬攻进去，绝对不可能！”
<br/>　　二魔老摇了摇头，继续道：“而且，据我们探听到的消息，苍穹神宫中有九名准帝级的傀儡人守护，硬攻不可取！”
<br/>　　众人闻言沉默了，无法攻破苍穹神宫的守护大阵，那怎么灭苍穹神宫。
<br/>　　“二魔老，那我等该怎么办？”少倾之后，殷棕沉声问道。
<br/>　　“二魔老，我们绝对不能让苍穹神宫成长下去，苏莫的天赋实在太恐怖了，修炼的速度非常的快，我们不能给他成长的时间！”阳鼎天告诫道。
<br/>　　“不错！”北堂平华点头赞同。
<br/>　　二魔老闻言沉吟了片刻，道：“现在有两个办法。”
<br/>　　“哪两个办法？”众人问道。
<br/>　　“第一个办法是等，等我圣宗宗主突破到武帝之境，灭苍穹神宫轻而易举；第二个办法也是等，等苏莫离开苍穹神宫，我们全力围杀！”二魔老声音低沉的说道。
<br/>　　众人闻言沉思，他们都知道黄泉魔宗宗主，不日就要踏入帝境，成为一名真正的武帝强者。
<br/>　　不过，具体什么时候能真正突破，他们心中也没有底，快则一月时间，慢则需要半年左右。
<br/>　　但是，他们却不能干等下去了！
<br/>　　要知道，苏莫这个妖孽，可是两个月左右提升一个境界，等黄泉魔宗宗主突破到帝境，怕是苏莫也会有了武帝的实力，到时候就麻烦了。
<br/>　　“还是第二个办法吧！”
<br/>　　阳鼎天微微皱眉，叹道：“若是第二个办法完不成，那就只能等第一个办法了！”
<br/>　　“嗯！”
<br/>　　“只能这样了！”
<br/>　　众人点头，表示赞同。
<br/>　　二魔老见此，转头看向身边戴着斗篷的黑袍人，问道：“大魔老，你觉得如何？”
<br/>　　不错，这名戴着斗篷的黑袍人，正是黄泉魔宗的大魔老。
<br/>　　灭帝玄宫是二魔老和殷棕、北堂平华三人出手。
<br/>　　而灭天涯海阁总部，则是大魔老和阳鼎天出手，灭了天涯海阁总部之后，大魔老就在阴阳神宫等待二魔老、殷棕和北堂平华。
<br/>　　众人的目光都转向了大魔老，目中隐隐有忌惮之色，这名大魔老的实力，非常恐怖，绝对是准帝中最强的存在。
<br/>　　而且，最主要的是，大魔老不是人，而是一具尸体。
<br/>　　大魔老原本是一具尸魁，吸收天地精华，苦修近千年，才修炼到了准帝的境界。
<br/>　　面对二魔老的询问，大魔老仿若木头一般，只是微微的点了点头。
<br/>　　“好，那就这么定了，我们时刻监察苍穹神宫，注意苏莫的动向！”
<br/>　　二魔老见大魔老点头，顿时对众人沉声说道。
<br/>　　咚！咚！咚！
<br/>　　就在此刻，蓦地，外面响起了急促的敲门声。
<br/>　　众人闻声眉头一皱，他们议事之时，下面的人居然敢来打扰！
<br/>　　“进来！”阳顶天轻喝一声。
<br/>　　偏殿的大门打开，一名面无表情的黑袍青年，大步的走了进来。
<br/>　　若是苏莫在此，必定能一眼认出此人，因为此人正是上官昊。
