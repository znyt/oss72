一开始大家并没有过多的关注到福伯这个人，毕竟幽兰苑虽然神秘，但是又不是什么豪门，这里的上流贵族们十分不屑，觉得幽兰苑跟自己完全不是一个档次的。
<br/>
<br/>        但是刚才福伯那凛冽的一眼，绝对是浸淫世阀豪门之家几十年才会沉淀出的气场和魄力，大家头皮一麻，迅速闭嘴了。
<br/>
<br/>        大家也很奇怪，一个幽兰苑的鬼夫老头竟然有这样的魄力。
<br/>
<br/>        这时就听福伯对着夏夕绾恭恭敬敬的叫了一声少奶奶。
<br/>
<br/>        不知道是谁抽吸了一声，
<br/>
<br/>        你听到了没，这个老头叫夏夕绾…少奶奶！
<br/>
<br/>        什么意思，他不是夏夕绾的…老公？
<br/>
<br/>        这怎么回事啊？
<br/>
<br/>        夏振国，李玉兰和夏妍妍都面色大变，他们对福伯是夏夕绾老公的事情从来没有起疑过，现在有点晴天霹雳。
<br/>
<br/>        李玉兰迅速出声道，“夏夕绾，这个老头叫你什么？我们去过幽兰苑，明明就看到这个老头在里面，他就是你的老公，你是不是跟这个老头串通了来骗我们？”
<br/>
<br/>        夏妍妍的脸色变幻未定，她也出声道，“绾绾，你嫁给了一个老伯已经是既定的事实了，你现在实在是没有必要来骗人了。”
<br/>
<br/>        夏夕绾勾了勾红唇，她走上前，一双澄亮的翦瞳淡淡的逡巡了一下众人，最后看着夏妍妍笑道，“怎么，出现在幽兰苑里的人必须是我老公么，难道就不能是…管家？”
<br/>
<br/>        福伯恭敬的站在夏夕绾的身后，“我家少奶奶说的没错，我是幽兰苑的管家。”
<br/>
<br/>        管家？
<br/>
<br/>        天哪！
<br/>
<br/>        夏妍妍完全没有料到这一场戏竟然来了一个惊天大反转，这个老头不是夏夕绾的老公，而是管家！
<br/>
<br/>        闹了半天，这只不过是一场乌龙。
<br/>
<br/>        “妈，这怎么回事，你不是说他就是夏夕绾的老公么？”夏妍妍暗中看了李玉兰一眼。
<br/>
<br/>        李玉兰往后退了两步，她也慌了神，“这…这…”
<br/>
<br/>        夏夕绾看着李玉兰，“真是抱歉呢，这件事让你们失望了，但不是老公就不是老公，难不成你们还想强行按头我老公？”
<br/>
<br/>        大家纷纷窃窃私语，“这究竟怎么回事啊，微博都热搜了好几天，原来这个老头只是管家，那夏夕绾的老公究竟是谁啊？”
<br/>
<br/>        李玉兰迅速意识到自己可能是上当了，这个夏夕绾故意误导了她，太可恶了！
<br/>
<br/>        “夏夕绾，这个老头只是你的管家，那你的老公是谁，他怎么没有过来给你庆生？”李玉兰迅速反客为主的逼问道。
<br/>
<br/>        夏夕绾今天一身裸粉色的薄纱长裙，她本来就肤若凝脂，现在被裸粉色衬的更加莹玉粉嫩，层层纱摆逶迤的垂在红地毯上，气质纤尘而绝丽。
<br/>
<br/>        被这么多人对峙，她从容不迫，精致如墨画的眉眼里始终盛着一弯澄亮的笑意，“看来你们对我这位老公都很感兴趣嘛，别急，他很快就会来。”
<br/>
<br/>        夏夕绾真正的老公很快就会来了。
<br/>
<br/>        看着夏夕绾淡定聪慧的样子，夏妍妍的心里突然有了一种很不祥的预感，虽然她暂时也说不清这种预感是什么。
<br/>
<br/>        总之今天的计划有点失控，夏夕绾没有嫁给一个老头，那她真正的老公又是谁？
