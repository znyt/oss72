　　释信无道：“也许这正是道风的意图呢？他是茅山弃徒，不方便再过问人间之事，用这个法子，或是能让法术公会主动找他的麻烦……”
<br/>　　静慧师太道：“不知道道风顶不顶得住。”
<br/>　　张无生道：“这就不是我们操心的事了，有道风的风之谷横一杠子，也是好事，我等只好好筹备龙华会便是。”
<br/>　　静慧师太道：“张师兄，我一直不明白，你为什么要召开这次龙华会，本来大家以为是法术公会幕后操控，但并不是，你何以主动发起龙华会，给法术公会有机可趁？”
<br/>　　张无生沉默了一下，道：“天劫已至，虽然暂时对人间未有大的影响，但人间法术界也不能独善其身，必须做好准备，我们这些老家伙是不能指望的了，这一场战争，必然要有更年轻的法师出来主持，带领法术界保持走在正正轨上。”
<br/>　　释信无颔首道：“任重而道远，非一般人所能担当。”
<br/>　　“正是如此。”张无生叹了口气，“本来叶少阳是第一人选，可惜他现在失踪，生死未卜，我们却是等不得了。”
<br/>　　王道乾插了一句：“可是这样一来，最后的赢家一定是法术公会，二代弟子之中，我们已经没有能跟星月奴七大弟子抗衡的人……”
<br/>　　释信无想到什么，问道：“那七大弟子，谁接触过，都是何等样人？”
<br/>　　张无生哼了一声，道：“都是悦然这般狂放之徒，其实我最担心的并非什么门户之见，而是这些人虽然来自人间，但从小就在轩辕山长大，他们的理念都与我们不同，说的难听一点，他们大多看不上我们人间法术界，让他们来执掌法术界，早晚要出事的。”
<br/>　　大伙缓缓点头，表现出了跟张无生一样的担忧。
<br/>　　慈心师太这时候说道：“我只是插一句，并没有别的意思，之前那个悦然，跟叶少阳不是一类人吗，为何你们从前能忍受让叶少阳做这首席弟子？”
<br/>　　张无生瞥了她一眼，“怎么能是一类人？”
<br/>　　“一样狂放自大，目中无人。”
<br/>　　张无生轻轻摇头，“叶少阳叛逆是真的，狂放不羁，从来不遵守礼法，但他绝没有目中无人，相反，他内心很善良，坚韧不屈，认为该坚守的东西，就一定会坚守到底，这是他的道。那个什么悦然，看上去跟叶少阳很像，不过他才是真正的狂放自大，跟叶少阳比……他拿什么跟叶少阳比，连皮毛都不如……”
<br/>　　一席话说的大家都沉默不语。
<br/>　　张无生叹了口气。“天道造作。没想到会发生这么多事，若是可以，我倒真希望让叶少阳来执掌法术界，也好过便宜法术公会。”
<br/>　　大家仍旧沉默。
<br/>　　道风一路前往龙虎山。
<br/>　　他现在非人非鬼，不用坐车，但也没有影视剧里那种瞬间移动的本事——御空飞行是要耗费修为的。像道风这样的，想到在人间移动到某个地方，最好的方法就是先去鬼域，寻找虚空裂缝。不同的虚空裂缝，通往人间不同的位置。
<br/>　　鬼域之中，有数不清的虚空裂缝，修为深厚的，也可以自己破开虚空，往来阴阳两界。如果人间有人召唤，被鬼魂感知到，便可以循着方向，找到附近的虚空裂缝，很快就会来到人间。这也是为什么人间法师只要作法，不管身在什么位置，总能把对应的鬼魂从阴司召来的缘故。
<br/>　　通过虚空裂缝穿越两界，等于是不同纬度之间的穿梭，不能用人间的经验来衡量。
<br/>　　道风先回到鬼域，找到龙虎山上的虚空裂缝，穿越过去……
<br/>　　深夜，月光照在这座拥有千年历史的名山上。
<br/>　　龙虎山，道门中最为古老也是声名显赫的一座山峰，山上古朴的道观，每一片瓦片都记载着这千年历史的荣光。
<br/>　　道风一路上山，穿过上清宫的时候，看到几个小道士在道场上演练道术，一个年纪大一点的在边上指导。
<br/>　　几个人都汗如雨下，练习得十分刻苦。道风从边上的树林里走过，没有惊动他们，走过去的时候，听见几个笑道的抱怨。
<br/>　　“师兄，我们每天这么苦练，真的有一天能当上天师吗？”
<br/>　　“当然了，不苦练怎么能行，你们看看茅山，虽然只有叶少阳一个弟子，但比我们所有人都厉害，我们龙虎山本来跟茅山齐名的，现在都被压下去了，我们一定要好生修行，争取有一天压过叶少阳，把属于龙虎山的荣光找回来！”
<br/>　　茅山与龙虎山，是道门南北二宗之中声名最大、地位最高的两大宗派，经常会被人放在一起比较，这些小道士为自己的山门抱不平，也是理所当然。
<br/>　　一个小道士弱弱地说道：“我听说，叶少阳那是天才，道风也是天才，我们这些凡夫俗子，就算练一百年也赶不上吧……”
<br/>　　那个师兄兜头给了他一巴掌，“苦练不一定赶得上，但是如果不练，一辈子都赶不上，不试试怎么知道！再说，你怎么知道叶少阳当年没有苦练过，就算再天才，不苦练能有今天吗！”
<br/>　　几个小道士不再言语，师兄又说了一番鼓励的话，令这些人心潮澎湃，几个人都发狠苦练起来。
<br/>　　道风站在树林中，饶有兴致地看着面前的一幕，思绪飞到了十年前，自己指导叶少阳修炼的那些日子。
<br/>　　那个师兄说的其实没有错，大家都知道叶少阳是人间第一天师，是法术界最耀眼的明星，但是很少有人知道，他是怎么一步步走到今天这样的。
<br/>　　“死道风，能不能把我从树上放下来，我我我坚持不住了！”
<br/>　　叶少阳被吊在树枝上，用两只脚尖点着树干，满头大汗地叫着。
<br/>　　道风在树下吐纳，偶尔晃动一下树干，加重叶少阳的负担。
<br/>　　他是一个严师，一开始叶少阳还很听话，长大一点，每天修行结束，都会要求自己带他下山去上两个小时的网，再吃一个卤煮火烧……
<br/>　　想起那段岁月，道风面瘫一般的脸上也是罕见地露出了笑容。
