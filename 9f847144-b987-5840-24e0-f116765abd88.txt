　　第178章
<br/>　　戎族大营内。
<br/>　　杜茂窝在帐篷内，他投靠蛮族后，并未获得足够的礼遇。勉强比奴隶高一级，是和几十个戎族战士挤在一个帐篷内，受人白眼，遭人耻笑。
<br/>　　毕竟对一个叛贼，任何一家势力都不提供尊重。
<br/>　　“我便说了，莫要轻举妄动，本打算在关键时刻，让你显显本事，咱们的地位也能随之提高，可是你却太冲动了。”一个蓬头垢面的年轻人埋怨道，他正是和杜茂一起从蛤蟆寨逃走的孟子云。
<br/>　　杜茂冷哼一声：“投靠蛮族，亏你也想得出来，日后子孙后代都抬不起头做人。”
<br/>　　孟子云不屑的笑笑：“蛮族又如何，帝国又如何，你不过是个恶棍，在帝国又有谁看得起你。叛贼，叛贼又如何，当年帝国开国之君刚起事时，不同样被所有人视作叛贼么？现在一百二十年过去，谁还说李氏是叛贼。”
<br/>　　杜茂顿了顿：“论口才我不如你，我只是为找程大雷报仇，只要杀了程大雷，我绝不在戎族这里多留一天。”
<br/>　　“你可莫要忘了，当初是我救了你。”
<br/>　　“你……”
<br/>　　“于国不忠，于父母不孝，现在这个义字也不要了么？”孟子云冷笑。
<br/>　　咚咚咚。
<br/>　　三通鼓响，营外突然传来骂阵声。
<br/>　　“戎族的渣渣们，你家程爷爷在此！那个活腻歪了，速速出来送死，吃你程爷爷杀个痛快！”
<br/>　　又来！
<br/>　　戎族众人一激灵，随着赤眉离开营寨，只见程大雷单枪匹马，骑着马在那里耀武扬威。
<br/>　　“程爷爷三岁学艺，九岁精通，十三岁天下无敌，拳打南山敬老院，脚踢北海后花园，八十以上的，三岁往下的，没人能吃得了我一拳。”
<br/>　　“本将军由神仙梦中传艺，师父是南极仙翁，天蓬元帅是我师弟，太白金星和我拜把子，本将军一口仙气就能吹得你们魂飞魄散，识相的，赶紧过来让我一斧头一个砍了。”
<br/>　　赤眉一帮人面面相觑。
<br/>　　“他在干什么？”
<br/>　　“好像是要和我们单挑。”
<br/>　　“又来。”赤眉不耐烦的挥挥手：“谁去把他杀了，少让他烦我。”
<br/>　　“我去，我去！”
<br/>　　一片踊跃的声音，本来大家见秦蛮和赵子龙都如此了得，觉得这程大雷既然能做他们的首领本事自然极强。可通过昨日的事情大家已看到，程大雷根本就是个酒囊饭袋。
<br/>　　最后由小山部的忽律雄出战。
<br/>　　“咦，怎么是你？”程大雷顿了顿，一扬手道：“娃娃，回去吧，你还没有资格与我对战，快快让昨日那汉子出来，再与我杀上三……千几个回合！”
<br/>　　忽律雄气得哇哇直叫，还大战三千几个回合，你也不怕累死。
<br/>　　“俺乃小山部头领忽律雄，狗贼去死吧！”
<br/>　　“小山部头领……”程大雷嘴里琢磨了一下，或许抓住他也够了。
<br/>　　忽律雄拍马而来，速度极快，程大雷攥紧斧头，向他脑袋上劈去。
<br/>　　“劈脑袋！”
<br/>　　“掏耳朵！”
<br/>　　“剔牙齿！”
<br/>　　三斧又快又急，招式刁钻古怪，忽律雄一惊，此人果然了得，昨日倒是被他骗了。
<br/>　　程大雷其实也就这三斧头，打完歇菜，他摁了摁腰上的剑，没敢拔剑。忽律雄用一柄大锤，这一剑挥出去，没杀了对方，小心被人家一剑把锤砸折了。
<br/>　　程大雷拨转马头，掉头就逃。
<br/>　　忽律雄本来以为输的是自己，还没弄明白怎么回事，只见程大雷忽然逃了。
<br/>　　他气得七窍生烟，拍马就追。
<br/>　　“狗贼，那里走！”
<br/>　　快马加鞭，追得又快又急，程大雷马术不如对方，眼看就有被追上的危险。
<br/>　　忽律雄越追越快，眼看就要追上程大雷，突然间，程大雷座下马突然提速，生生又和忽律雄拉开了距离。
<br/>　　瘦马的附加技能：逃逸。
<br/>　　眼瞅着就能杀了程大雷，忽律雄怎么肯放弃，口中哇哇大叫，追得更急。
<br/>　　“忽律，回来，莫要追了！”赤眉大叫，眼见忽律雄已到城头弓箭的射程之内。
<br/>　　“秦蛮，子龙，还不动手，更待何时！”程大雷马上大叫。
<br/>　　城门忽然打开，秦蛮、赵子龙、小白狼、高飞虎、高飞豹等山寨的好手同时冲杀出来。秦蛮一个回合就把忽律雄砸到马下，几个人一拥而上，擒住他就往城中走。
<br/>　　“我就知道有诈！”赤眉大手一挥：“冲锋！”
<br/>　　又是一场攻城战，在抛下几百具尸体后，赤眉无功而返。关键是程大雷把忽律雄吊在城头，你们不是喜欢用投石机、弓箭么，那就往自己人身上射。
<br/>　　以彼之道还施彼身，你们会明白，这滋味并不好受。
<br/>　　战斗过后，程大雷出现在城头上。
<br/>　　“赤眉，别废话，把我家林冲放了，咱们一人换一人。”
<br/>　　赤眉咬紧牙关，怪不得这人今日点名要姓林的出战，却原来不是为了杀人，而是为了救人。
<br/>　　小山部在北蛮部势力不小，赤眉还需要小山部的支持，自然，他若不答应放人，小山部的人会立刻反了他。
<br/>　　如此，也正好施恩于小山部。
<br/>　　“忽律头领乃是我从小就敬仰之人，莫说一个奴隶，就是千个万个，我也绝不让忽律头领有损。”
<br/>　　“好啊，那你就拿一千个奴隶和我换吧。”
<br/>　　“……”赤眉。
<br/>　　双方定下交换人质的过程，赤眉那边放林冲，程大雷这边把忽律雄放下城墙，双方兵丁谁也不许向前。
<br/>　　“狡猾的帝国人，我们草原的勇士不像你们一样言而无信！”
<br/>　　的确，戎族虽然野蛮，但狡猾这个字眼往往是用来形容帝国，和他们无关。
<br/>　　交接过程走得很顺利，戎族的确没有趁机放冷箭，而程大雷这边也没有，关键是没找到合适机会。
<br/>　　程大雷把林冲用绳索吊上城头，他一副身躯早已伤痕累累，可见在戎族手中遭受了怎样虐待。
<br/>　　据林冲所言，他是林少羽的叔父，林少羽一身本事都是他传授。这次是因林少羽离家出走，特来北地寻找，却没想到落在戎族手中。
<br/>　　急忙将他交给刘悲医治，看着他身上被折磨的伤口，程大雷不禁有些唏嘘。
<br/>　　水浒中的林冲就是一生命运多舛，老婆老婆死了，官位官位丢了，兄弟兄弟背叛他，最后反而做了自己最讨厌的山贼。
<br/>　　而今日的林冲，显然经历也很悲催。
<br/>　　是个苦逼的英雄呐。
<br/>　　嘟，正义的少年喔，在你的用心经营下，山寨蒸蒸日上，条件达成，恭喜你将山寨升到三级。
