　　“呵呵！”
<br/>　　苏莫话音一落，太苍便轻笑了起来，脸上带着玩味的笑容，道：“你可真够狂的，看来你自认为能抗衡我们？”
<br/>　　“一群土鸡瓦狗，放马过来吧！”司空炎同样踏出一步，朗声说道。
<br/>　　众人闻言，顿时脸色一沉，个个对司空炎怒目而视，此人太狂了。
<br/>　　“司空炎，等下我来对付他们，你负责抄后路，然后抢夺宝物！”苏莫意念一动，向司空炎传音说道。
<br/>　　“嗯！”司空炎面色不变，打败这些人容易，困难的是，在不暴露太多的情况之下抢夺所有人，那就有些困难了。
<br/>　　“真的是不知死活！”太苍的眸中，闪过一道犀利的杀机，随即他转头看了一眼凤晴。
<br/>　　凤晴面色清冷，微微点了点头，他们已经商量好了对策。
<br/>　　“一起出手！”太苍暴喝一声，手中出现了一柄大刀，大刀一挥，暴斩而出。
<br/>　　一道夺目的刀芒，如同犀利的闪电，裹挟着惊天的杀机，向苏莫和司空炎斩去。
<br/>　　“出手！”
<br/>　　“灭了他们！”
<br/>　　“杀！”
<br/>　　众人纷纷怒喝，各式各样的攻击，纷纷施展而出，如同一股洪流。
<br/>　　不过，由于人数太多，通道中空间有限，这百余道攻击，还未临近苏莫和司空炎，便互相接触到一起，引起了猛烈的爆炸。
<br/>　　轰！轰！轰隆隆！！
<br/>　　巨响震天，通道剧震，剧烈的暴炸让得百余道攻击，化为了如洪水一般的冲击波，向苏莫和司空炎杀去。
<br/>　　虽然百余道攻击，没有接触到苏莫和司空炎，便彻底爆开，但是，其爆发的冲击波威力，却是更强了一分。
<br/>　　“行动！”苏莫向司空炎喊了一句，便脚下一踏，迎着洪流一般的冲击波冲了上去。
<br/>　　“给我开！”
<br/>　　苏莫怒喝一声，一剑斩出，气势一往无前。
<br/>　　这一剑，只是单纯的一剑，不是吞天噬地，也不是混沌剑轮，只是五行源力配合上一部分血脉力量，斩出的普通一剑。
<br/>　　苏莫不指望这一剑能有什么效果，他靠的也不是蛮力，毕竟，自己不能暴露太多。
<br/>　　而司空炎，则是身形一闪，从旁边的通道离去，他要从另一处通道，绕到众人的后方，才能将所有人一网打尽。
<br/>　　他并不担心苏莫，因为他清楚，苏莫拥有无敌的防御，同阶之中，就是一群万古天骄也不可能伤到苏莫。
<br/>　　轰！
<br/>　　闪耀的五行剑气，重重的斩击在攻击洪流之上，顿时奔溃开来。
<br/>　　强大的攻击洪流，只是微微一顿，溃散了一小部分，便继续碾压而来，瞬间将苏莫淹没了。
<br/>　　“这……不会死了吧？”
<br/>　　“这么强大的攻击，若是不死那才是怪了！”
<br/>　　“死了就太可惜了！”
<br/>　　迷宫城之外，所有看到这一幕之人，都不由得心中一紧，担心苏莫被灭杀。
<br/>　　毕竟，百余人联手，如此可怕的攻击，同阶之中，就算是擅长防御的万古天骄，也很可能直接陨落。
<br/>　　很多人甚至认为，苏莫被冲击波洪流湮灭，肯定会渣都不剩。
<br/>　　就连龙洌、龙城都是心中一紧，这么厉害的奴仆，要是死了那就太可惜。
<br/>　　虽然这是龙腾的奴仆，但不代表他们以后得不到。
<br/>　　只有龙双双心中惊疑，因为她发现，即便是此刻，龙腾依旧胜券在握的模样。
<br/>　　高空之中，两位龙族的中年执事，此刻都不禁心中一紧，有些担心苏莫陨落。
<br/>　　毕竟，如此天才人物，若是陨落了那就太可惜了。
<br/>　　“哈哈哈！看你还不死？”通道之中，巫铨朗声大笑，如同可怕的冲击将苏莫淹没，他不信苏莫不死。
<br/>　　这种威力的冲击波，就算是二品中级的武者，都是必死无疑，更遑论一品低级了。
<br/>　　“不过如此！”太苍冷笑一声，心中大定，此人的实力并没有那么夸张。
<br/>　　凤晴倒是没有得以，面色平静，眼眸紧紧的盯着冲击波。
<br/>　　下一刻，巫铨的朗笑声、太苍的冷笑声，顿时戛然而止。
<br/>　　只见苏莫，居然冲过了冲击波，向众人冲了过来，速度奇快无比。
<br/>　　“怎么可能？”太苍和巫铨等人，见到此情此景，顿时心中剧震，个个瞪圆了眼眸。
<br/>　　虽然此刻的苏莫，披头散发，貌似非常的狼狈，但是身上的气息无比的强大，没有丝毫的伤势。
<br/>　　这种众人震惊不已，这样都不死，而且丝毫无伤，完全不可思议。
<br/>　　咻！
<br/>　　苏莫的身形，如同是利剑洞穿长空，速度快到极致，瞬间就临近了众人。
<br/>　　“灭了他！”太苍怒喝，便准备再次出手，然而一切都晚了。
<br/>　　只见苏莫的身上，突然冲起了绿色的光芒，照耀了整条通道，将所有人都笼罩在了其内。
<br/>　　天梦元神！
<br/>　　苏莫动用了天梦元神，这算是此刻，唯一能动用的元神了。
<br/>　　一瞬间，所有人都面色一呆，纷纷被拉近了梦境之中。
<br/>　　苏莫的身形，同样停了下来，用天梦元神对付这么多人，他亦是全身心的沉入了梦境之内。
<br/>　　迷宫城之外，瞬间雅雀无声，除了龙腾之外，其他所有人俱是没有想到，苏莫不仅没死，还开始了反击。
<br/>　　“这是天梦元神，这个家伙居然有这种特殊的元神！”有人惊呼出声，瞬间就认出了天梦元神。
<br/>　　这种元神虽然不是无比逆天，但是，却是最特殊的元神之一，诡异而强大。
<br/>　　众人恍然，原来是天梦元神，这个屠霄是要在梦境之中，击败所有人吗？还是牵制住所有人，让炎空出手？
<br/>　　在所有的关注之人，苏莫和太苍等百余人，彻底陷入了石化之中。
<br/>　　而司空炎，绕了一个圈子，很快就要绕到众人的身后了。
<br/>　　嗖！
<br/>　　司空炎速度极快，转过几条通道，便来到了太苍等人的后方，看到了石化的众人。
<br/>　　“坚持住！”司空炎看了苏莫一眼，见苏莫脸色有些苍白，立刻开始行动，收取众人身上的宝物。
<br/>　　他没有直接攻击这些人，因为，强大的攻击，很可能还未攻击到众人的身上，庞大的气势就会将所有人惊醒。
<br/>　　毕竟，天梦元神的梦境，是要靠苏莫的精神力支撑的，苏莫一次性将这枚多人拉近梦境之中，承受的压力可想而知。
<br/>　　他知道，苏莫坚持不了多久，即便苏莫精神力超级强大，也坚持不了多久。
<br/>　　精神力的支撑不够，这些人很容易从梦境中挣脱。
<br/>　　唰唰唰！
<br/>　　司空炎的速度非常之快，如同一阵风一般，在人群之中闪掠，将众人的储物宝器，甚至是手上的兵器，都全部收取了，几乎是一件不留。
<br/>　　这一切，在两个呼吸的时间内，便全部完成了。
<br/>　　迷宫城之外的人，全部都傻眼了，居然还能这样干？简直是不费吹灰之力，便抢夺了所有人。
<br/>　　“淘汰吧！”
<br/>　　收取了所有人的宝器之后，司空炎来到了苏莫的身前，怒喝一声，双拳齐出，两道浩大的拳茫，如同两轮燃烧的烈日，向众人猛烈的轰击而去。
<br/>　　轰隆隆!~
<br/>　　拳茫威势惊天，裹挟着恐怖的火焰之力，瞬间就杀到了众人的身前。
<br/>　　“什么？”
<br/>　　“不好！”
<br/>　　“怎么可能？”
<br/>　　众人纷纷惊醒，顿时面色狂变，他们适才在梦境之中，正在努力挣扎出梦境，已经快要成功了，但突然就脱离了梦境。
<br/>　　现在，还未完全反应过来，攻击就要临身了，电光火石之间根本无法反击。
