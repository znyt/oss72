　　万界山？
<br/>　　苏莫和姜神夜闻言一怔，这是什么地方？他们从来没有听闻过。
<br/>　　就在两人怔神之间，一道挺拔的身影突兀的出现在了他们的面前。
<br/>　　这是一位中年男子，年约四十岁许，面目白皙，留着一缕短须，其穿着一身道袍，显得仙风道骨，气质不凡。
<br/>　　“见过前辈！”
<br/>　　“见过前辈！”
<br/>　　苏莫和姜神夜不敢怠慢，立刻抱拳行礼，此人虽然看似平平淡淡，没有什么气势，但越是这样的人，就越是恐怖。
<br/>　　“你们跟本道来吧！”
<br/>　　中年男子淡淡的点了点头，随即抬脚向山腰走去。
<br/>　　苏莫和姜神夜对视一眼，然后便跟着中年男子而去。
<br/>　　这万界山上，植被也有不少，均是奇花异草，景色绚丽，唯美异常。
<br/>　　有些西瓜大小的花朵，居然在不停的变换着颜色，呼吸之间便变换了十几种颜色。
<br/>　　有些树木的枝干，居然是一条条的手臂，而且这些树木居然还能自行走动。
<br/>　　“成精了！”
<br/>　　苏莫看的暗暗震惊，这些树木应该是树妖吧，吸收天地精气成妖，拥有了灵智。
<br/>　　除此之外，这庞大的万界山上还有大量的建筑，宫殿、楼宇、城堡，多不胜数。
<br/>　　只不过，此地好像并没有多少人气，苏莫眼眸扫视四方，除了他们三人之外，山上再无一人。
<br/>　　很快，三人便来到了山腰处，一片不大的广场之上。
<br/>　　广场上耸立着一座巨大的人型雕像，雕像高达百丈，全身金光刺目，仿佛是由纯金打造的一般。
<br/>　　此雕像屹立在广场上，背负着双手，头颅微微抬起，目光仰望天穹，给人一种欲与天比高的的气概。
<br/>　　磅礴，大气、唯我独尊！
<br/>　　不过，雕像的面目极为模糊，让人看不出其真正的面容。
<br/>　　“苏莫、姜神夜，你们两人天赋不凡，可愿意加入我万界山？”中年人目光望着苏莫和姜神夜，面带淡笑的问道。
<br/>　　苏莫和姜神夜闻言讶然，果然，和他们猜测的不错，真圣界果然是在选拔天才。
<br/>　　他们两人天资最好，被这万界山选中了！
<br/>　　姜神夜略作沉吟，抱拳问道：“敢问前辈如何称呼？”
<br/>　　“呵呵，本道道号真贤子，人称真贤道圣！”中年人微微一笑。
<br/>　　道圣？
<br/>　　姜神夜和苏莫闻言一惊，这个称号，说明此人是一名武圣境强者。
<br/>　　“真贤前辈，你想让我们加入万界山，总要告诉我们万界山是什么势力吧！”姜神夜说道，他虽然对这个万界山颇为感兴趣，但并不想加入。
<br/>　　他所在的三神宗，乃是古灵星河最强大的势力，他在宗门内是最顶尖的天才，受到最好的培养，怎么可能轻易加入其它的势力！
<br/>　　“也好，本道就为你们讲解一番！”真贤道圣微微颌首，他也知道不为两人讲解一番，两人不可能加入。
<br/>　　少倾，真贤道圣徐徐说道：“万界山，是一个特殊的势力，非妖孽不可入，招式弟子的标准，最低都要有双生战魂，天赋和战力都要远超常人！”
<br/>　　“最低都要有双生战魂？”
<br/>　　姜神夜和苏莫闻言，顿时倒吸了一口凉气，这招收弟子的标准未免太严苛了吧！
<br/>　　在古灵星河，任何一个超级势力，门下拥有双生战魂的弟子，也是屈指可数吧！
<br/>　　而这只是最低的标准，若是这样的话，万界山能招收的弟子实在是少之又少。
<br/>　　这是真正的万里挑一，而且是从超级天才之中万里挑一。
<br/>　　据姜神夜所知，他们三神宗弟子三百余万，天才如云，天骄如雨，但拥有双生战魂的天才，也才区区五十余人。
<br/>　　“就算有双生战魂，也不一定有资格加入万界山；当然了，也有例外之处，若是有非常强大的特殊体质，就算没有双生战魂，也能达到万界山的标准。”真贤道圣说道。
<br/>　　“前辈，这万界山在哪里？具体是什么势力？”苏莫沉声问道，他不关心万界山招式弟子的标准，只关心这个势力的具体情况。
<br/>　　“万界山无处不在！”
<br/>　　真贤道圣微微一笑，手指指向山峰上方，那无数光芒闪耀的虫洞，道：“那里可以通向四大星域的任何一个星河，同时在任何一个星河都可以来到万界山，而真圣界的真圣碑就是入口。”
<br/>　　嘶！
<br/>　　苏莫和姜神夜暗暗心惊，这万界山居然拥有如此通天的能力，未免太过不可思议了吧！
<br/>　　据他们所知，就算是最强大的星空传送大阵，都只能在星河内的星辰之间传送，而不能跨星河传送。
<br/>　　比如古灵星河的武者，想要去往其它星河，只有靠自己飞行，飞到其它星河之中。
<br/>　　所以，一般的武帝境武者，一辈子都不可能离开一个星河，只有拥有特殊的飞行宝器，或者是强大的武尊境以上武者，才能考虑跨越星河。
<br/>　　不然的话，从一个星河飞到另一个星河，不说期间的玄力消耗，百年时间乃至数百年时间，都要耗费在路上了。
<br/>　　所以，一般情况下，只有武圣境的武者，才会随意的离开一个星河，就算是一般的武尊，都不会轻易飞跃一个星河。
<br/>　　但这万界山的虫洞，居然能通向四大星域的任何一个星河，可以说是通向了整个宇宙的任何一个地方，这怎能不让人震惊。
<br/>　　这么说来，这万界山是在整个宇宙中招收弟子了，这实在是很惊人。
<br/>　　“我们万界山，是隐藏在宇宙中的势力，不为外界所知，所以你们没有听说过很正常。”
<br/>　　真贤道人面带微笑，继续道：“你们两人符合我万界山招收弟子的标准，只要愿意加入万界山，将能得到想象不到的好处！”
<br/>　　苏莫和姜神夜闻言，互相对视了一眼，这万界山貌似非常不简单，背景非常的可怕。
<br/>　　但是他们了解的太少了，不可能凭借对方几句话，就贸然加入。
<br/>　　“前辈，能有什么好处？”苏莫好奇的问道。
<br/>　　“不可说！”真贤道圣微微摇了摇头。
<br/>　　苏莫无语，你既然要我们加入，却又不说出具体的好处！
<br/>　　“前辈，我们都有自己的宗门，如何能方便加入？”姜神夜谨慎的问道，有好处他自然不想拒绝，但也要问清楚情况。
<br/>　　若是加入万界山，必须退出原来的宗门，那他是绝对不可能加入的。
<br/>(本章完)
