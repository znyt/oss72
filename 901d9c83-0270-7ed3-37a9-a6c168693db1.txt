　　四六长老瞬间出手，打了慕容槐和慕容蝎一个触手不及，将两人震出议事厅外。
<br/>　　议事厅内还有许多没有恢复实力的执事，先天武者的战斗很容易波及并伤到他们，两位长老自然不会让战斗在议事厅中进行。
<br/>　　至于萧逸，刚才一招‘噬火百刃’秒杀了一个慕容家执事，顿时震住了另外七人。
<br/>　　但慕容家的三位精英执事率先从震慑中反应过来。
<br/>　　其中一个后天八重的执事一刀劈出。
<br/>　　萧逸轻轻一挡，反手一掌将他震退。
<br/>　　“好强，这小子是后天八重。”慕容家执事吃了一惊。
<br/>　　“后天八重？这小子半月前也不过后天一二重吧。”
<br/>　　“一定是妖血炎心果的功效，该死，那等珍稀的天材地宝本应是属于千军公子的。”
<br/>　　“宰了这小子。”领头的后天九重执事大喝一声。
<br/>　　七人立刻出手，并包围了萧逸。
<br/>　　“呵。”萧逸冷笑一声，“珍贵的天材地宝就该属于慕容千军？你们慕容家的人，还真是不要脸。”
<br/>　　战斗再次打响，萧逸力敌七人，却不落下风，仅仅有一些吃力。
<br/>　　“这小子怎么回事，我一身后天九重的修为，再加上其余六人相助，竟压不住他？”领头的执事面露惊疑。
<br/>　　“噬火百刃。”萧逸手中火剑一边挥扬，一边施展武技。
<br/>　　“小心。”一个后天八重的精英执事喝了一声，想起之前萧逸秒杀一个普通执事，当即挡下这道火刃。
<br/>　　但他这一挡，却直接口吐鲜血，被震出十数步才站稳身子。
<br/>　　同为后天八重，萧逸完全碾压这个执事，就是因为玄阶武技足够强大。
<br/>　　“哼，小子，别以为只有你才有玄阶武技。”三个精英执事面容冰冷，同时释放武技，“碧波叠浪掌。”
<br/>　　三人不愧是精英执事，施展的碧波叠浪掌比之慕容千军还要强悍几分。重叠的波涛一浪接一浪打向萧逸，威势更猛，威力也强得多。
<br/>　　“崩山斩。”萧逸手持火剑，狠狠一劈，一股滔天的气势如崩山裂石般，一剑将所有波浪劈碎，散作一地水花。
<br/>　　崩山斩乃玄阶中级武技，比之玄阶低级武技碧波叠浪掌要强得多。
<br/>　　“瞪瞪瞪...”层层波涛虽被击碎，但萧逸也被震退了三步。
<br/>　　三个精英执事大喜，“我们三人联手，加之玄阶武技，足可压制这小子，其他执事看准时机出手，一定要将萧逸击杀。”
<br/>　　三人见刚刚的对战取了上风，面露喜意。
<br/>　　但，萧逸却玩味一笑，自语道，“原来我的实力在这个水准，那么，也该结束战斗了。”
<br/>　　“什么？”三人听到这话语，顿时惊怒交加，“混账，你刚才在拿我们练手？”
<br/>　　“哼，不知天高地厚的小子，不过是在虚张声势罢了，我们上。”七人再次联手，同时已经做好了施展武技的准备。
<br/>　　萧逸站在原地，丝毫没有将七人的攻击放在眼里，暗道，“时间紧迫，也不知家族那边怎样，得快些结束战斗才是。”
<br/>　　“升龙。”萧逸低喝一声，身上爆发出一股恐怖气势。
<br/>　　“好强的气势，这小子怎么回事？竟比我这个后天九重的气势好要强上许多？”慕容家执事脸色一变。
<br/>　　下一秒，萧逸凭空消失在原地。
<br/>　　当他再次出现时，已经在一个精英执事身后，火剑轻轻一挥，划出一道漂亮的弧度。
<br/>　　“嗤”，精英执事脖子上，忽然出现了一条血痕。下一秒，喉咙中鲜血如水般喷出，不多时无力地跌倒在地，成为了一具尸体。
<br/>　　萧逸没有管这道尸体，身体再次凭空消失。
<br/>　　嗖、嗖、嗖、、、
<br/>　　慕容家执事们只看到一道火红身影不断消失出现，眼睛完全跟不上萧逸的速度。
<br/>　　但萧逸的每一次出现，其中一个执事脖子上都会多出一条火焰血线。
<br/>　　七次消失出现，七道血痕。
<br/>　　当萧逸停止快速移动，站定身子时，七个慕容家执事，轰然倒地。
<br/>　　那七条喷射而出的水柱，染红了议事厅的地面。
<br/>　　七个后天武者，其中一个还是后天九重，两个后天八重，竟在萧逸手上数秒时间都支撑不住，完全秒杀。
<br/>　　“好...好强。”那些还被黑墨曼陀罗压制实力的萧家执事愣愣地看着，面露惊骇。
<br/>　　萧逸转过身，快速往议事厅门口而去。
<br/>　　外头，一声声巨大的轰鸣声传来。四六长老与慕容槐与慕容蝎的战斗打得热火朝天，矿脉内的树木、建筑被打得破烂不堪，一片狼藉。
<br/>　　先天武者的战斗，果然不同凡响。
<br/>　　“嗯？是那小废物。”慕容槐一眼看到了站在议事厅门口的萧逸。
<br/>　　“他怎么可能有命走出来。”慕容蝎面露惊疑。
<br/>　　四大先天武者缠斗着，但谁也无法占得上风，战斗处于僵持。
<br/>　　四长老一掌震开慕容槐，问道，“萧逸，慕容家那群杂碎如何了？”
<br/>　　萧逸高声道，“已全部被我击杀。”
<br/>　　“好，很好。”四六长老大笑一声，对慕容槐和慕容蝎的攻势更加凶猛。
<br/>　　“混蛋。”慕容蝎愤怒地喝道，“萧逸，你竟杀我们慕容家执事，此仇不共戴天，我必取你小命。”
<br/>　　慕容槐和慕容蝎其实早就猜到了，萧逸能安然无恙地走出议事厅，这便证明，他们的执事已经成为八具尸体了。
<br/>　　“杀我？就看你们有没有那个本事了。”萧逸冷笑一声，手持火剑，脚步一跃而上，朝慕容槐和慕容蝎攻去。
<br/>　　“崩山斩。”萧逸大喝一声。
<br/>　　慕容槐正眼都不瞧萧逸，只顾着和四长老缠斗，微微感受了下萧逸的气息，心中不屑道，“区区一个后天八重，老夫一根手指就能捏死。”
<br/>　　慕容槐心头不屑着，身上猛地爆发出一股气势。
<br/>　　萧逸的火剑劈来，即将劈到慕容槐身上时，却猛地被这股气势封锁，火剑无法再前进一分。
<br/>　　先天与后天的差距是非常大的，一个先天一重，足以秒杀数个后天九重，两者的实力简直是云泥之别。
<br/>　　而慕容槐乃是先天六重武者，甚至不用动手，单凭气势就能将后天九重震成重伤。
<br/>　　“小子，没用的，去死吧。”慕容槐冷笑一声。
<br/>　　“萧逸，快退，你的攻击连他的气势都破不了。”四长老喝了一声。
<br/>　　“哦，是吗？”萧逸没有退，反而冷笑一声，心头暗喝，“升龙”。
<br/>　　在接近慕容槐前，他特地没有使用升龙，就是想让慕容槐放松警惕。
<br/>　　此时，慕容槐感受到萧逸暴涨的气势，顿时脸色一变，“先天一重？怎么可能，这小子明明只是后天八重的修为。”
<br/>　　“崩山斩，破。”萧逸大喝一声，手中的火剑重重劈下。
<br/>　　“不好。”慕容槐反应过来时，已经迟了。
<br/>　　火剑带着崩山裂石之势，破了他身上的气势，而后重重劈下，在他身上劈出一条狰狞的伤口。
<br/>　　“噗。”慕容槐猛地吐出一口鲜血。
<br/>　　四长老见状，立刻趁势攻击。
<br/>　　原本两人胶着的战斗，瞬间变为一面倒。
<br/>　　慕容槐那道狰狞的伤口上，不断溢出鲜血，显然是受了不轻的伤势，根本不是四长老的对手。
<br/>　　慕容蝎大急，立刻打算救援。
<br/>　　六长老冷笑道，“你的对手是我。”
<br/>　　慕容蝎被拦住，无法施以援手。
<br/>　　另一边，慕容槐几乎没有还手之力，四长老一掌一掌打下，慕容槐一口一口鲜血吐出，最后终于伤势不支，被四长老一掌击毙。
<br/>　　临死前，慕容槐颤抖地指着萧逸，“玄阶中级武技，你刚才施展的是玄阶中级武技...紫云城最强的武技不过是玄阶低级，你...你怎么可能拥有...”
<br/>　　话还未说完，他已经无力地倒下，双目瞪得大大的，显然是死不瞑目。
<br/>　　若不是一时大意，被萧逸一剑重伤，他根本不会被四长老杀死。
<br/>　　慕容蝎见慕容槐身死，顿时大急，知晓自己不是四六长老的对手，立马逃遁。
<br/>　　四六长老没有去追。
<br/>　　萧逸疑惑道，“不追吗？”
<br/>　　四长老摇摇头，道，“我们这些先天境的武者很强的，如果不是彼此实力相差太大，对方一心想逃，另一方根本杀不了。”
<br/>　　“也是因为这个原因，刚才慕容槐才丝毫没有警惕。他想着打不过我们还能逃跑呢，谁知道你小子那么猛，一剑就将他重伤了，他连跑的机会都没有就被我杀了。”
<br/>　　萧逸点点头，道，“是慕容槐自己大意了，竟妄图用气势和身体挡我一剑。否则，我不可能将他重伤。”
