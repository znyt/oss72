　　但是在于法师来说，只有道门天师、佛门禅师以上等级的法师，才能够感知到魂之羁绊的存在。
<br/>　　“魂之羁绊！”
<br/>　　张道长又重复了一遍，望着叶少阳，嘴巴缓缓张开，说道：“你怎么可能感知到魂之羁绊的存在？”
<br/>　　叶少阳耸了耸肩，“我之前就说了，我是天师，说了你不信。”
<br/>　　“可你的法力，最多只是个方士。”
<br/>　　叶少阳不想跟他解释，因为根本解释不清，转脸对着刘四，说道：“你家里只有这一个儿子，在生他之前，还有没有生过别的孩子，但是早亡没有养大的？”
<br/>　　刘四夫妻听了这话，当场震惊。刘四睁大眼睛盯着叶少阳，失声道：“你、你你怎么知道？”
<br/>　　叶少阳笑了笑，对他来说，事情是明摆着的：
<br/>　　魂之羁绊，除了父母子女、三世情人之间，兄弟姐妹之间，有时候也会存在魂之羁绊。这婴儿才出生不到一年，自然不可能遇到什么三世情人，父母也健在，既然他感知到那三个恶鬼与婴儿之间有魂之羁绊，那只有一种可能：这几个恶鬼是他的兄弟姐妹。不过前提是死去的。
<br/>　　刘四跟妻子看了一眼，然后磕磕巴巴地讲述起来：原来，在这个儿子之前，两人年轻时候也曾生过孩子，一连生过三个，两男一女，结果无一例外都是养到不到一周岁就死了，当时也找先生看过，说是夫妻二人命犯七煞，注定无子，想要养活一个儿子，只有吃斋烧香，多做善事，积累阴德，十年之后或许有子。
<br/>　　两人照着这么做，果然就在去年老来得子，生下了这么一个宝贝儿子，没想到周岁还没过，就出了这档子邪事。
<br/>　　叶少阳听完之后，看了张道长一眼，说道：“正好三个。”
<br/>　　张道长自然知道是什么意思，脸上有些发烫，缓缓点头。
<br/>　　真相清楚了，那三个纠缠男孩的恶鬼，正是刘四夫妇之前生的那三个孩子死后的阴魂，也就是幼子小顺的哥哥和姐姐，所以他们彼此之间会有魂之羁绊的存在。至于那三个死孩子的魂魄为什么没有去阴司报道，而是留在人间，并且要附在“弟弟”的身上，虽然暂时弄不清楚，但也有迹可循。
<br/>　　叶少阳想了一会，说道：“你把那三个死孩子都埋在哪里了？”
<br/>　　刘四道：“就在俺家老房子外面的石井里了。”
<br/>　　叶少阳皱眉道：“石井里？”
<br/>　　刘四用力点头，说道：“是啊，俺们这的规矩，成年之前的孩子如果早丧，不能进祖茔，也不能立坟头，只能随便找个地方埋了，俺当时觉得省事，就去扔在老房子的石井里，然后用土盖上了。”
<br/>　　叶少阳看了张道长一眼，张道长说道：“石井在哪，快带我们去。”
<br/>　　刘四急忙点头，又看了孩子一眼，想问问孩子怎么办。
<br/>　　张道长也怕那三个恶鬼前来伤害孩子，于是化了符水喂孩子喝了，然后让他们夫妻抱着孩子，跟他们一起过去。
<br/>　　他毕竟也是个真人。叶少阳见他化符水，也没有跟他争抢，走到一脸呆滞的刘老汉面前，冲他笑了笑，“我没骗你，才刚开始呢。”
<br/>　　刘老汉傻傻地看着他。
<br/>　　刘老汉跟刘四夫妻一样，起初他捉鬼失败，加上张道长来了之后说的那些话，让他们都觉得叶少阳只是个初出茅庐的小道士，因为自己的莽撞差一点害死了小顺，同时自己也差点送命。刘老汉更是尴尬地不行，后悔听信叶少阳的话，把他带过来，差点误事。
<br/>　　结果他后面那些分析，本以为张道长会嘲讽和反驳，谁想到张道长竟然相信了，并且配合他的所作所为，这个强烈的反转，不光是他们，连张道长那一干弟子都有点转不过弯来，那几个之前嘲讽过叶少阳的，更是感到自己的脸皮火辣辣地疼。
<br/>　　刘四在前面带路，前往他家的老宅。叶少阳和张道长一起，并肩走在他身后。张道长心中一直对叶少阳能够发现魂之羁绊耿耿于怀，终于忍不住问他。
<br/>　　“我说了我说是天师，你又不信。”叶少阳还是这句话回他。
<br/>　　张道长还想说什么，前面刘四已经来到一间破旧的院子前面，停下脚步，表示前面就是自己家的老宅。
<br/>　　叶少阳等人跟着走了进去，随着刘四穿过几乎倒塌得只剩下四面墙的房屋，来到后院。
<br/>　　叶少阳抬眼看去，后院长着两棵大槐树，枝繁叶茂，几乎盖住了半个院子。
<br/>　　“那口井就在这儿了。”刘四来到两棵树的中间，脚踢着一口井的井沿说道。
<br/>　　叶少阳跟张道长对视了一眼，心中都是倍感无奈，法术界有云，二槐交生，二鬼把门。把自家亲人的尸骨埋在这种地方，真是……想不出怪事都难。
<br/>　　叶少阳走到石井跟前，往下看去，井很深，下面一片漆黑。
<br/>　　“有没有手电？”
<br/>　　叶少阳问完，半天没反应，抬头看去，大家都是一脸茫然地看着他，这才明白自己又说错话了，耸了耸肩，摸出一张灵符，凌空一抖，燃烧起来，丢进了井里。
<br/>　　灵符往下落了有三四米深，熄灭在了泥土上。
<br/>　　“下面没水？”叶少阳问道。
<br/>　　“这口井废了很多年了，早就没水，所以我才会把死掉的孩子埋在这下面。”
<br/>　　叶少阳缓缓点头，说道：“砸开！把尸体挖出来！”
<br/>　　刘四一听就蒙了，转头朝张道长看去。
<br/>　　张道长手拈胡须，沉吟了一下说道：“砸开！”
<br/>　　他跟叶少阳想的是同一件事，那三个恶鬼既然留在人间作祟，一定是跟他们的尸体有关，更不要说尸体都掩埋地这么诡异，肯定要挖出来看看的。而井口太小，当初刘四把尸体扔进井里，肯定不费什么事，但是现在想要再挖出尸体，就不那么容易了。
<br/>　　张道长让刘四回家去拿杨镐和铁锹过来，自己在井口四周打下了四根铜钉，用红线在上面缠绕，布置了一个七星法结，这样做是为了将井口封印住，免得万一那三个恶鬼就在下面，趁着他们挖井的时候暴起伤人。
