　　叶少阳吞了下口水，心想得亏她凶大，不然都卡不住。
<br/>　　“废话什么，快叫出来！”小九二话不说，冲了上去。
<br/>　　女魃反击。
<br/>　　一个是万妖之主，一个是上古尸王，叶少阳都搞不清这俩妹子谁厉害点，感觉上差不多，是一个级别的，一时间难分胜负。
<br/>　　“狐姐姐，叶少阳到底哪里好，能让你一个妖王委身于他，当的还是小老婆，是不是他床上很厉害啊，狐姐姐，不如把他借给我一晚，让妹子我也享受一下？”
<br/>　　擦！叶少阳虽然知道她目的是在干扰小九，但说的这话……还是太那啥了些。叶少阳不想听她污言秽语的，正要冲过去，一扫眼看到鬼车落在自己边上一座岩石上，挠着头说道：“我说你这人倒是聪明了，你是怕被人打死了没人埋吗，坐着棺材到处跑？”
<br/>　　“叶天师，你但去便是。”
<br/>　　他的话，叶少阳当然是不信的，哪敢把后心露给他，犹豫着不动。
<br/>　　摇光仙子这时赶到，也加入了战斗，跟小九夹击女魃。叶少阳则盯着鬼车，盘算着如果抢到了东岳火精，怎么对付右君。
<br/>　　他不知道这次来的是右君本人，还是跟以往一样是一缕精魄的化身，后者不用说，如果是前者……作为太阴山的三号人物，他的实力，自己肯定不是对手……
<br/>　　正思索间，突然身后传来一声呼唤：“少阳！”
<br/>　　叶少阳回头一看，惊的下巴都合不上了。
<br/>　　“冷玉！？”
<br/>　　芮冷玉冲他一笑，落在他身边，不等他开口，说道：“是军师把我从图中世界交出来的，我知道女魃的弱点，让我来对付她！”
<br/>　　“你知道她的弱点？”叶少阳不敢相信。
<br/>　　“是阿呆告诉我的。”
<br/>　　“哦……”
<br/>　　芮冷玉想到他是吃醋了，在他鼻子上刮了一下，对他道：“女魃怕火，你准备好焚天符，我上去帮忙，你瞅准机会贴她！”
<br/>　　说完也冲了上去，加入了战斗。
<br/>　　叶少阳捏了一张焚天符在手中，暗中观察着，芮冷玉没用法术，而是摊开双手，有赤红色的血从掌心和五指渗出，在她手中形成了数道长鞭一样的东西，如灵蛇舞动，朝女魃进攻。
<br/>　　而女魃，也是召唤出了几道血光当做魂器，两人手段竟然差不多。
<br/>　　芮冷玉手上这是……将臣之血？
<br/>　　看到这一幕，叶少阳心情是糟糕的，随后安慰自己，自己一直不是都认为，手段没有善恶之分吗，只看用在什么地方，芮冷玉虽然操控的是将臣之血，但只要她的心还没变，那就没关系……
<br/>　　四个女人，围在一起打斗，而且实力都是不凡，这场面，在三界之内差不多都是罕见的吧。叶少阳一眼扫过去，发现这四个美女，还真是完全都不一样：
<br/>　　摇光仙子，人如其名，气质高贵，如同仙女一般，对男人来说，是真正的女神级别，对这种女人，你只会仰视和爱慕，但一般不会产生什么邪恶的欲望。
<br/>　　小九身为妖王，是邪魅的化身，高高在上不可一世，姿色也是倾国倾城，让弱者不敢仰视、强者想要驾驭。
<br/>　　女魃不管是气质还是装扮，都充满了邪恶的欲望，多看几眼就忍不住吐口水，但又太过危险，撩不起的类型。
<br/>　　芮冷玉……看到她战斗的样子，叶少阳心情复杂起来，以前看她，是优雅与野性的结合，如今，操控着将臣之血的她，看上去又多了一种妖邪的感觉，叶少阳但愿自己是想多了。
<br/>　　“冷玉，你是不是打错人了，你身边这个妖王，抢了你的男人，你不去撕了她，却来找我做什么，我要是你，我一天也不能忍！”
<br/>　　纵然知道她是在挑拨，小九和芮冷玉内心还是忍不住颤了一下，互相看了一眼，小九脸色微微一红，道了声：“见过主母！”
<br/>　　“小九你不要听她胡说，一起灭了她。”芮冷玉不带任何表情。
<br/>　　摇光仙子看看他们三个，笑道：“女魃你有脸说别人，你男人后卿喜欢人家，你不是也忍了！”
<br/>　　一句话说得女魃和芮冷玉脸色都变了。
<br/>　　“贱婢找死！”
<br/>　　女魃人影一闪，一道分身飞起，身边跟着六道血柱，一起砸向摇光仙子，摇光仙子急忙作法，震碎了血光，却被女魃分身击中，惨叫一声飞出去，正好落向叶少阳站着的方向。
<br/>　　叶少阳飞身上前，一个公主抱将她抱在怀里，朝她脸上看去，嘴角已经渗出了血。
<br/>　　“你没事吧？”叶少阳问了声，刚要松开她，摇光仙子突然呻吟了一声，主动勾住了他的脖子，低声道：“你抱着我，容我调息下，你看住女魃，他们很快就赶来了，她逃不掉。”
<br/>　　叶少阳只好抱着她，转头看去，的确有不少人影正在凌空飞来，正要开口让芮冷玉和小九小心女魃逃走，嘴巴刚张开，突然后背一麻，一股黑气，从他胸前贯穿过去，与此同时，摇光仙子的尖叫响起。
<br/>　　叶少阳喉头一甜，喷出一口血。
<br/>　　“少阳！”
<br/>　　小九和芮冷玉发现不对，毫不犹豫地放下女魃，朝叶少阳扑过来。
<br/>　　“是右君干的，快！”摇光仙子从叶少阳怀里跳下来，指向一边的鬼车。
<br/>　　右君冷笑一生，操控鬼车，朝远处飞走了。
<br/>　　“少阳，你怎么样！”小九和芮冷玉一起抱着他，关切之情溢于言表。
<br/>　　叶少阳捂着右//胸，摇了摇头，指着一个方向，道：“快去追女魃！”
<br/>　　两人转头看去，原来就在两人回撤的瞬间，女魃已经跑了。鬼车紧随其后。
<br/>　　“小九你照顾少阳，我去追她！”
<br/>　　芮冷玉说着立刻追了上去。
<br/>　　小九抱着叶少阳，在路边草丛坐下来，撕开满是血污的衣服，检查了一下，当场就要哭了：之前那道黑气，贯穿了叶少阳的身体，在胸前留下了一个血洞，还在汩汩往外冒血。
<br/>　　这时候林三生等人也赶到了，看到这一幕也吓呆了。
<br/>　　叶少阳说到底还是人类，受了这样的伤，他们谁都没有办法。
<br/>　　“快送医院！”瓜瓜大声喊。
