　　此时，慕清霄手中出现一颗乳白色丹药，屈指一弹，便向天鬼皇袭去：“此丹名造化，高九品，生死人，肉白骨，乃是极品疗伤丹药。”
<br/>　　早在古剑奇谭位面的时候，慕清霄就将《朱子率》中的炼丹技术完全掌控，更是成为了九品炼丹师。
<br/>　　炼制造化丹的灵药，自然是靠着天墉城的名声与势力收集而来，乃是真正的九品丹药，九品之上，便是十品丹。
<br/>　　十品，相当于斗破位面的帝品丹药。
<br/>　　造化丹药，生死人，肉白骨，只要伤者有吊着一口气，便能将生命从阎王手中抢过来，治疗天鬼皇的伤势，一枚造化丹绰绰有余，甚至有些浪费。
<br/>　　天鬼皇眼疾手快，挥手就将丹药捏在手中，嗅着丹药传来的清香，想也不想就吞入腹中。
<br/>　　造化丹入腹，天鬼皇伤口中的黑色剑气被逼出，鲜血也停止流淌，剑痕愈合，最终结疤脱落，伤口再生……
<br/>　　转眼间，天鬼皇胸口狰狞的剑痕便消失不见，取而代之的是新生婴儿一样的肌肤。
<br/>　　“多谢阁下，此恩情本座记下了。”
<br/>　　说罢，天鬼皇手中长刀一挥，一道赤红色刀气瞬间将鬼冥吞噬，旋即爆出一团血雾，肉身与灵魂同时泯灭，鬼王陨落。
<br/>　　“鬼冥已死，鬼厄定有所察觉，本座有伤在身，请阁下待会出手将鬼厄击杀，也算是完成了对本座的承诺。”
<br/>　　慕清霄微微点头，既然已经承诺了天鬼皇，他自然会做到。
<br/>　　而且，鬼厄似乎在破境，一旦突破，天鬼皇定然不是其对手，更别说后者此刻有伤在身，哪怕刚才对战鬼冥的时候，也是靠着天鬼一族血脉取巧获胜。
<br/>　　待血雾消散，慕清霄的视线落在鬼王庙的后院深处，那里是一座偌大的宫殿，丝毫不比前院的正殿小。
<br/>　　这时，天鬼皇开口道：“这正殿乃是双煞鬼王兄弟的老巢，其中定然凶险万分，不如让本座带路吧。”
<br/>　　闻言，慕清霄嘴角浮现一抹弧度，摆了摆手，星眸中浮现三颗勾玉，随即缓步向正殿中走去。
<br/>　　“阁下？”
<br/>　　天鬼皇眉头一皱，像他们鬼王级别的，虽一身实力已经强悍如斯，但在自己的住处，反倒会设下许多杀阵。
<br/>　　越是强大，越是怕有别的鬼王觊觎他们的位置。
<br/>　　比如火鬼王为了统治极乐世界，更是利用天然岩浆当作屏障，连阎罗鬼王都暂时拿她没办法。
<br/>　　哪怕是在酆都，他建造的宫殿中，就有不下百种杀阵，若是有修士闯入，绝对是有死无生，不死也要脱一层皮。
<br/>　　他可不相信，双煞鬼王的老巢会不设阵法。
<br/>　　天鬼皇想要开口提醒，但慕清霄却无动于衷，心中不禁有些疑惑。
<br/>　　忽然，想起慕清霄在进入鬼界前所说的话语，后者能一眼看出传送阵四周的阵法，难道他精通阵法不成？
<br/>　　某些庞大的杀阵，哪怕他们鬼王都不敢轻闯，可是慕清霄就这样大摇大摆的进去了，简直就是将安全置于危险当中。
<br/>　　但是想到后者深不可测的实力，天鬼皇眉头皱的更紧，一咬牙快步跟了上去。
<br/>　　然而，跟在慕清霄身后，缓步行走在正殿中，一路下来却安然无恙，根本没有触发任何阵法，仿佛只是行走在普通殿宇中一样。
<br/>　　这令天鬼皇有些困惑，难道这双煞鬼王如此自大，自己的老巢居然不设防备？
<br/>　　这种念头在天鬼皇脑海中一闪即逝，这时，走在前方，慕清霄清朗的声音响起：“这里的阵法确实挺多的，但对我来说，却形同虚设。”
<br/>　　这平淡无比的一句话，让天鬼皇心中一惊，但慕清霄平淡话语中，那股不容置疑的气度，更令他有些恍惚。
<br/>　　这种层次的强者，不屑用假话骗他，更何况他们两现在可是一根绳上的蚂蚱。
<br/>　　想到阵法对慕清霄形同虚设，天鬼皇有些庆幸，庆幸自己做出的决定，这种人物，绝对是他最不想招惹的。
<br/>　　走在前方，慕清霄脚步动作依旧散漫，但行走的路线却蕴含着某种定律与规则。
<br/>　　正殿中确实布下了许多的杀阵，但在拥有血轮眼的慕清霄面前，任何阵法都形同虚设。
<br/>　　星眸中的勾玉散发着妖异的光芒，扫视着正殿中的某些死角，等天鬼皇经过时，终于发现，某些地方有些许的不同。
<br/>　　有的地方摆放着花瓶，有的地方摆放着罗刹鬼神石像，甚至有的地方散发着能量波动。
<br/>　　而前面的慕清霄，似乎根本不需要分辨，也不需要试探，这布满阵法的正殿中，犹如他自家的后花园一样。
<br/>　　其中的阵法，更是被轻易的绕过，连触发的机会都没有。
<br/>　　“嗯？”
<br/>　　就在这时，慕清霄脚步顿住，停在了一座高数仗的黑色罗刹鬼神石像上。
<br/>　　“阁下，为何停下了？”
<br/>　　天鬼皇有些不解，眼前的罗刹鬼神石像与前面的没有任何不同，转身就是另外一条通道，难道这鬼神石像有古怪？
<br/>　　收起血轮眼，慕清霄道：“气息波动就是这里传出的，恐怕就是这里了。”
<br/>　　鬼厄或许用了某种方法，让天鬼皇无法察觉出他的气息波动，但慕清霄的灵魂力量极强，境界更在鬼厄之上，自然能够轻易的找到他的藏身处。
<br/>　　闻言，天鬼皇取出赤红色长刀，毫不犹豫的朝着罗刹鬼神石像劈出。
<br/>　　刀光一闪，罗刹鬼神石像的脸部就出现一丝平整的裂痕，随后裂痕越来越大，伴随着一阵轰鸣声，鬼神石像自中间分裂成了两半。
<br/>　　旋即，一个黝黑的洞口便出现在他们的视线中。
<br/>　　看到黝黑洞口的瞬间，一股黑气席卷而出，黑气化作人形脸庞，狰狞可怖。
<br/>　　天鬼皇目光一凝，刚要抬起长刀，慕清霄星眸中闪过一丝不屑，一抹黑炎横空，伴随着一声尖锐凄厉的惨叫，黑气连带虚空，全都化作虚无。
<br/>　　天鬼皇一惊，根本没看到慕清霄有任何动作，那神秘的黑色火焰究竟是如何出现的？
<br/>　　居然连洞口的空间都焚烧成了虚无，这该有多么的恐怖！
<br/>　　天鬼皇的小心肝一颤一颤，在虚空恢复原样后，慕清霄浅笑一声，道：“快点结束，带我火鬼界。”
