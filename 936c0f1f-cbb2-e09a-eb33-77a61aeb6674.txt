<br/>　　简直是那虚无奇，根本不像是他的父亲一样，反倒是像是他的仇人。
<br/>　　大声的唾骂起来。

<br/>　　楚风眠饶有兴趣的看着虚堇，像是这虚堇，这样的性格的武者，简直他前世都少有见到。

<br/>　　虚堇跪在地上，还是连连求饶到，这哪里像是一尊圣者的样子。
<br/>　　一尊圣者，就算是死，也不会如此讨饶的，这虚堇为了活下去，却是没有任何一丝的尊严了。
<br/>　　这让楚风眠的眼神中，不由的露出了几分厌恶。
<br/>　　“就凭你，当我的狗都不配！”
<br/>　　楚风眠瞥了一眼那虚堇，顿时冷哼一声，一掌直接拍下。
<br/>　　“楚风眠，你好狠！”
<br/>　　感觉着一股灵力，压迫过来，要将她的身躯都给撕碎，虚堇身躯剧烈的颤抖这，大骂起来。
<br/>　　“废话真多！”
<br/>　　楚风眠冷哼一声，一掌的力量加强了几分。
<br/>　　这虚堇，顿时身躯破碎，魂魄都被楚风眠拍碎，直接毁灭。
<br/>　　这样的人，就算是想要臣服楚风眠，也不会答应他。

<br/>　　楚风眠一眼望去，看看刚刚虚无奇灭亡的地方，漂浮这几个空戒。

<br/>　　这黑色木牌上面刻画的无数纹路，也跟北邙圣牌相差不少，不过却是要粗糙的多。

<br/>　　“少主不用担心，这多余的五个，可以去北邙城卖一个好价钱。”
<br/>　　拓跋鼎在一旁，笑着说道。
