　　那时候整个狼牙的特殊晶体数量都不多，想要投入研究需要一定的数量，毕竟前期大量的材料通常都是以失败为结局的报废。
<br/>　　而解毒剂这一想法更是只有楚涵提出的猜想，究竟行不行以及如何让其实现，着实难倒了所有研究部成员。
<br/>　　更别说那时候同时进行的研究，还有等阶提升剂和新人类觉醒剂，相对于解毒剂这一没头没脑的空谈来说，这两项药剂更容易实现，因为楚涵不仅提供了猜想，也列出了最初的方程式。
<br/>　　知道第一步该如何走的方向，接下来的只是一步步的推论和试验，于是这两项药剂的发明与出现，都在进程之内。
<br/>　　但唯独解毒剂，别说方程式了，楚涵连最初的设想都没有，只提供了原材料，还只有特殊晶体一种。
<br/>　　其余的还需要什么，一概不知！
<br/>　　这等同于无解的项目，袁熙叶却在另外两种药剂的紧急调配中，每日都挤出时间来对解毒剂进行单独的研究，甚至这一项目除了她，研究部的其余人连插手都不知道如何开始。
<br/>　　于是这一项目也就因此被隐下，楚涵没再提过，更没有追问过进程。
<br/>　　其余人也渐渐将这件事放下，被归类到了不可能实现的目标。
<br/>　　但直到三个月前狼牙战团出兵前往分界线，南北联盟军的一场大战就要展开的时候，楚涵却收到了来自老妈袁熙叶的一则消息。
<br/>　　那时候丧尸引诱剂，等阶提升剂，新人类觉醒剂，这三大药剂在狼牙已经属于最普遍的资源，研究部的成员调配起来失误率也越来越低。
<br/>　　收到那则消息的楚涵便当机立断，在面对大尸潮的两个月准备期间，让狼牙各部门转移到段氏基地，唯有袁熙叶守着狼牙基地的研究部继续。
<br/>　　他不敢保证研究的成功与否，但这项研究绝对不能停，哪怕整个狼牙倾巢出动抵抗边境的大尸潮，狼牙研究部也要照旧，并且材料无限供应。
<br/>　　以上这些都属于狼牙内部的绝对机密，知道此事的人不超过十人，而当下觉醒剂问世之际，鲁初雪也阴差阳错的第一时间得知。
<br/>　　空间并不大的屋内依旧一片安静，江左没出声，唯有手指带有节奏的拍着水杯，显示出了他此时此刻的心急。
<br/>　　鲁初雪不知过了多久终于回了神，她一个颤栗后深深看了江左一眼，而后便‘刷’的一下子起身，快步跑了出去。
<br/>　　江左一愣，连忙站起来就想追出去。
<br/>　　他不明白这狼牙机密事件刚告诉了这名后勤总管，怎么就不管不顾的跑了？
<br/>　　只是当江左还没来得及跑出太远，鲁初雪在朝着什么人大喊的声音就传了过来：
<br/>　　“回来！清单有变动！所有前往基地传递清单的人都待命，已经出发的给我追上去拦下！”
<br/>　　江左停下脚步，有些惊讶也有些跟不上节奏，他自然知道已经有部分后勤部的人出发前往了那些基地进行物资交接，这是每过一段时间段氏基地与那些基地的常规来往。
<br/>　　但他实在没想到鲁初雪竟然是如此雷厉风行的人，行动派啊！
<br/>　　江左顿时笑容灿烂异常，心想这事应当已经搞定了，便回身走向研究部的所在地，决定坐等原材料提供。
<br/>　　只是两个小时后……
<br/>　　鲁初雪再次出现在与江左争辩的屋内，并且没过几分钟江左便被人带了过来。
<br/>　　江左望着屋内的鲁初雪，总感觉与之前的状态有着千差万别。
<br/>　　鲁初雪朝着江左点了点头，语速极快的开口：“特殊晶体我已经列在需求清单上了，每个基地都发了一份，但由于这东西是稀缺物品，所以我同时也给当下正在南署基地的楚涵上将传去了消息，询问是否可以将此事告知南署联盟军的各将领。”
<br/>　　“毕竟资源是那些将领的基地送来的，这次原材料的供应同样如此。”鲁初雪说着有些口干，快速喝了口茶继续道：“楚涵上将的回复没等到，但等到了何峰的同意，并且他说他的同意可以代表楚涵，只是当下楚涵上将不在旁边，事情又紧急，便由他代理同意。”
<br/>　　“得到许可后，我立即向南部联盟军发去了最快急报，当然由于我知道的也不具体，并且这件事又紧急，我只能先讨要一份有关此事的特权，传过去的也是简报，只有那些将领明白了前因后果，你们研究部的原材料才能得以提供。”
<br/>　　江左听着这么一大段话，好半响才明白过来鲁初雪是在说什么，于是嘴角抽了抽道：“只是一则信任许可，你不用说的这么复杂，而且这件事用得着走这么弯弯绕绕的程序么？”
<br/>　　在江左看来狼牙研究部向来都是将需求清单上交，没多久后勤那里就拉来了他们需要的东西，虽说有很多材料不容易获得，但狼牙的后勤部也会在最快时间内收集再送过去。
<br/>　　不存在什么程序可走，一切都极简。
<br/>　　鲁初雪翻了个白眼：“你们研究部都是被楚涵惯得！要什么有什么！这里可不是狼牙，楚涵上将也忙着打仗没功夫给你们提供便利，他要是在我们当然不用这么麻烦，直接一句话的事连理由都不需要，估计那些将领啥都不知道就将特殊晶体都送过来了！”
<br/>　　“但问题就是楚涵不在这里，你和我没这个特权懂不懂？”鲁初雪敲着桌面道：“正常程序你都嫌麻烦，这才是第一步呢！”
<br/>　　“啊？”江左呆住：“这不都搞定了么，还需要什么？”
<br/>　　“搞定个毛！”鲁初雪再次翻白眼，道：“由于没有楚涵本人的亲自出面，光是一份简报是不够的，原材料会有但数量绝对不可能让你满意，所以接下来……”
<br/>　　说着，鲁初雪手握笔摊开一张白纸后，朝着江左一抬下巴：“说吧，解毒剂的具体参数？放心研究机密我当然不问，我需要知道解毒剂的效果，面向人群，亦或是分类？等阶提升剂有好几种，觉醒剂更配合着训练极限程度，你们研究部的新发明，应该不仅仅是有祛毒效果这么简单吧？”
<br/>(本章完)
