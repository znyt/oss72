　　晚上，
<br/>　　四人准备去学校附近的酒吧去玩。
<br/>　　唐威忽然冲着李虎一脸坏笑道：
<br/>　　“老四，把你的那位大胸妹也叫上呗！”
<br/>　　李虎先是露出一抹羞色，继而怒声道：
<br/>　　“老唐，你特么再敢喊我们家倩倩大胸妹，就别怪我翻脸不认人！”
<br/>　　唐威哈哈一笑，
<br/>　　“好好！我不说还不行吗？不过你们家倩倩那个确实大啊！”
<br/>　　李虎两眼一瞪，
<br/>　　“你还说！”
<br/>　　……
<br/>　　听到两人的对话，叶尘很快便搞清楚了状况，
<br/>　　原来这一个多月，李虎这家伙，居然勾搭上了一个经贸学院的美女，两人如今正处于暧昧期，还尚未确定关系。
<br/>　　叶尘原本也没太当回事，忽然眉头一皱，想起了一些前尘往事，不由得脱口而出，
<br/>　　“徐倩倩？”
<br/>　　原来，
<br/>　　叶尘恍然想起前世，李虎便恋上了一个做徐倩倩的女生，结果却被这个女人百般算计，最后还被她一脚踢开，投入到另外一个公子哥的怀抱，直接导致李虎性情大变，郁郁寡欢了好几年。
<br/>　　若非李虎突然提及这个名字，叶尘还真未必能想起来这个人。
<br/>　　只是让叶尘没有想到的是，如今这一世，两人竟然这么快，就又纠缠在了一起。
<br/>　　或许，这就是命运的力量吧。
<br/>　　李虎哪里知道这些，听到叶尘一口就道出自己心仪女生的名字，顿时为之一呆，旋即笑道：
<br/>　　“尘哥，莫非你也认识倩倩？是通过嫂子认识的吧？既是这样的话，你把嫂子也喊上，刚好可以帮我撮合撮合！”
<br/>　　李虎口中的嫂子，指的自然是萧若曦。
<br/>　　萧若曦也是经贸学院，彼此认识也并不奇怪。
<br/>　　叶尘听到李虎这么说，顿时觉得有些不太容易回答。
<br/>　　他既然知道这个女人心思不纯，又岂会再去撮合他们？
<br/>　　可是，他现在如果实话实话，此时的李虎又肯定不会相信，说不定反而会怨恨自己。
<br/>　　一念及此，叶尘只好随口敷衍了一句，
<br/>　　“若曦不喜欢酒吧这种场合，还是不喊她了吧！”
<br/>　　李虎听到这话，顿时一脸失望，但是见叶尘一脸决然，也不敢再多说什么了。
<br/>　　很快，
<br/>　　李虎躲进卫生间，和那徐倩倩打了一个电话之后，顿时一脸欢喜的走了出来，冲着唐威和陈东，笑道：
<br/>　　“老大，老三，别说哥们不照顾你们，倩倩说了，她把他们宿舍的其他几位美女也带上，你们两个可要好好把握机会！”
<br/>　　唐威听到这话，顿时两眼放光，陈东则挠了挠头，老脸一阵发红。
<br/>　　……
<br/>　　暗魅酒吧，是天海大学附近，最有名的一间酒吧，据说是某位天海大佬开的场子，所以无人胆敢在这里闹事。
<br/>　　四人来到酒吧，
<br/>　　“倩倩，这里！”
<br/>　　李虎冲着四名打扮时尚的女孩高喊了一声。
<br/>　　“还真是她！”
<br/>　　叶尘看到徐倩倩，不由得再次暗暗摇头，心里已经开始寻思着，如何让李虎看清楚这个女人的真面目，省得他再落得前世那样的下场。
<br/>　　四名女孩看到容貌俊逸的叶尘，明显全都美眸微微泛光，
<br/>　　“李虎，这位就是你经常说起的叶尘同学吧？”
<br/>　　“听说你在和我们学院的萧若曦学姐交往，不知道此事是真是假？”
<br/>　　“果然是个大帅哥呢！”
<br/>　　……
<br/>　　几人你一言我一语，对着叶尘议论开来。
<br/>　　叶尘对徐倩倩没有好感，连带着对其他几个女孩，自然也懒得理会，只是微微点了点头，便不再啃声。
<br/>　　徐倩倩等人见叶尘如此高冷，顿时纷纷露出一抹不悦之色。
<br/>　　虽然，叶尘因为和萧若曦交往的事情，在天海大学有着不小的名气，可更多的都是一些负面传闻。
<br/>　　毕竟，萧若曦是天海大学最有名的白富美，广南首富家的千金，而叶尘的真正身份并不为人所知，在萧若曦的那些追求者眼中，他自然只是一只侥幸吃到天鹅肉的癞蛤蟆罢了。
<br/>　　“哼！不过就是长得好看点的小白脸，拽什么拽？”
<br/>　　四名女孩原本还想通过叶尘，结识一下萧若曦，这位经贸学院的风云人物，见叶尘对她们如此冷漠，心中不免有些忿忿。
<br/>　　李虎看气氛有些尴尬，连忙招呼众人进入酒吧。
<br/>　　众人应该都是酒吧里的常客了，一进入酒吧之后，当即便有工作人员迎了上来，将他们迎入一个卡座之中。
<br/>　　李虎一口气点了一大桌，价格不菲的红酒，以及一些果盘、小吃等等。
<br/>　　这里虽然是相对安静的清吧，可是在音乐和灯光的渲染之下，众人很快便进入了状态。
<br/>　　李虎和徐倩倩两人坐在一处，不知道在低声聊着什么，但是从两人的神情中，却看可以看得出来，李虎似乎一直在殷勤讨好，而徐倩倩则一脸傲娇，甚至还隐隐透着一抹轻视。
<br/>　　至于唐威、陈东和其他三名女孩，则在玩着一些酒桌上的游戏，唐威擅长交际，时不时说一些笑话和荤段子，逗得众人时时大笑。
<br/>　　就连陈东也时不时的能插上一句。
<br/>　　唯独叶尘，一个人坐在最角落的位置，自顾自的品着红酒，丝毫没有要参与其中的意思。
<br/>　　唐威三人知道叶尘的脾气，也不觉得有什么奇怪，但是落在徐倩倩三人眼中，顿时越发觉得，叶尘就是一个中看不中用的绣花枕头，索性也都不去理他。
<br/>　　时间一久，叶尘也觉得有些无趣，
<br/>　　他原本之所以答应前来，是想找个机会，让李虎看清楚徐倩倩的真实面目，可眼下实在找不到合适的时机。
<br/>　　就在叶尘准备起身离去的时候，忽然一道阴测测的声音响起，
<br/>　　“哟！还真是倩倩啊，听说你被方少给甩了，这么快就又钓到新凯子了？还真是神速呀！”
<br/>　　众人纷纷循声望去，
<br/>　　只见，
<br/>　　说话的是一名流里流气的魁梧青年，穿着一身名牌，染着一头黄毛，身上隐隐透着一股煞气，显然不是一个简单角色。
<br/>　　徐倩倩看到此人，顿时脸色大变，连忙站起身来，娇躯也跟着微微颤动，强颜欢笑道：
<br/>　　“豹哥说笑了，我们……我们就是普通同学而已……”
<br/>　　李虎听到这话，顿时脸色有些难看。
<br/>　　而那黄毛听到这话，嘴角不由得微微一扬，露出一抹嘲弄之色，
<br/>　　“原来是这样啊！那刚好，我那边来了一位贵客，你帮豹哥我好好去招待招待吧！”
<br/>　　黄毛青年一边说着，便直接伸手朝徐倩倩的手腕抓去。
