　　不管李毅恒的想法如何，月圆之夜还是到了。
<br/>　　但老天似乎并不想看到这咸阳城外发生的事情，用一片硕大的乌云彻底的笼罩了整个天空，使得明明是月圆之夜，但地面上却黑的伸手不见五指。
<br/>　　可就算是这样，咸阳城东那仅有的一片密林中却已经或明或暗的挤满了人，如果好奇心数一数的话，估计一点也不比林子里的家雀少。
<br/>　　这些人来自各大势力，有五姓七望的，也有勋贵、权贵的，不过人数最多的还要数大唐情报科和隐王的势力。
<br/>　　不过林子里的人虽然多，但却没有任何人发出一点声音，这些人或在树上，或树下，有些在林子外面，全都静静的等着。
<br/>　　可是随着时间一点一点的过去，可是主人公独孤玉凤却一直也没有出现，这让林子里的人耐心开始一点点减少，一种叫作焦躁的情绪开始在密林中蔓延。
<br/>　　丑时末，换成现代的时间也就是凌晨三点左右，密林中间唯一的凉亭里面不知什么时候突然多出一个人来，一个身穿白色百褶长裙的女人——独孤玉凤！
<br/>　　没想到，她竟然真的敢来？！密林中那些本以为自己被耍了的众人完全愣住了。
<br/>　　他们相信自己的眼睛从来就没有离开过那个亭子，可是依旧没有看到独孤玉凤是如何出现的，单单就这一下，便已经体现出这位前大唐情报科副科长的能力。
<br/>　　当然，仅仅这样的话并不足以让众人惊讶，虽然做到这一点并不容易，但是能来并不一定就能走，密林之中等着独孤玉凤的人数不下三百之众，相信这位前情报科副科长就算是战力再强大，在露了真身之后也很难在这么多人中间从容离开。
<br/>　　所以真正让人惊讶的是独孤玉凤的胆子，在明知来了就无法离开的情况下，她依旧敢现身出来，这足以说明，此人并不是想象中的那么简单，来了就必然有所准备，只是所有人都不知道他的后手是什么。
<br/>　　而就在众人惊讶于独孤玉凤会以什么手段离开的时候，这位前情报科副科长开口了，嗓音清脆悦耳：“诸位，既然来了就不要再藏头露尾的了，都出来吧！”
<br/>　　“黑夫人好胆色！明知今天晚上这里便是龙潭虎穴，竟然还敢来，果然是女中豪杰，在下佩服。”就在独孤玉凤话音刚落的同时，凉亭的外面突然出现了一个人影，用一种可以刺破人耳鼓的声音赞叹道。
<br/>　　黑夫人，这是独孤玉凤嫁给黑子之后的新绰号，既可以表示她已经嫁人，又可以代表她的性格，一举两得。
<br/>　　但是独孤玉凤显然并不吃来人的那一套，只是微微皱了皱眉：“哼，想不到，大名鼎鼎神龙见首不见尾的鬼面修罗竟然会在此地出现，不知道杨天那小子知道了会不会进退两难。”
<br/>　　那声音的主人嘿嘿一阵怪笑，用一种满不在乎的语气说道：“这个不劳黑夫人操心，在下既然敢来，自然就有办法离开。”
<br/>　　“希望如此吧！”独孤玉凤淡淡扫了他一眼，随后又看向另一个方向：“情报科的人，站出来露个脸，让姑奶奶看看是谁！”
<br/>　　随着声音落下，黑暗中走出两个黑影，用一种奇怪的声音说道：“副科长，是我们！”
<br/>　　“天佐，天佑？看来杨天还真是看得起我，竟然把你们两个给派来了。”独孤玉凤话里的虽然带着惊讶的意思，但语气却一点变化都没有，显然对这两人的到来并不真的意外。
<br/>　　“副科长，跟我们回去吧，陛下说了，只要你肯回头，以前的事情既往不咎。”还是那种奇怪的声调，但有些聪明人已经可以听出来，这是两个人同时在说话，只是他们的步调十分一致，所以才会形成这种奇怪的声调。
<br/>　　但独孤玉凤又岂是别人三言两语就能劝回头的？不说她接受的任务是什么，就算是没有接受李承乾的任务，走到这一步之后也不可能真的回头，就算更退一万步，她真的回头了，肯认错，难道回去之后就真的能像以前一样么？
<br/>　　所以双胞胎兄弟的话显然是白说了，独孤玉凤只是轻轻一笑：“你们兄弟还是不要再劝我了，这么多年相信你们也知道我的性格是什么样的。”
<br/>　　“可是副科长，您就不替科长想想么？”天佐、天佑又再次开口问道。
<br/>　　独孤玉凤沉默了，良久之后，用一种冰冷的声音说道：“生死有命，富贵在天，你们回去之后有机会的话就告诉他一声，这次是我独孤玉凤对不起他，来生做牛做马再还他吧！”
<br/>　　“副科长……”
<br/>　　“双煞，你们还是不要白费口舌了，你们的副科长已经不是当年的副科长，所以听老夫一句劝，早早退走吧！”那刺耳声音的主人鬼面修罗不等天佐、天佑把话说完，已经插了进来，打断了他们兄弟。
<br/>　　双胞胎兄弟被那鬼面修罗一阵冷嘲热讽弄的有些下不来台，哼了一声同时开口说道：“鬼面，这是我情报科的家务事，我劝你最好还是少插嘴，小心祸从口出。另外，如果我是你就绝对不会来这里趟这趟混水，毕竟东西再好，也不如小命重要。”
<br/>　　“好啊，老夫就等着你们呢，有种你们现在就过来抓老夫，看看如果完不成任务，回去皇帝陛下会不会剥了你们的皮。”
<br/>　　“你……”双胞胎兄弟被那鬼面修罗用话一挤兑，顿时怒火万丈，但最终却也没有真的与他动手。
<br/>　　虽然鬼面修罗是情报科黑名单上的要犯，但和独孤玉凤相比，显然差了不是一个级数，所以为了这次的任务，他们只能忍下来。
<br/>　　而就在此时，另外一个方向上也走出一个人来，远远的未语先笑：“诸位，相信这次大家都是为了我们这位前大唐情报科的副科长而来，在没有达到目的之前，我想我们是不是应该摒弃前嫌，合力先把她拿下再说呢？”
<br/>　　“放屁，这是我大唐情报科的事情，哪里轮到得你们动手，如果识相的话我劝你们最好马上离开，否则的话，日后天涯海角情报科奉陪各位到底。”
<br/>(本章完)
