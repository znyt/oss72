        童筱颖不明白，明明季雅珊认识她，此刻却又装作是第一次与她见面的样子，跟她格外客套。
<br/>
<br/>        而且，她能从季雅珊的眼神里读出，她似乎有什么难言之隐。
<br/>
<br/>        另外，季雅珊所提到的“八爷”？！
<br/>
<br/>        若不是季雅珊提醒，童筱颖差点都忘了，“夜惑”全名夜色魅惑娱乐城，里面是有十位“爷”控股，费尔南坐镇当执事。
<br/>
<br/>        除了十爷，其余九位爷，童筱颖没见过。
<br/>
<br/>        所以说，她起初以为是大哥给她的惊喜，现在看来，是她想多了。
<br/>
<br/>        十有八九，她是被人给劫来的……
<br/>
<br/>        理由是什么？
<br/>
<br/>        难道是因为大哥的缘故吗？
<br/>
<br/>        直到浴缸里的水渐渐变凉，童筱颖才起身离开浴缸，抓起毛巾擦干净身体后，穿上了那套季雅珊给她留下的干净衣服。
<br/>
<br/>        这是一件雪白的棉纱连衣长裙，质感轻柔贴身，穿在身上很舒适。
<br/>
<br/>        童筱颖刚走出浴室门口，只见一个身着墨绿色骑马装、身段颀长的男人不知何时出现在了这个房间那边的雕花窗前。
<br/>
<br/>        “洗完呢？”男人冷冷地开口，一边说着，一边转过身来，薄而阔的唇角，随意勾起一抹似笑非笑。
<br/>
<br/>        他脸上戴着黑色的假面眼镜，剑眉浓密，假面眼镜的两个眼洞里，那双深遂的瞳仁绽放着凌厉的光芒。
<br/>
<br/>        当他紧盯着她看的时候，给人一股冷峻的威严，甚至略显残酷的感觉。
<br/>
<br/>        “你是八爷？”童筱颖将男人打量完后，淡定地反问。
<br/>
<br/>        男人挑了挑眉：“哟，不错，一猜就中。”
<br/>
<br/>        “……”童筱颖顿时凝眉不语。
<br/>
<br/>        八爷微眯起眸子，开始仔仔细细地打量起童筱颖来。
<br/>
<br/>        他将她上上下下打量个通透后，不禁啧啧嘴，调侃道：“你跟阿十的亡妻婴宁长得还真有几分相似，是不是姓‘冷’的男人，都喜欢长成你和婴宁这样的女人？”
<br/>
<br/>        姓冷的男人？！
<br/>
<br/>        十爷也姓“冷”吗？
<br/>
<br/>        童筱颖只是听着，仍旧默不吭声。
<br/>
<br/>        当初，她被哥哥诺司峻的女朋友唐倩兮带入的“夜惑”，成为里面的驻唱歌女“小谢”，误惹过十爷，幸得季雅珊相救。
<br/>
<br/>        至于十爷的亡妻婴宁，她也有所耳闻，十爷深情，心念亡妻，当时她还替季雅珊感到担忧，直到后来季雅珊跟她断了联系，此刻，往事又被提起。
<br/>
<br/>        “如果把你变成第二个‘婴宁’，那个傅夜沉是不是也会变成跟阿十那一个德行？”八爷挪步走到童筱颖的跟前，猝不及防地伸手掐住了童筱颖的脖子。
<br/>
<br/>        童筱颖丝毫不作任何挣扎，反而十分淡定地凝视着八爷的眼睛。
<br/>
<br/>        她越是这样心如止水，越是让八爷感到不爽。
<br/>
<br/>        “咚咚咚”房门忽然被人扣响，随后响起了季雅珊的声音：“八爷，您该吃药了。”
<br/>
<br/>        “进来。”八爷随之松开了童筱颖。
<br/>
<br/>        童筱颖呛得咳了几声。
<br/>
<br/>        季雅珊端着一个黑色的盘子从玄关处走了过来，盘子上放着一杯清水和两粒未开动的锡箔纸包装的胶囊。
<br/>
<br/>        童筱颖见季雅珊欠身将盘子举案齐眉，对这个八爷毕恭毕敬，心里更加疑惑不解。
<br/>
<br/>        八爷把药吃了后，对季雅珊摆手道：“把这个女人带到阁楼去关着。”
<br/>
<br/>        “是。”季雅珊微微颔首，然后看向童筱颖，接着说道，“小姐，请跟我来。”
<br/>
<br/>        童筱颖捂着自己的脖子，瞥了八爷一眼后，立即尾随季雅珊而去。
<br/>
<br/>        季雅珊带着童筱颖去了顶楼，她关上阁楼的大门后，又关掉了自己的耳麦，性急口快地说道：“筱颖，你现在要做的就是不要承认自己是傅夜沉的女人。”
<br/>
<br/>        “你认识傅夜沉？”童筱颖难以置信地瞪大了眼睛。
<br/>
<br/>        季雅珊点了点头，接着说道：“他救过我的命，没有他，只怕我早就死了。另外，我不知道这个八爷跟傅夜沉之间有什么仇什么怨。总而言之，他很恨傅夜沉，没少给他找麻烦。”
<br/>
<br/>        “那你跟十爷呢？”童筱颖接着问道。
<br/>
<br/>        季雅珊不禁黯然神伤地垂下眼帘，答非所问地转开了话题：“你要记住我所说的话，不要承认你跟傅夜沉之间的关系。”
<br/>
<br/>        “我要怎么离开这里？”童筱颖见季雅珊不愿提及她自己的私事，于是不打算再追问下去，她得先关心自己的事情才行。
<br/>
<br/>        季雅珊顿了顿后，才回答道：“我不知道，我来这里三年了，就从来没出去过。”
<br/>
<br/>        “不如我们一起逃？”童筱颖提议。
<br/>
<br/>        季雅珊微微摇了摇头：“下山的那条路是唯一的出入口，有狙击手埋伏。你起初若是直接走下山去，一定会被暗杀。还好，你主动往古堡里来了。”
