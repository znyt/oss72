　　挥手捏断一头毛僵的脖子，刘哔身上也增添了几道深可见骨的伤痕，借助‘不死’技能的特性，他应对起这些僵尸也不算困难。
<br/>　　而前方刚刚摆脱刘哔的鬼修少年，看到场中的战况，便也顾不得身体上的伤势，咬牙重新加入了战团。
<br/>　　他可不想等到刘哔解决掉几头毛僵之后，自己独自面对他，那样的话，他根本没有半点胜算。
<br/>　　随着鬼修少年的加入，刘哔所面临的压力又重新增大了起来。
<br/>　　毕竟双拳难敌四手，更何况这些敌人的境界，实际上和他也相差不了许多。
<br/>　　几乎每时每刻，刘哔身上的伤痕都在不断增加着，如果换成普通修炼者的体质，这些伤势足以要了性命。
<br/>　　当然，普通的修炼者也不会像他这样完全不顾及防御罢了。
<br/>　　而这样的打法，自然也将战斗过程加快了许多，交手数十回合后，刘哔斩下了第二头毛僵的头颅。
<br/>　　此时场上，除了三头毛僵之外，就只剩下遍体鳞伤的鬼修少年了，胜利的天平，逐渐向刘哔这边开始倾斜。
<br/>　　知道这样拖下去迟早会输掉这场战斗，鬼修少年逼退了刘哔，暂时拉开了两人之间的距离，手掌之上，有一团阴厉无比的气息散发出来。
<br/>　　刘哔眉头一皱，似乎是察觉到什么，硬扛下了三头毛僵的攻击，身如闪电般的向鬼修少年冲去。
<br/>　　对方的样子，明显是需要时间凝聚一次强大的攻击，刘哔并不想给他机会。
<br/>　　虽然说他的反应已经很快了，但终究还是晚了一步，随着一句晦暗难懂的话语从鬼修少年嘴中吐出，场上突然刮起了一阵阴风。
<br/>　　“我愿将生命交付于您，请您降下神力，诛杀世间一切敢于冒犯您的敌人。”
<br/>　　对眼前的攻击仿若未闻，鬼修少年双手合在胸前，脸上尽是一片虔诚。
<br/>　　当其话语落音的时候，刘哔心中突然涌上一股极度危险的感觉，随后，眼前飘起一阵漆黑如墨的黑雾，将他整个人给包裹了进去。
<br/>　　这种能量，让他感觉有些熟悉，像是鬼气，但似乎还掺杂着一些东西。
<br/>　　“暗黑，审判！”
<br/>　　一声低喃从黑雾中响起，是鬼修少年的声音，刘哔满脸戒备，挥手用天地灵气凝聚出几层猩红色的防御罩。
<br/>　　但这些防御罩似乎并没有产生多大效果，黑雾中一个个透明光点如同箭矢一般，铺天盖地的向刘哔用来，只是瞬间的功夫，就突破了防御罩，将他身体洞穿出无数个小洞。
<br/>　　从进入这个世界以来，这是他所面临最大的危机，此时他都找不到这些攻击是从什么地方发出来的，唯一能做的，就是用全身的能量护住头部。
<br/>　　透明光点如同无穷无尽一般，不断在刘哔身边飞舞穿插着，其上强烈的腐蚀性，将他的身体腐蚀成一汪血水。
<br/>　　就像是惊涛骇浪里的一叶扁舟，刘哔索性将所有防御力量都凝聚到头颅处，苦苦支撑着。
<br/>　　十分钟……半小时……
<br/>　　不知道坚持了多久，就在他体内的灵种即将枯竭的时候，四周的黑雾逐渐开始变得稀薄，那些恐怖的透明光点的数量，也终于不像开始那般密密麻麻了。
<br/>　　感受到四周的变化，刘哔心中一喜，立即将剩余全部的灵气都爆裂开来，冲破了这片黑雾凝聚而成的牢笼。
<br/>　　被削弱的黑雾牢笼，再也束缚不住刘哔，只是瞬间，一颗人头拉着猩红色的能量匹练从中冲了出来。
<br/>　　黑雾逐渐消散，露出了里面的光景，此时那几头毛僵，都像是失去了生命一般，歪七竖八的躺在地上，没有丝毫动静。
<br/>　　鬼修少年的身影，立在场地中央，满脸皱纹，身体表面的皮肤，也变得无比苍老，再也看不出原来的一点痕迹。
<br/>　　没敢有丝毫停顿，逃出升天之后，刘哔立即用系统修复了身体，宛如一柄利剑一般冲了过去。
<br/>　　手起，杖落，鬼修少年没有半点反抗，被龙头拐杖敲中了的头颅，如同一颗敲烂的西瓜一般裂开，红白之物流了满地。
<br/>　　直到攻击落下的时候刘哔才发现，鬼修少年已经死了，施展这个技能的代价，是他的生命。
<br/>　　“叮~恭喜宿主完成任务，下一环任务已经解锁，请宿主打开系统进行查看。”
<br/>　　“叮~恭喜宿主完成连锁任务，奖励抽奖机会一次，积分一万。”
<br/>　　系统提示的声音连续响起，刘哔微微一愣，进入任务面板查看了一下，恍然大悟。
<br/>　　连锁任务的最后一环，是击杀鬼修少年，但那几头毛僵和鬼修少年一起死掉，他所有的任务自然也就一起完成了。
<br/>　　只是有些可惜的是，鬼修少年和最后三头毛僵并不算是自己击杀，故此也没能获得这四份积分。
<br/>　　不过还好，尸丹现在已经到手，连锁任务也已经完成，等从这里离开之后，紫女的封印也就可以解除了。
<br/>　　虽然中间发生了很多意外和危险，但是收获还是非常丰盛的，对此，刘哔心中也十分满意。
<br/>　　稍稍恢复了一下体力，刘哔将龙头拐杖收进了空间戒指，走到鬼修少年和几具毛僵尸体身边，搜索一番后，发动了吞噬技能。
<br/>　　“叮~恭喜宿主成功吞噬练气期鬼修一名，获得技能‘鬼步’，请宿主进入技能面板进行查看”
<br/>　　“叮~吞噬失败。”
<br/>　　“叮~恭喜宿主成功吞噬毛僵一只，获得技能‘坚韧’，请宿主进入技能面板进行查看。”
<br/>　　“叮~恭喜宿主成功吞噬毛僵一只，获得百年尸气一份，已储存到宿主体内。”
<br/>　　“叮~吞噬失败。”
<br/>　　“叮~恭喜宿主成功吞噬毛僵一只，获得百年尸气一份，已储存到宿主体内。”
<br/>　　连续几道系统提示音响起，只是让刘哔有些意外的是，其中有两只毛僵的尸体竟然吞噬失败了。
<br/>　　但想来也是情有可原，毕竟这两只最先击杀的毛僵，死去的时间也实在是太久了，保存度自然没有那么完好。
<br/>　　不过，能获得两个技能，也让他非常满意，但百年鬼气有什么用处，他还没有丝毫头绪。
<br/>　　没有着急打开系统查看，刘哔先将那些零散的低级僵尸尸体挨个吞噬，因为死去时间太长，大部分都失败了，只吸收了寥寥几个，获得了几份不同年份的尸气。
