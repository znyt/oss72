<br/>　　苏莫心中猜测，但是，他很快就否决了这个猜测，那紫霄堂堂无上神王，紫霄剑宫之主，貌似没有必要对付他。
<br/>　　苏莫想不明白，败不再多想，一边飞行，一边查看吞噬元神之中的那银色大剑元神。
<br/>　　此元神，威力倒是不俗，虽然远不如万化神剑，但是，比之普通的元神，也强大很多倍。
<br/>　　难怪那名男子，只有这一种元神，却能修炼到三品的修为。
<br/>　　“很适合绫裳！”苏莫暗暗的点了点头，绫裳的天赋真的很差，这元神正好可以给予对方。
<br/>　　少倾，苏莫不再多想，加快速度赶路，向时空界域而去。

<br/>　　紫霄神王阴沉着脸，看着下方脸色惨白、身上染血的许焱。

<br/>　　“此子实力如此之强，连你都不是对手？”紫霄并未动怒，面无表情的问道。
<br/>　　“是，比我略强三分！”许焱重重的点了点头，虽然他不愿意承认，但是那秦陨的实力，确实是比他强不少。
<br/>　　“敢肆意屠杀同门，简直是胆大包天！”紫霄神王眸中，闪过犀利的冷芒。
<br/>　　“宫主，此人必须严惩！”许焱厉声说道，适才多亏他跑的快，不然也要交代在那里了。
<br/>　　“嗯！”本宫主知晓了，你下去吧！”紫霄微微点头，随即挥了挥手。
<br/>　　“这……宫主，不派人捉拿秦陨吗？”许焱疑惑的问道。
<br/>　　“不用，他会回来的！”紫霄微微摇头，那秦陨根本不用去捉拿，会回到神剑山的，毕竟有那个女人需要他救治。
<br/>　　至于死了三个普通的弟子，紫霄并不在意。
<br/>　　“是！”许焱无奈的点了点头，既然宫主如此说了，他也不好再说什么。
<br/>　　最后，许焱只能憋着闷气，告辞离开了紫霄大殿。
<br/>　　离开了紫霄大殿之后，许焱脸色铁青，越想越气。
<br/>　　因为，不仅他自己受了重伤，苗妃的死，更是让他很是心痛。
<br/>　　苗妃不仅是和他关系很好，更是他床第间的玩物，就这么死了，实在是可惜。
<br/>　　“秦陨啊秦陨，我不会放过你！”许焱心中暗暗发狠，这秦陨必须擒拿或者杀死。
<br/>　　不然的话，对方若是畏罪潜逃，很可能再也不回来了。
<br/>　　随即，他立刻开始去找人，现在秦陨还没有逃远，或许还能追踪的到。
<br/>　　他自己不是秦陨的对手，只能去找帮手了。
<br/>　　很快，许焱便来到了紫霄剑宫的一处阁楼之前。
<br/>　　“慕容师兄可在？”许焱双手抱拳，向着阁楼内轻喊了一声。
<br/>　　“何事？”
<br/>　　少倾，阁楼内传出低沉而浑厚的男声，中气十足。
<br/>　　“慕容师兄，我有要事求见！”许焱沉声道。
<br/>　　“进来吧！”低沉的声音再次响起，随即，阁楼的大门打开了。
<br/>　　许焱见此，便走进了阁楼之中，见到了在阁楼中蒲团上盘膝打坐的青年男子。
<br/>　　这名青年男子，身穿白衣，身形高瘦，皮肤很白，给人一种森然之感。
<br/>　　此人的面容，俊朗而有一丝邪异，狭长的眼眸，隐隐有雷光闪动，让人心中生出一种压迫之感。
<br/>　　“许焱，你有何事？”白衣男子看向许焱，淡淡的问道。
<br/>　　“慕容师兄，苗妃被人杀了，程金和元明也被人杀了！”许焱恭敬的向白衣男子抱拳，面带悲戚之色的说道。
<br/>　　这位白衣男子，名为慕容雷利，可不是普通人，乃是紫霄剑宫的一位万古天骄。
<br/>　　修为虽然和他相同，同样是三品中级，但是战力确是达到了四品的可怕地步。

<br/>　　慕容雷利闻言，双眸一挑，眸中闪过犀利的寒芒，冷声问道：“是谁杀的？”
<br/>　　“秦陨，一名绫裳剑宫的弟子。”许焱道。
<br/>　　慕容雷利闻言，略一沉吟，疑惑的问道：“此事，你应该禀告宫主，或者神王殿的执法长老，为何来找我？”

<br/>　　“修为二品中级，但是战力比我还要强，我不是他对手，此人现在刚刚离开神剑山！”许焱道。
<br/>　　“比你还要强，万古天骄吗？”慕容雷利低声自喃，心中有些意外。
<br/>　　因为，神域虽然很大，但是，万古天骄也就千人左右而已。
<br/>　　“是的！”
<br/>　　许焱点头，随即略一沉吟，道：“慕容师兄，若是你能擒拿了秦陨，我赠你十块雷天晶！”

<br/>　　他之所以愿意去杀秦陨，不单单是雷天晶，更因为苗妃算是他的女人。

<br/>　　随即，慕容雷利和许焱一起，火速的离开神剑山。
