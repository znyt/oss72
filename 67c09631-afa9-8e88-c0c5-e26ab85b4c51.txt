夏夕绾已经不在了。
<br/>
<br/>        陆寒霆迅速睁开了眼，刚才的惺忪睡意全无，昨晚的记忆也慢慢的涌入了脑海。
<br/>
<br/>        昨晚他发病了，然后将她给丢在了床里。
<br/>
<br/>        她血液里的腥甜和那股少女体香像是罂粟，让他上瘾沉迷，最后他按着她的菱唇…
<br/>
<br/>        陆寒霆一下子就坐起了身，然后起身下床，他在偌大的房间里四处寻找，橱帽间，沐浴间…都已经没了她的身影。
<br/>
<br/>        “绾绾…”
<br/>
<br/>        她走了么？
<br/>
<br/>        她应该是走了。
<br/>
<br/>        昨晚他将她弄出多少伤口已经记不清了，最后他还做了那样卑鄙无耻的事情，那个阴郁狠戾，狰狞嗜血的自己令他都觉得厌恶和害怕，更何况是她呢？
<br/>
<br/>        昨晚，她一定受了很多伤，一定害怕极了，所以，她才走了。
<br/>
<br/>        以后，她还会回来么？
<br/>
<br/>        陆寒霆拿出了手机，翻出了夏夕绾的电话号码，他想拨过去，但是手指最终都没能按下拨号键。
<br/>
<br/>        ……
<br/>
<br/>        陆寒霆冲了一个冷水澡然后下楼，陆老夫人迎上来，“寒霆，你今天起得很晚啊，绾绾怎么那么早就走了，听吴妈说，绾绾好像凌晨五点多就出门了，你们吵架了？”
<br/>
<br/>        陆寒霆抿了一下薄唇，“奶奶，我们没有吵架。”
<br/>
<br/>        “那就好，女孩子是要哄得嘛，而且绾绾那么乖那么软，特别好哄了，你待会儿打个电话给她，带她出去吃个烛光晚餐。”
<br/>
<br/>        她真的很乖软…
<br/>
<br/>        所以，特别好哄，特别好骗。
<br/>
<br/>        陆寒霆英俊的眉眼里溢出了几分温存和宠爱，“奶奶，我想我不会打电话给绾绾了。”
<br/>
<br/>        陆老夫人一滞，“为什么？”
<br/>
<br/>        “没遇到绾绾之前，我从来都不觉得我是一个不正常的人，直到遇到绾绾之后，没有任何时刻让我无比清晰的意识到我终究不是一个正常人，趁我现在还可以放手的时候，就让她走吧，不然…”
<br/>
<br/>        不然他怕以后他对她的掌控和独占欲超脱了自己的控制，会不择手段的将她禁锢在自己的身边。
<br/>
<br/>        昨晚让她看到那样不堪的自己，真的很抱歉。
<br/>
<br/>        他不能再放任自己。
<br/>
<br/>        虽然，如果现在连她都要失去，那他的病情…
<br/>
<br/>        “奶奶，有你陪在我的身边就行了，我去公司了。”陆寒霆抱了奶奶一下，然后转身出门。
<br/>
<br/>        陆老夫人目送着自己的孙子离开，她重重的叹息了一声，她年纪终究是大了，还能陪几年？
<br/>
<br/>        她的陪伴并没有让他变好，那是因为光有她的陪伴是不够的，她的寒霆从小就没有爱，他很孤独。
<br/>
<br/>        他需要一个女孩，让他爱与被爱。
<br/>
<br/>        可是，他的身体…
<br/>
<br/>        陆老夫人精神矍铄的腰背因为这一声叹息而轰然弓塌了下来，显出了几分老态。
<br/>
<br/>        这时吴妈从楼上匆匆走了下来，一脸喜色的小声道，“老夫人，刚才我去收拾了少爷少奶奶的房间，我发现床单要换洗了！”
<br/>
<br/>        什么意思？
<br/>
<br/>        陆老夫人看着吴妈，刚才弓塌的腰背瞬间又直了回去，年轻了好几岁，“你说真的？”
<br/>
<br/>        陆老夫人是什么人，陆家是帝都第一豪门商胄之家，她坐镇陆家几十年，新婚夜那条喜帕上的雪梅是真是假的她会看不出来？
