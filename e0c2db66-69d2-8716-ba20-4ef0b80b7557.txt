        &lt;h3&gt;第三卷第二百八十章  刘诚之到甸城&lt;/h3&gt;
<br/>
<br/>        刘诚之到达甸城的时候，先是找个地方住下，他没有去找何少华，而去找陆玉。
<br/>
<br/>        陆家和他们家也是相识的，陆玉认识他，目前他是一个执挎的少年公子，工作对他来说就是可有可无。
<br/>
<br/>        陆玉在甸城的身份地位不一般，他一个纨绔子弟去找陆玉，讨点便利也是正常的。
<br/>
<br/>        他在甸城溜溜达达的转了几天，哪里有好吃的，好玩的，他都会去看一看，当地最高档的酒店和酒吧他都要去住一晚，喝一次，他住进了皇后酒店，到皇后酒吧喝酒，一个人喝得酩酊大醉，拽着人家小姑娘不撒手，酒店的保安准备将他扔出去，刘诚之睁着微红的双眼醉醺醺的说：“你们谁敢动我一下？我的姐姐是，是陆玉，本地最厉害的人。你们动我试一试？啊！来啊！试一试？”
<br/>
<br/>        陆玉是这里的常客了，每次来都是谷正娟和秦少卿陪着，保安一听，心里有些发楸，只好给谷正娟汇报。
<br/>
<br/>        谷正娟正在和谷强研究后期生意怎么开展，谷强和祁峥用了两年的时间将甸城信息化的方案已经完成了，当然，在这两年里，祁峥和陈子昂秘密的进行着联系，实行的方案也有陈子昂的一份功劳，陈子昂对祁峥说：“我们是商人没错，我们不但要赚钱我们更重要的任务和责任是推进一个地区的发展和改变，所以无论赚多少钱，要保证我们的良心不会疼。”
<br/>
<br/>        陆玉帮着申请的项目资金已经下来了，新的公司也已经注册完毕，现在只等项目启动。
<br/>
<br/>        这是谷强设计的一个局，是生是死，就要看后期怎么走了，谷强原本不想拖着陈子昂到甸城的，可是在他们仅有的几次通话中，他知道她聪明的小百灵已经不是小孩子了，她已经是能够运筹帷幄，和他一起战斗了，如果说能够毁灭基地里的东西搭上陈子昂和他的生命他也是愿意的，虽然他的想法有点自私。
<br/>
<br/>        “后期用新的公司来做这个事情，只是利润如何分配，还有祁峥和即将要来的陈子昂又如何能甘心嘴里的肉被别人夺走？”谷正娟问谷强。
<br/>
<br/>        “他们赚取他们该得的就可以了，这俩个人能在短短的时间脱颖而出，就是因为他们不贪心，所以我才看中了这俩个人，你放心吧，只是陆家怕不是简简单单的放手。”谷强对谷正娟解释着
<br/>
<br/>        “我们现在所做的一切难道不是为了陆家的将来吗？你以为陆玉真的是为了秦少卿而来？为爱而来？我就没有见到过陆玉爱过任何人。”谷正娟不屑的说。
<br/>
<br/>        “那个.......”
<br/>
<br/>        “嘀嘀嘀”的敲门声打断了谷强的话。
<br/>
<br/>        “怎么回事？”谷正娟皱着眉头打开门，问来人。
<br/>
<br/>        “外面有个人喝醉了，大喊大叫说是陆玉的弟弟。”来人说。
<br/>
<br/>        “陆玉哪来的弟弟？扔出去不就行了。”谷正娟嫌弃来人打扰了他们的谈话，不耐烦的说。
<br/>
<br/>        “好吧！”来人转身就走。
<br/>
<br/>        “等等，我去看看，也许真的是认识陆玉。”谷强心里有些疑虑，谁会冒充陆玉的弟弟来引起陆玉的注意？
<br/>
<br/>        “那我随你一起去吧。”只要和陆玉有点牵扯的，谷正娟都不愿意看到，谷正娟不想谷强和陆玉有来往，这也就是陆玉到甸城一直都没有更深层次接触到谷强的原因。
<br/>
<br/>        其实，谷强希望能够再接触到一些和谐的问题。
<br/>
<br/>        俩个人到了酒吧，刘诚之已经安静了下来。
<br/>
<br/>        “是你找陆玉吗？”谷强问刘诚之，刘诚之觉得眼前的这个男人如此面熟，似乎他在哪里见过，只是他的身上散发着一股冷冽的气息，似乎是三九寒冬落入冰窖的感觉。
<br/>
<br/>        “什么叫我找她啊，她是我姐姐啊，她不应该来见我吗？”刘诚之醉醺醺的问谷强。
<br/>
<br/>        “哇，你这个男人好漂亮啊，啧啧，是不是我姐夫？啊，不对，我姐夫是张函，我认识，不是你，你是谁？”刘诚之摇摇摆摆的想伸手抓住谷强，被谷强将手捏着不能动弹。
<br/>
<br/>        “疼疼疼，你松手。”刘诚之大喊。
<br/>
<br/>        “放开他，他可能真是陆玉的亲戚，给陆玉打个电话吧。”谷正娟说。
<br/>
<br/>        “你打吧。”谷强很少和陆玉来往。
<br/>
<br/>        过了一会，秦少卿陪着陆玉过来了。
<br/>
<br/>        “姐姐，玉姐姐，你可来了，他们都欺负我。”刘诚之见了陆玉哭唧唧就向陆玉身上扑，被秦少卿给挡住了。
<br/>
<br/>        “你是谁，别挡着我，姐姐。”刘诚之摇摇晃晃。
<br/>
<br/>        “诚之，你怎么来了？不是听说你上班了吗？”陆玉看见刘诚之还是有点吃惊。
<br/>
<br/>        “上个劳什子班，我就想吃吃喝喝游山玩水，看看美丽的姐姐。”
<br/>
<br/>        “别闹了，你在哪里住着？”
<br/>
<br/>        “我在皇后大酒店住着啊，虽然上班没有拿多少钱，你也知道咱们家里谁还缺钱？”刘诚之絮絮叨叨。
<br/>
<br/>        “你也别喝了，我们送你回房间，你好好休息，我明天来看你。”陆玉皱着眉头看着刘诚之，他们家和刘家关系一般，大家都认识，他的父亲始终想拉拢刘家，可是刘诚之的父亲比较迂腐，谁也不得罪谁也不帮扶，反而有时候会和她父亲怼几句。
<br/>
<br/>        “玉儿姐姐，你明天真的来看我吗？我来甸城好几天了，就是想见见你。”
<br/>
<br/>        “行，我知道了，少卿，谷强，你们送诚之回房间吧，他是我的一个小弟弟。”陆玉对谷强和秦少卿说。
<br/>
<br/>        “我一个人送就行了，这小子弱不禁风的，我还能拽得动。”谷强说着就将刘诚之拉着送到房间。
<br/>
<br/>        “少卿，我们回去吧。”陆玉不想多留，她确实是怀孕了，不到四个月，可是她总是提不起精神来。
<br/>
<br/>        “你小心点。”秦少卿温柔的扶着陆玉，让谷正娟感觉一阵恶寒。
<br/>
<br/>        “正娟，我明天迟点过来，诚之你就帮着照顾一下。”陆玉对谷正娟说。
<br/>
<br/>        “好，知道是你的弟弟就好了，我就是怕有人冒充。”谷正娟解释着。
<br/>
<br/>        “他是......没事，他不是冒充的，我明天过来接他。”陆玉原本想着给谷正娟说刘诚之的身份，话到嘴边又咽回去了，坚决不能让谷正娟他们接触另外的一方势力。
<br/>
<br/>        刘诚之被谷强扔在了床上，醉醺醺的已经睡过去了，谷强静静的看了一会刘诚之才离开。
<br/>
<br/>        等谷强离开，刘诚之睁开眼睛，这个谷强的眼神太犀利了，他闭着眼睛都能感觉到他那审视的眼神。
<br/>
<br/>        只是这个人好眼熟，似乎在哪里见过？到底在哪里见过，刘诚之想不起来。
<br/>
<br/>        不过已经成功引起陆玉的注意这就够了，看今晚的样子跟紧陆玉，刘诚之可以在甸城横着走了。
