　　场中的状况对CCG一方逐渐变的开始不利了起来，指挥官站在后方，远远向战场中心眺望着。
<br/>　　“和修局长，敌人的实力太强劲了，这样下去，第一梯队恐怕不是对手。”指挥官身边，一个挂着准特等身份牌的搜查官开口，语气有些担忧。
<br/>　　略一思索，和修的表情有些犹豫，作为CCG的局长，这次战役的总指挥，他需要考虑的地方实在太多。
<br/>　　但如果放任敌人这样突破下去，这次战役或许就要以失败告终了。
<br/>　　他有些遗憾，此时自己的得力助手，被派遣去了另一边的战斗当中。
<br/>　　“第一队努力撑下去并将敌人分散，第二，三队行动，负责将分散的敌人各个击破，由筱原特等负责指挥这两个部队。”拿起手边的通讯器，和修下达了命令。
<br/>　　通讯器中响起一个低沉的男声，筱原接到了命令，立即向各区的负责人通知了下去，原本正在待命的第二，三两支队伍，顿时涌入了战场。
<br/>　　跟随着二十区的队伍，刘哔也悄悄进入了战场中，现在咖啡厅附近非常混乱，他只要小心一些，应该不会被芳村给发现。
<br/>　　但不敢暴露自己的全部实力，他猎杀喰种的速度也比寻常慢了不知道多少。
<br/>　　不过即使是这样，他的表现依旧引来了一些人的关注，毕竟他们这次面临的对手，都是身经百战的中高阶喰种，一个新人能做到这种地步，也着实让人感到不可思议。
<br/>　　似乎是察觉到什么，古间圆儿的动作停顿了一下，急速向刘哔这边靠来。
<br/>　　一名SS级喰种的动作是十分迅速的，况且战场上的情况无比混乱，正处于激斗状态中的刘哔，并没注意到古间圆儿的袭击。
<br/>　　等他察觉到不对劲的时候，古间圆儿已经冲到了他的面前，此时想要防御，已经有些来不及了。
<br/>　　“高次，精神次元！”
<br/>　　眼看魔猿的利爪即将插入刘哔心脏的时候，一声浑厚的呐喊声从他耳边响起，顿时，一股巨大的冲力从身边袭来，入目望去，全是如水一般的蓝色。
<br/>　　不远处的空地上，一个长着浓密八字胡的黑发男子，手持着一柄和机枪有几分相似的库克因，枪管处真往外飘着淡淡的青烟。，
<br/>　　“你的武器，还真是危险呢。”魔猿从地面上站起身来，先前突然之间的攻击，让他受了不小的伤势：“喏，你的同伴，好像不行了呢。”
<br/>　　远处的战场中央，刘哔残破的身体挂在树上，从左胸开始，整整一半的身体已经不知去向。
<br/>　　“啊，为了杀你，这也是没有办法的事呢。”仰了仰头，黑发男子的鼻孔处喷出两道气流，随后他将视线转移到魔猿身上：“所以你啊，还是快点回到山上去吧……小猴子。”
<br/>　　不屑的嗤笑一声，魔猿的身影在半空中腾挪，转瞬之间，就和黑发男子激战到了一起。
<br/>　　毕竟是SS级的喰种，论身体素质，要比人类好了许多，从短暂的交锋中看来，男子倒还稍稍落入了下风。
<br/>　　只是所有人都没注意到的是，原本树上那名已经死去的搜查官，不知何时已经失去了踪迹。
<br/>　　或许有人察觉到了，但谁都不会去在意，战场这么混乱，说不定是哪一只喰种吃了他的尸体，用以补充能量。
<br/>　　没有谁，会去在乎一个新人搜查官的死活，只要能将敌人击杀，有所牺牲也是十分正常的事情。
<br/>　　战场的另一端，黑色杜宾那边的情况也好不了多少，本来她们就不擅长正面作战，面对数量多出数倍的敌人，自然是损失极为惨重。
<br/>　　不过她们的首领‘黑狗’，也是一名SS级的高级喰种，有这样一名强大的喰种，她们倒也能勉强支撑下去。
<br/>　　可是这一切，在一名带着眼睛的白衣男子加入之后，开始变的一边倒了起来。
<br/>　　“哟，好不久不见了，黑狗。”眼镜男站在混乱的战场中央，看到其手中极为熟悉的库克因，周围的喰种皆是不敢上前。
<br/>　　这家伙，她们再也熟悉不过了，不知道有多少同伴，都死在了这个男人手中。
<br/>　　冷哼一声，黑狗也停下了手中的屠杀，缓缓走到了眼镜男面前。
<br/>　　或许她不是眼镜男的对手，但她，必须要去面对。
<br/>　　似乎是接到了什么命令，四周的搜查官们都向四周散开，唯一留在场中的，只有宛如战神一般的眼镜男子。
<br/>　　目光有些决然，黑狗召唤出自己的赫子，如同一柄利剑一般，向眼镜男冲去。
<br/>　　……
<br/>　　就在所有人都打的不可开交的时候，刘哔已经悄悄来到了一处楼顶，他此前受到的伤势，已经全部修复好了。
<br/>　　那个该死的搜查官，下手倒也狠毒，幸好自己闪躲的及时，方才保住了性命。
<br/>　　即使这样，也让他这段时间赚取的积分，花的七七八八了。
<br/>　　建筑群下，那名身穿白色军装的指挥官站在后方，正下达着什么命令，刘哔趴在楼顶上，将整个人都隐藏了起来。
<br/>　　大概过了一刻钟的功夫，刘哔脸上露出了得逞的笑容，随后，他从空间戒指中取出面具，戴在了脸上。
<br/>　　建筑群下方的街道上，一名浑身笼罩在灰色袍子中的喰种，侧身对着众人，不知何时出现在了街道上。
<br/>　　就在前一秒的时候，这名喰种站立的地方还是空荡荡的，就连一直紧密关注着的刘哔，都没有发现任何异常。
<br/>　　果然是SSS级喰种，就连速度，也是到了一种骇人听闻的地步，指挥官站在人群后方，眼神十分凝重。
<br/>　　这场战斗中的主角，现身了。
<br/>　　灰袍男子缓缓转过身来，眼中带着喰种特有的红光，凌厉的气势，让现场所有的搜查官们都是心中一颤，紧张了起来。
<br/>　　他们是负责警戒封锁的第四梯队，本来所有人都在庆幸，他们不必像前三支队伍一样，面对那些恶狗和猴子，但没想到的是，他们所要面对的对手，比那些家伙更加可怕。
<br/>　　店长的面容隐藏在灰袍下面，在夜色的遮掩下十分模糊，从中传出的苍老声音，也不像刘哔第一次听到时那般慈祥了。
