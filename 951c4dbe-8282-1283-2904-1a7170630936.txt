去……”<br/>　　<br/>　　    “吧嗒”王惊蛰把鞋脱了下去，然后伸手悄悄的拿起一个枕头。<br/>　　<br/>　　    茅小草脸上笑意顿时僵住了，手抓着身下的被子，开始磨牙了。<br/>　　<br/>　　    时间缓缓而过，茅小草没感觉身边躺了人，就闭着眼睛翻了个身又转回来了，然后悄悄的露出一条缝隙，就看见王惊蛰蜷缩着身子，躺在靠墙的地板上睡着了。<br/>　　<br/>　　    “人真是个好人啊……”茅小草莫名的感叹了一句:“就是蠢了点，你就禽兽不如了还能怎么样呢？我哪里会舍得把你踢下去哟”<br/>　　<br/>　　    于此同时，齐泉和丁武去的小区外面，一共六个人翻身就从小区外的栅栏跳了进去，然后直奔着三号楼摸去。<br/>　　<br/>　　    车里，午桥手里把玩着个打火机跟老马说道：“你们这么做，可就把巫门给彻底得罪死了”<br/>　　<br/>　　    “本来都刺刀见红了，还什么得罪不得罪的？”老马叹了口气，说道：“老陈是真生气了，必须得要个说法了，这个说法肯定得建在对方流血上才行，不然他的愤怒怎么体现啊？这个吴洋不是策划的么，就拿他说事好了，要是事情到此为止就完事了，大家相安无事，真要是还不完的话老陈也还有别的手段，至于你说得罪巫门？呵呵，以前破四旧的时候，上面扫了多少人啊，在强力机器的镇压下，不也一样老老实实的蹲在牛棚里了么，现在啊也一样，老陈真要是奔着弄死他们去的话，巫门在大在厉害，还能大得过头顶这片天么？谁当家做主，他们心里得有数啊”<br/>　　<br/>　　    午桥听着他的话，顿时皱了皱眉，他们这群人有通阴阳的懂风水的，也有能未卜先知算前路的，但不管牛逼闪电到什么地步，老马说的很对你在大还能捅破自己脑袋上的天么？<br/>　　<br/>　　    想和更多志同道合的人一起聊《天命赊刀人》，微信关注“热度网文或者rdww444”与更多书友一起聊喜欢的书
