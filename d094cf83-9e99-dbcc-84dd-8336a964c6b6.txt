        &lt;h3&gt;第三卷第一百四十八章  我只是不小心进了家门&lt;/h3&gt;
<br/>
<br/>        张函见到了潘青松，细细的问了陈子澜子在甸城的遭遇。
<br/>
<br/>        潘青松知道的也不是很多，只是告诉了陈子澜出车祸的情况。
<br/>
<br/>        当潘青松提到甸村的时候，张函心里一沉。几年前他去那里将闫欢带回来了，只是后续的事情他便没有再去关注，那几年他一门心思的寻找陈子昂。
<br/>
<br/>        他找到陈子昂之后这几年一边努力的工作，一边和陆玉周旋，极力的保护着陈子昂母子，对于几年前的事情他都没有想起过。
<br/>
<br/>        他记得他到甸村的时候，几个小孩一直跟着他，他还给过一个小孩相机。那个小孩叫什么名字来着？叫什么尼？
<br/>
<br/>        不知道那个孩子如今怎么样了？
<br/>
<br/>        “我去看三姐，他们都不欢迎我，而你，睡得和死猪一样。”张函实在是有些闹心，陈家对他的态度真的和以前简直是天壤之别。
<br/>
<br/>        “谁喜欢渣男？你没被扔出去就已经不错了。”潘青松也是毫不客气的给张函再添了一下堵。
<br/>
<br/>        “我和子昂分开是不得已，而且子昂怀孕的事情我压根就不知道。当时我从国外回来的时候就被他们在机场截住，然后我和子昂就没有然后了。”张函的声音充满了悲伤和苍凉。
<br/>
<br/>        “和子昂分开将近四年了，我才知道我有了孩子，三姐夫，你知道那种爱而不得的感觉吗？你知道明明那是你的孩子你还不能见的那种痛苦吗？”
<br/>
<br/>        潘青松沉默不语，张函说的他都不能理解，他和陈子澜相敬如宾、夫妻同心、一家人更是其乐融融。
<br/>
<br/>        “不，三姐夫，你不理解，十年，我就像是见不得光的老鼠，我努力的隐藏着孩子的存在，可是现在怕是要被冯睿暴露了，我不想子昂和孩子有任何事情，我努力的变强，可是我还是这么的弱小，不能够保护他们。”张函底下头痛苦的说。
<br/>
<br/>        “我不能够理解你，如果你不愿意结婚谁能强迫你，难道你现在的老婆会强迫你和她在一起？”潘青松冷冷的说，说的再怎么无奈，无非就是放不下自己的身份而已。
<br/>
<br/>        “她不会强迫我，但是她会对子昂和孩子动手，我不能让他们母子有任何意外发生。”张函解释着。
<br/>
<br/>        “你的意思是你用你的人身自由换取陈子昂母子三人的安全？你也太看得起陈子昂他们了吧？”潘青松像是看白痴一样看着张函。
<br/>
<br/>        “陆玉那人你不了解。你不知道她是什么样的人，我已经怕了她，可是又甩不掉。”张函长叹一声。
<br/>
<br/>        “你找我不是说这些事情的吧？”
<br/>
<br/>        “不是！”张函心里一个激灵，他干嘛要给潘青松说这么多？难道是心里想得到陈子昂的原谅？
<br/>
<br/>        “明天你得回单位工作，你走的这几天挖到了一条大鱼，但不是西城，而是洛城。”
<br/>
<br/>        “哦。”潘青松还没有回去上班，并不知道是什么情况，也就没有发表意见。
<br/>
<br/>        “你回来了我就要回去了，我将张倩楠留下，配合你工作。”张函看着外面雾蒙蒙的天气，这几年西城的空气真的很不好，雾霾很严重，可是他的心里更是一片沉重。
<br/>
<br/>        虽然他不是单机作战，但是他感觉他肩上的担子更沉重，重得好似压得他透不过气来。
<br/>
<br/>        潘青松也看着外面雾蒙蒙的天气，嘟囔了一句：“马上要过新年了呢。”
<br/>
<br/>        “这个新年你只能在医院度过了。”张函惋惜的说。
<br/>
<br/>        张函飞回了北京，最近工作忙到处出差。他很想念俩个孩子，而且他还要回去见见他的母亲，他希望能从母亲那里知道一些事情。
<br/>
<br/>        孩子已经放寒假了，陈子昂也回到了北京，当陈子昂接到张函的电话，张函说他想见孩子。
<br/>
<br/>        陈子昂问：“你不是在西城吗？什么时候回来的？”
<br/>
<br/>        “我才下飞机。”张函边往出走边和陈子昂说话。
<br/>
<br/>        如今，能听到陈子昂的声音也很好。
<br/>
<br/>        “我还在公司，那你自己去家里接孩子吧。晚上早点送回来。”陈子昂也没有多说，就挂了电话。
<br/>
<br/>        每次张函都会以看孩子为借口给陈子昂打电话，虽然陈子昂也就是一两句话，但是张函心里都是愉悦的。
<br/>
<br/>        张函打了个车直奔陈子昂的家里，北京的冬天飘着朵朵雪花，风微微冷冽。
<br/>
<br/>        张函敲开陈子昂的家门，春姨打开门，看了一眼张函：“俩个孩子有些感冒，今天出门怕是不合适。”
<br/>
<br/>        张函站在门外，心里一沉：“我带着孩子去看医生吧。”
<br/>
<br/>        “我们已经看过医生了，就是买了些药，外面有些冷，所以......”
<br/>
<br/>        “奶奶，你在和谁说话？”房间里面传出来了嘟嘟的声音并且伴着咳嗽声。
<br/>
<br/>        “是你爸爸。”春姨对着房间里面说。
<br/>
<br/>        “爸爸来了，爸爸。”嘟嘟赤着脚丫子跑出来，直接扑进张函的，怀里。
<br/>
<br/>        “爸爸，快进来啊。”团团也跑出来了，他们可是好多天都没有看见爸爸了。
<br/>
<br/>        张函看了一眼春姨，他知道春姨是落家的人，这么多年就像是陈子昂的母亲，春姨不让他进去，他是不好进去的。
<br/>
<br/>        “奶奶，外面好冷，快点让爸爸进来。”团团给张函拿了一双拖鞋。
<br/>
<br/>        嘟嘟拉着张函进了家门。
<br/>
<br/>        春姨叹了口气，还好子昂今天不回来，子昂之前交代过，张函来看孩子是可以的，但是不准张函进家门。
<br/>
<br/>        张函是第一次到陈子昂的家里，家里干干净净，清清爽爽，客厅和餐厅都养不少的绿植，在冬天也显得春意怏然。
<br/>
<br/>        俩个孩子直接拉着爸爸在家里转了一圈，告诉张函哪个是妈妈的卧室，哪个是书房，哪个是他们的卧室。然后带着张函去了他们的儿童房。
<br/>
<br/>        父子三个人在房间里玩得不亦乐乎。
<br/>
<br/>        “爸爸，你去哪儿了？这么久都不来看我们。”嘟嘟重重的鼻音。
<br/>
<br/>        “对不起，爸爸出差了。”张函拿出纸巾替嘟嘟擦了擦鼻涕。
<br/>
<br/>        团团也眼巴巴的看着爸爸，虽然他们的个子也很高了，但是毕竟还是小学生，在自己的爸爸面前小孩子的性格就完全暴露出来了。
<br/>
<br/>        “你看看你们俩人，怎么就这么不小心都感冒了呢。”张函又给团团擦了擦鼻涕。
<br/>
<br/>        “不感冒你能进来吗？”嘟嘟小声嘟囔着。
<br/>
<br/>        “爸爸，以后你就可以光明正大的来家里了。”团团也很开心，这么多年，爸爸还是第一次到家里来。
<br/>
<br/>        要说张函和陈子昂俩个人，其实从张函和孩子见面之后，张函陪孩子的时间感觉都比陈子昂多，每次的家长会都是张函去开。
<br/>
<br/>        张函在俩个孩子的心目中的地位是不可撼动的，要不然也不会几个人上演了一场逼亲大戏。
<br/>
<br/>        陈子昂回家看到的情景就是，嘟嘟和团团依偎在张函的身旁，一大两小三只和谐的盘膝坐在地毯上笑语连连。
