　　一月后。
<br/>　　中域东面范围，衡元地域。
<br/>　　中心大城，衡元城内。
<br/>　　萧逸快步踏入猎妖殿衡元主殿，刚要前往任务处交任务，却见任务处排满了猎妖师。
<br/>　　见状，萧逸只好略微等待。
<br/>　　倒是这时，任务处，一个工作人员忽然眯了眯眼，看向萧逸这边。
<br/>　　工作人员似是想到了什么，脸色一变，竟是不顾排着长队等待着交接任务的猎妖师，快步走来。
<br/>　　“喂，你做什么。”
<br/>　　已经排到任务处前沿的几个猎妖师，不满地说了声。
<br/>　　工作人员，未理会，而是快步走向萧逸。
<br/>　　“敢问阁下，可是易霄分殿主？”工作人员来到萧逸身前，沉声问道。
<br/>　　“何事？”萧逸皱了皱眉，反问道。
<br/>　　“真的是易霄分殿主？”工作人员脸色一喜，连忙行了一礼。
<br/>　　萧逸摆摆手，示意不必多礼。
<br/>　　周遭猎妖师闻言，脸色一惊，“紫炎易霄？”
<br/>　　几乎是一瞬间，周遭所有猎妖师的目光，悉数聚焦而来。
<br/>　　衡元地域，是一个极其辽阔的大地域，猎妖师的数量极其惊人。
<br/>　　聚集在衡元主殿这里的猎妖师，也是密密麻麻，几乎将主殿之内挤了个水泄不通。
<br/>　　但此刻，却是一个个猎妖师，自动退却一步，让开一条道路。
<br/>　　“易霄分殿主可是来主殿交接任务？您先请。”
<br/>　　“对对对，易霄分殿主先请。”
<br/>　　“……”
<br/>　　原本拥挤的主殿，霎时分出一条通道。
<br/>　　原本在任务处排着长队的猎妖师，竟是纷纷退让。
<br/>　　众人看向萧逸的目光，没有半分不满，反倒充满了崇拜和惊喜。
<br/>　　“这…”萧逸见状，愣了愣。
<br/>　　人群中，一个猎妖师皱眉道，“这是何人？好大的架子。”
<br/>　　“让我等一众猎妖师给他让路？即便他是分殿主头衔又如何…”
<br/>　　这个猎妖师，话还未说完，已被周遭一道道不善的目光注视。
<br/>　　看着周遭不善的目光，这个猎妖师不禁打了个寒颤，连忙止下了话语。
<br/>　　身旁，倒是一相熟的猎妖师低声道，“笨蛋，谁让你多嘴的？”
<br/>　　“易霄分殿主也是你敢讥言的？”
<br/>　　“怎么？此人来头很大吗？”这个猎妖师一惊，“他背后有何等靠山？”
<br/>　　身旁相熟的猎妖师笑笑，“来头倒不是很大，也没有靠山。”
<br/>　　“但易霄分殿主，如今可是我们猎妖殿的大人物，我等猎妖师人人敬仰之。”
<br/>　　“怎么，你不知道？”
<br/>　　那猎妖师摇了摇头，“前些天闭关了。”
<br/>　　“啧啧。”相熟的猎妖师笑笑，“那你这些天可是错过了不少事情。”
<br/>　　“你可知道易霄分殿主如今被称作什么？”
<br/>　　不等那猎妖师发问，相熟的猎妖师已兴奋道，“邪修的噩梦。”
<br/>　　“你又可知道，易霄分殿主这些天杀了多少邪修？”
<br/>　　这一次，不等这个相熟的猎妖师说话，周遭其他猎妖师，已纷纷出言。
<br/>　　“那可是数之不尽了。”
<br/>　　“……”
<br/>　　一句句兴奋之言道出。
<br/>　　那一开始出言的猎妖师，愣了愣，满脸惊色。
<br/>　　什么时候，人人闻之色变的邪修，变得从武者口中道出如此随意，乃至轻描淡写了？
<br/>　　什么时候，让人闻风丧胆的邪修，竟是似乎变得异常狼狈？
<br/>　　他发现，自己闭关的这些天，着实错过了许多事情。
<br/>　　周遭的兴奋之言，仍旧不绝于耳。
<br/>　　“易霄分殿主初初出手时，就创下了两天时间，连毁四个邪修分部的记录。”
<br/>　　“虽然后来沉寂了十天半月，但，那不过是邪修噩梦的前夕罢了。”
<br/>　　“这个我知道。”一个猎妖师兴奋接话。
<br/>　　“之后，易霄分殿主，可是以风卷残云之势，一路横扫。”
<br/>　　“所过之处，邪修分部一一尽毁，邪修一一斩杀。”
<br/>　　“都说邪修行踪诡秘，可易霄分殿主比他们更神出鬼没，他们怎么死的都不知道。”
<br/>　　“都说邪修手段莫测，藏身险地，可却不抵易霄分殿主的滔天紫炎。”
<br/>　　“火焰焚烧千里，一个邪修也休想逃脱。”
<br/>　　“短短一月时间，横扫近百地域，各地域邪修，清剿殆尽。”
<br/>　　“……”
<br/>　　“嘶。”那一开始说话的猎妖师，此刻倒吸一口凉气。
<br/>　　“短短一月时间，横扫近百地域的邪修？”
<br/>　　萧逸听着周遭不绝于耳的话语，微微尴尬。
<br/>　　他不是爱出风头之人，便径直走向任务处，打算交接了任务便离去。
<br/>　　不错，这一月时间，他横扫了近百地域。
<br/>　　以他现今的实力，横跨一个地域，不过寥寥数个时辰足矣。
<br/>　　特别是发现了太阴太阳之眼的另一逆天之效后，以天地之力的形式感知地域，那些邪修根本无所遁形。
<br/>　　自然的，他要找这些邪修很容易。
<br/>　　一路横扫，那些邪修分部、还有在地域之内的各个邪修，被他一一斩杀。
<br/>　　他知道，邪君府，是个庞然大物。
<br/>　　以他一人之力，着实难以撼动。
<br/>　　但见一个杀一个，杀到这些邪修再不敢猖獗行事，再不敢随意冒头，倒也暂时合他意。
<br/>　　这些天，他一路往东面行走。
<br/>　　这不，此刻来到了衡元地域。
<br/>　　衡元地域，不算中域闻名的地域，但也是一流地域之一。
<br/>　　其地域异常辽阔，与四方域相比，也相差不多。
<br/>　　在中域东面范围，衡元地域处于中间部分，倒也是个有名的地域了。
<br/>　　而他特地来这里的主殿一趟，自然是交接任务的。
<br/>　　邪修中，有不少都是八殿通缉的名单。
<br/>　　猎妖殿、炎殿和药尊殿，乃是联盟殿。
<br/>　　他在此交接，可以直接把三殿的任务点和任务奖励一并拿了。
<br/>　　之后接了任务，即可离去，继续追杀邪修。
<br/>　　这时，萧逸走向任务处。
<br/>　　恰在此时，一个工作人员快步而来，道，“易霄分殿主，殿主有请。”
<br/>　　“殿主？”萧逸皱了皱眉。
<br/>　　殿主，自然是这座主殿的负责人。
<br/>　　虽然不知何事，但萧逸还是跟着这工作人员，走向殿内后堂。
<br/>　　路上，工作人员面露敬畏之色与崇拜表情。
<br/>　　“易霄分殿主声名在外，属下也没想到今日真能见到您。”
<br/>　　“你们早知我来？”萧逸语气有些疑惑。
<br/>　　“嗯。”工作人员回答道，“殿主说，按您的风格，一般完成了大量的任务，都会到沿途地域的主殿走一遭，交接任务。”
<br/>　　“我们衡元地域，是附近最大的地域。”
<br/>　　“故殿主猜测您会来此，也早便让我们记下了画像，好让我们能及时认出您。”
