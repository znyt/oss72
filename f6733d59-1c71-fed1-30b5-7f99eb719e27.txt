　　“嗯？不说话？怎么，还嫌上次揍的不够狠？我可依然还记得你被我们打的鼻青眼肿的可怜样子。”
<br/>　　曹豹冷笑着拿着匕首道。
<br/>　　“既然如此，我不介意给你脸蛋上来几刀子。”
<br/>　　“小子，见到我们豹哥还不跪下来喊爸爸。
<br/>　　“哈哈，这家伙怕是吓傻了吧。”
<br/>　　几个小弟哈哈大笑，还拿出了匕首，在那里威胁起来。
<br/>　　然而下一刻他们个个都笑不出来了。
<br/>　　甚至一个个都目瞪口呆。
<br/>　　因为林峰掏出了他的武士刀。
<br/>　　“不好意思，这个太长了，才从楼梯上转过弯。”
<br/>　　陈勾拿出了他的精钢长枪。
<br/>　　而孙畅也拿出了尼泊尔军刀。
<br/>　　“我去，豹哥，他们武器似乎比我们牛逼多了。”
<br/>　　曹豹同样震惊了，这尼玛武士刀也就罢了，这长枪是个什么鬼。
<br/>　　这何止是牛逼多了，简直不是一个级别的，他们这小匕首和人家没资格比。
<br/>　　“妈的，这群家伙哪里弄来的武器。天上掉下来的不成？”
<br/>　　曹豹眼中异色一闪而逝。
<br/>　　“小子，不错嘛，居然可以搞到这些玩意，这样吧，乖乖把这武器交出来，豹哥我可以冰释前嫌，既往不咎，顺便还让你当我小弟。”
<br/>　　林峰都要笑了。
<br/>　　“你是不是太自信了点。我当你小弟？”
<br/>　　陈勾和孙畅对手一眼，都摇头道：“听说这曹豹是什么学校扛把子，如今看来也就是个低能啊。”
<br/>　　若是之前他们绝对不敢这样说，但是现在，死在他们手中的丧尸也有几百头了，丧尸都不怕。还怕一个曹豹。
<br/>　　“你说什么？”
<br/>　　曹豹眯着眼，居然有人敢这样子和他说话。
<br/>　　“看来有些人不长记性啊。”
<br/>　　“给我废了他们！”
<br/>　　他大手一挥，几个小弟立即拿着匕首刺了过去，他们都是跟着曹豹混的问题学生，捅人根本不是多大的事情。
<br/>　　“去死吧！敢这样和豹哥说话！”
<br/>　　“妈的，拿个长枪就当自己是赵云了不成？死胖子！”
<br/>　　噗嗤！
<br/>　　然而下一刻。
<br/>　　这两个小弟人头飞起倒在了血泊之中。
<br/>　　出手的是林峰。
<br/>　　这是他第一次杀人。但没有觉得任何的不适。
<br/>　　因为人和丧尸，实际上差不多。丧尸杀的多了，杀人也就压根没有什么抵触了。
<br/>　　“你的气息……你居然是四阶的？怎么可能，你杀了多少丧尸了已经？”
<br/>　　曹豹顿时面色大变，显然没有想到之前被他堵在厕所里揍的林峰变得如此威猛。
<br/>　　“曹豹是吧，我和你往日无缘近日无仇，你却因为吴倩倩那个婊子揍我，既然今天遇到了，那么就别想走了。”
<br/>　　林峰杀气森森道。
<br/>　　“你敢动我？我爹是曹龙！江城人称翻江龙的曹龙！你应该听说过吧？你动我，你死路一条！”
<br/>　　曹豹色厉内荏道。
<br/>　　“动你又如何？”
<br/>　　林峰冷笑，举起武士刀就是一刀劈了过去。
<br/>　　曹豹立即闪躲，在武士刀面前，他根本没有还手的机会。
<br/>　　“给老子挡住他！不然老子死了，你们都别想活！”
<br/>　　曹豹大吼一声旋即掉头就跑。
<br/>　　几个小弟都一咬牙全部朝林峰杀来。
<br/>　　他们知道林峰不简单，但是曹豹的老爹见过他们，曹豹出了事，他们必死无疑，曹豹老爹有多厉害他们一万个清楚。而且在他们看来，曹豹老爹那么厉害的人肯定也还活着。
<br/>　　“找死！”
<br/>　　林峰一道斩杀过去顿时把一个人劈翻在地。
<br/>　　另外一人拿着匕首就是刺来。
<br/>　　刷。
<br/>　　下一刻，他脑门上就多出一个飞镖，直挺挺倒在地上。
<br/>　　而这时。
<br/>　　陈勾和孙畅立即加入战团，顿时，局面一边倒。
<br/>　　武器不如林峰这边，实力不如林峰这边，自然是被屠戮的结局。
<br/>　　几个小弟全都是倒在了血泊之中。
<br/>　　剩下一些学生都看傻了眼了。
<br/>　　林峰扫了他们这群天真的学生一眼，猛追了出去。
<br/>　　追风靴的速度爆发出来，顿时就看到了之前跑出去的曹豹。曹豹看到林峰追出来了面色大变。
<br/>　　压根没有想到自己小弟那么没用，死得那么快，也没想到林速度这么快。也就这愣神的功夫，曹豹就被一个旁边冲过来的丧尸扑倒在地。
<br/>　　旋即就是一群丧尸全部扑了过来。
<br/>　　“草，给老子滚！”
<br/>　　曹豹在疯狂的挣扎，他的实力也有二阶了。
<br/>　　若是站起来打，还可以杀十几个，但此时他被扑倒在地，动作受限，必死无疑。
<br/>　　显然，曹豹也知道自己要死了。
<br/>　　“林峰，救我！只要你救我！从此以后你就是我哥！我以后不会再找你麻烦！肯定！”
<br/>　　曹豹满脸恐惧的伸出手希望林峰能够施之援手，此时此刻能够救他的只有林峰。
<br/>　　“救你？曹豹，是你太天真了，还是把我当弱者了。”
<br/>　　林峰冷笑道。
<br/>　　“你……林峰，你难道真的不怕我爹吗？我告诉你，我爹他肯定活着！到时候他肯定要来找我，若是找不到我！他肯定要问！一旦知道是你做的！你必死无疑！我爹手下有好多人！这群人都是社会上的大哥！他们的手段不是你可以想象的！！”
<br/>　　曹豹大吼道。
<br/>　　吼！
<br/>　　然而下一刻一个丧尸扑了过去。

<br/>　　看到他顿时冲了过来。
