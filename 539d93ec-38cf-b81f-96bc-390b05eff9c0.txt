　　伴随着强大的药效涌入体内，小医仙玉体紧绷，银牙紧咬着粉唇，原本红润的脸颊，都在痛楚下，变得有些苍白。
<br/>　　慕清霄一直站在木桶前守候着，不时说上一句鼓励的话，当然，话语上都是他自己在占便宜。
<br/>　　痛苦足足持续了半个小时，淬体液的药力吸收的很干净，药效很不错，碧绿色的水源消失，桶内漆黑且散发出恶臭的水源取而代之。
<br/>　　七彩韵灵果是六品灵果，淬体液是五品，而小医仙不过九星斗者巅峰实力，用这些灵药简直奢侈无比。
<br/>　　若是让别的修炼者知道，不知道会多么的羡慕嫉妒恨。
<br/>　　当然，越是高等的灵药，带来的效果就越是显著，看木桶中的杂质就知道，估计改造的相当不错。
<br/>　　几分钟后，小医仙紧绷的玉体松懈下来，旋即她的气息开始提升，药效改造完肉身后就开始滋润，直接打破障壁。
<br/>　　一星斗师，一星斗师巅峰，二星斗师，二星斗师巅峰......六星斗师巅峰，丝毫没有停止的趋势。
<br/>　　慕清霄剑眉一挑，高级灵药中蕴含的力量，对于九星斗者级别的小医仙来说，终究太过庞大，以至于出现当前的状况。
<br/>　　突破虽是好事，但倘若继续突破下去，力量无法控制，根基也会受损，对她来说，百害而无一利。
<br/>　　手掌轻轻握住小医仙的手腕，恐怖且柔和的力量瞬间传入她的四肢百骸，药效转眼就被压制下去。
<br/>　　慕清霄双手结印，将这股庞大的药效压制，且封印在她的玉体深处。
<br/>　　如此一来，伴随着小医仙的修炼，药力也会完全融入她的身体，不仅没有危害，更会在需要的时候，助她轻松提升。
<br/>　　最后，小医仙的气息停止在了七星斗师初期，如此迅速的突破，若是让别的修炼者看到，恐怕连眼珠子都瞪掉在地上。
<br/>　　处理完毕，慕清霄将昏迷过去的佳人自木桶中抱了出来，最后放入另外一个，事先注满清水的木桶中。
<br/>　　强行压制邪火，替她把玉体清洗干净，擦干水珠，温柔的将她放在床榻上，盖上被子。
<br/>　　将木桶移到门口，慕清霄叫来客栈的仆役，给了几十金币的小费，让他们将木桶抬走。
<br/>　　瞬间收入几十金币，这可是顶他们一年的工资，几名仆役自然是兴高采烈的工作，看向他的目光更加亲切，就像是看自己亲爹一样。
<br/>　　当然，慕清霄可不想有他们这样的儿子：“对了，城里没发生什么事情吧？”
<br/>　　几名仆役面面相觑，眼中满是后怕，道：“公子您最近可千万别在城里走动，血宗弟子现在正发疯似得找人呢。”
<br/>　　“嗯？怎么回事？”
<br/>　　仆役神情一愣，道：“公子，您真的不知道？”
<br/>　　“我一直待在房间里，哪能知道这么多事情？你刚才说血宗弟子发疯是怎么回事？”
<br/>　　闻言，仆役觉得有道理，面前的公子哥自从来到客栈后，根本就没出过们，屋内也没啥动静，虽然怪异了点，但好在出手阔绰。
<br/>　　“公子，我们也是听说，好像血宗的少宗主死了，而且丢失了很重要的东西，血宗宗主怀疑仇人躲在黑印城中，现在正四处搜寻呢。”
<br/>　　说罢，仆役接着道：“所以小的提醒公子，最近不要出去走动，免得殃及池鱼。”
<br/>　　慕清霄微微点头，再次递过去五十枚金币，道：“给我准备一张枫城的路线与地图，多余的就赏给你们了。”
<br/>　　接过金币，仆役激动无比，道：“公子您放心，保准办妥，我们就不打扰您了哈。”
<br/>　　正所谓有钱能使鬼推磨，仆役虽是普通人，但好歹能办点事，慕清霄倒是没想到，血宗竟然如此肆无忌惮的搜寻。
<br/>　　将房门关上，慕清霄嘴角牵起一丝笑意，正所谓事不关己高高挂起，闹吧，闹的越大越好，虽然人是他杀的，但谁又知道呢？
<br/>　　设下结界，慕清霄看了一眼沉沉睡去的小医仙，盘坐在床榻上，开始修炼起来。
<br/>　　“嗯唔...”
<br/>　　傍晚时分，床榻上传来一阵舒爽的呻吟，慕清霄睁开双眼，望着坐起身来佳人，微微一笑，道：“醒了。”
<br/>　　微微颔首，小医仙低头一看，柔美至极的脸颊上，瞬间涌上一片红晕，因为此刻的她正一丝不挂，玉体全都暴露在空气中。
<br/>　　“嘤！”
<br/>　　惊叫一声，小医仙连忙扯住被子，遮住春光，绝美的脸颊上绯红如霞，红扑扑的煞是可爱。
<br/>　　片刻后，回过神来，望着自己如新生婴儿般的肌肤，美眸扑闪扑闪的，察觉到自己体内强大的力量后，更是不敢置信。
<br/>　　“斗师，七星斗师，天呐...”
<br/>　　不知何时，慕清霄已经来到床榻前，正当小医仙愣神之际，一把将她搂紧怀中，手掌在她娇躯上游走起来，坏笑一声，道：
<br/>　　“我的小美人，改造的效果不错，赶快让为夫检查检查，看看有没有什么问题。”
<br/>　　见他不要脸的模样，小医仙忍不住白了他一眼，可却乖巧的靠在他的怀中，任由他胡作非为，内心甜蜜至极。
<br/>　　在她痛苦的时候，面前的男人给了她承诺，陪在自己身边，逗她开心。
<br/>　　虽然平时好色的很，有些不要脸皮，但也只有他不断鼓励自己，安慰自己，甚至将最好的东西都留给自己。
<br/>　　经历的多了，小医仙也接触了到世界上黑暗的一面，更清楚慕清霄对她的好。
<br/>　　不管是给她的斗技，丹药，或者一掷千金拍下的七彩韵灵果，炼制的淬体液，拿到外面估计能让数之不尽的修炼者疯狂。
<br/>　　想到此处，心中的爱意已经不是一句两句能够表达，抬起臻首，美眸痴痴的望着他神骏的脸颊，莲藕般的玉臂搂住他的脖颈。
<br/>　　慕清霄微微一愣，望着她的模样，俯下身去，含住她柔软清香的粉唇，撬开唇齿，两人舌尖交缠，激烈的拥吻。
<br/>　　半刻钟后，唇齿分离，慕清霄捏了捏她小巧的鼻子，眼底闪过一丝温柔，道：“仙儿，努力提升势力吧，到时候为夫才好吃了你。”
<br/>　　小医仙紧紧依偎在他的怀中，微不可查的点了点头。
