　　紫府被灭！
<br/>　　紫府是什么？紫府可是洪荒之中除了巫、妖两族之外的第三大势力，为洪荒散修联盟，其首领是道祖钦点的男仙之首东王公。
<br/>　　这样一方势力，就这样被灭了，更是给其他势力敲了一个警钟。
<br/>　　这次大战的经过是，妖族举兵，杀向紫府联盟。事情起因就是东王公仗着自己这些年不断发展，野心膨胀，想要扩张更大的地盘，这引起了妖族的不满。妖帝帝俊与东皇太一率兵，直接打到紫府的门前，经过一场大战，紫府大败。
<br/>　　其中，东王公被帝俊亲自斩杀，西王母因为手中有道祖当初赐下的昆仑镜，能够洞察天机，穿梭虚空，借机逃走了。帝俊等人之后倒是没有再追杀她，因为紫府都被灭了，西王母一人还翻不起多大浪花。更重要的是，西王母手中有昆仑镜，这件神器中涉及到空间、时间等法则，若是一心想要逃走，谁也不能抓住她。
<br/>　　妖族也没必要为了一个不能抓住的人，而不断浪费人力。更何况，这西王母自己也很识趣，自从那次大战之后，便一人回到西昆仑，潜心静修，不再参与洪荒大事。
<br/>　　自从那次大战后，洪荒就再次陷入了诡异的平静之中，没有大战发生。谁都知道，这只是暂时的，以巫、妖二族的实力，不可能不爆发大战。
<br/>　　在这诡异的平静中，人族迎来了相对和平的爆发阶段，武者不断涌现，不过只要一到了天地法相武相境，便会被接到人祖殿中静修。那里能够遮蔽天机，至少能够示敌以弱，给人族的发展多赢取一些时间。若是其他种族知道这些，那样只会加快人族的毁灭。
<br/>　　昆仑山中。
<br/>　　元始正在与老子下棋，突然，元始对老子说道：
<br/>　　“二弟，你有一场机缘在首阳山。”
<br/>　　“首阳山吗？既然如此，那贫道便去一趟。”太清老子若有所思道。紧接着，他一步踏出，便消失于昆仑山。
<br/>　　……
<br/>　　“嗯？”
<br/>　　这一日，玄天正在武宫中的玄天殿中闭关，突如其来，他感应到一股颇为强大的力量，从东海之滨传来，令他猛然惊醒。他双眼睁开，射出一道神光，穿梭虚空，看到了东海之滨处，一道熟悉的身影。
<br/>　　“太清圣人？”
<br/>　　玄天心中一动，顿时好似想到了什么，他从怀中取出一方大印，上面有五方大帝虚影，大印之下刻着“崆峒”两个道纹大字。
<br/>　　这正是人族气运至宝崆峒印！
<br/>　　收起大印，玄天随即踏步走出玄天殿，径直往南方人祖殿走去。
<br/>　　寿山是洪荒极为广阔的一片山脉，分为东南西北四方，其中，人族圣地人祖殿以及女娲庙，就在南方，西方多峭壁，北方多荒漠，东方则是人族聚集之地，再往东，便是东海之滨了。
<br/>　　玄天踏步而来，不过瞬息，便来到人祖殿。
<br/>　　“拜见武祖！”
<br/>　　对于这位创出人族修炼之法，将人族变得强盛起来的的人祖，几乎所有人族强者，都发自内心的敬重。
<br/>　　来到人祖殿后，玄天第一件事就是让燧人氏等三祖召集人族强者。
<br/>　　不多时，燧人氏、有巢氏、缁衣氏三人便将人族太昊氏、太康氏、九头氏等部落诸多人族强者召集而来。
<br/>　　人族的祖老，不是什么人都可以担当的，这必须要对人族做出巨大贡献的人，或者大贤才有资格担任。
<br/>　　玄天也不废话，直接说明目的：“太清圣人来到人族，你们可知此事？”
<br/>　　“知道，大概百年之前，太清圣人便来到人族部落传道。”太昊氏上前说道。
<br/>　　“大哥，太清圣人此举，对于我人族可有不妥之处？”燧人氏、有巢氏、缁衣氏三人为人族初祖，事关人族未来，他们的感应变得十分敏锐，听说此事，直接问道。
<br/>　　“任何事都有两面性，人族之中已有武道，但多一项仙道，也不无不可。虽然此举会让人族失去一部分气运，但，也让人族多了一项选择。”
<br/>　　“大哥，这是否太过冒险了？”缁衣氏有些担心道。
<br/>　　“无妨，你们不必太过担忧，此事我早有准备。你们看，这是什么？”
<br/>　　玄天说着，抬手便取出一方古朴大印，无边气运流转，光华浮现，道韵自生。
<br/>　　“这是崆峒印，应运而生，可为人族气运至宝。自今日起，我将此印封藏于人祖殿之中，为人族镇压气运。此后，便不用担心人族气运大量流逝了。”
<br/>　　此宝颇为特殊，既非先天灵宝，也非后天灵宝，威力也可弱可强，强盛时，可堪比先天至宝，但弱小时，却连下品先天灵宝也不如。但它有一样固定神通，那便是镇压人族气运。
<br/>　　也只可能是人族，妖族、魔族、巫族、仙族等，都不可能，只有人族能使用它。
<br/>　　众人知道玄天自有谋算，便不再多言，他们依从玄天之言，将崆峒印供奉于人祖殿中，顿时，气运汇聚，异象频生。
<br/>　　东海之滨的太清圣人忽有所感，看向人祖殿，若有所思。他不停，继续向人族部落传道。
<br/>　　百年之后，太清圣人来到玄天殿，与玄天武祖谈论。
<br/>　　“见过太清圣人，圣人为何到我人族传道？”玄天对着太清圣人行了一礼，接着明知故问道。
<br/>　　太清圣人淡淡看了玄天一样，心中却极为惊讶玄天的修为，人族仅仅发展数千年，眼前此人竟能修炼到准圣后期，真是不可思议。当然，也仅仅只是惊讶。他随即淡淡开口道：
<br/>　　“贫道到此，是为求道而来。”
<br/>　　“所谓得道，即为失道。圣人若是如此，便会与人族结下大因果。”玄天淡淡道，他知道太清圣人看不出自己的真实身份。
<br/>　　“武祖果然是有道之人。不过，若是人人都惧怕因果，还如何成道？”太清道人先是夸赞玄天一句，接着淡淡的回答道。
<br/>　　“圣人所言甚是，不过，有些因果，是需要付出代价的。”先天淡淡道。
<br/>　　太清圣人瞧了他一样，好似知道他心中所想，良久，他淡淡道：
<br/>　　“承人族因果，保人族不灭！”
<br/>　　“如此甚好！”
<br/>　　玄天大喜，有了太清圣人这一句话，人族便不会灭亡了，以后若是人族危难之际，太清圣人也不会冷眼旁观。
<br/>　　人族的未来，至少又多了一层保障。
