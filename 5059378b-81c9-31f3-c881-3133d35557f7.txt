2015年秋，夏夕绾坐在火车上，火车从乡下开往海城。
<br/>
<br/>        九岁那一年她被丢在乡下，今天才被接回，原因只有一个，夏家要将女儿嫁到幽兰苑去冲喜。
<br/>
<br/>        听说幽兰苑里的那位新郎已经病入膏肓了，夏家有两个女儿，都不愿意嫁，所以夏家就将一直寄养在乡下的她接了回来，让她替嫁去冲喜。
<br/>
<br/>        夏夕绾坐在卧铺上，手里拿着一本书看着，这时门突然被推开，外间冷冽的寒风伴随着一股甜腥的血液味侵袭而来。
<br/>
<br/>        夏夕绾抬眸，只见一道高大英挺的身躯从外面倒了进来。
<br/>
<br/>        昏迷不醒了。
<br/>
<br/>        很快，几个黑衣人冲了进来，“老大，现在没人，直接送他下黄泉。”
<br/>
<br/>        “谁说没人的？”
<br/>
<br/>        为首的刀疤男看向了夏夕绾。
<br/>
<br/>        夏夕绾没想到意外骤热而至，这个突然倒在她车厢里的男人给她带来了致命的危险，刀疤男眼里是浓浓的杀意，很明显想杀人灭口。
<br/>
<br/>        夏夕绾不动声色的看了一眼他们手里的武器，迅速惊慌的求饶道，“不要伤害我，我什么都没有看见。”
<br/>
<br/>        刀疤男走上前，看着夏夕绾的小脸，她脸上戴着一块面纱，看不见真容，但一双翦瞳流露在外面。
<br/>
<br/>        那翦瞳无比澄亮，顾盼流转之间，竟然摇曳生姿。
<br/>
<br/>        刀疤男从来没有见过这样一双漂亮夺目的眸子，一瞬间就被摄住了心魂，再加上这些日子都没有碰过女人，当即心生了邪念。
<br/>
<br/>        “小美人，我们可以不伤害你，不过你必须把兄弟们给伺候好了。”
<br/>
<br/>        夏夕绾纤长的羽捷颤动，楚楚可怜道，“我不想死，我好害怕，只要你们不伤害我，我一定好好伺候你们。”
<br/>
<br/>        女孩软糯温存的乞求让刀疤男再也忍不住，直接扑了上去将夏夕绾给压在了身下。
<br/>
<br/>        “老大，你先来，我们把这个人送上路，然后再给兄弟们乐一乐。”
<br/>
<br/>        在充斥着低俗的笑声和温软的女人乡里，刀疤男放下了武器，伸手去扯夏夕绾的衣扣。
<br/>
<br/>        但是下一秒，一只纤白的小手握了上来。
<br/>
<br/>        刀疤男抬头，一下子就撞上了女孩那双澄亮的翦瞳，现在她的瞳仁里退去了惊慌软弱，闪烁着冷冽的碎光。
<br/>
<br/>        “你！”
<br/>
<br/>        刀疤男想开口，但是夏夕绾抬手，无比利落的将手里的一根银针刺进了刀疤男的脑袋里。
<br/>
<br/>        刀疤男两眼一闭，直接晕倒在地。
<br/>
<br/>        “老大！”
<br/>
<br/>        几个黑衣人一惊，想上前，但是这时倒在地上的男人倏然睁开了眼，探手就夺过了黑衣人手里的武器。
<br/>
<br/>        一个接一个，黑衣人全部倒在了地上。
<br/>
<br/>        简直是迅雷不及掩耳之势。
<br/>
<br/>        夏夕绾坐起身，她早就知道这男人是假装昏迷的，他身上的血是别人的。
<br/>
<br/>        夏夕绾抬眸看着男人，男人也在看着她，他有一双极其深邃的狭眸，如鹰隼般犀利，眸底还蓄着两个小深渊，任谁跟他对视一眼都会被吸下去。
<br/>
<br/>        “少爷，我们来迟了。”
<br/>
<br/>        救援的人赶到了，开始井然有序的善后，心腹手下将一个干净的帕子递给男人。
<br/>
<br/>        男人动作优雅的擦了擦手，然后迈着稳健的步伐来到了夏夕绾的面前，骨节分明的手指捏住了她小巧的下颌。
<br/>
<br/>        他眯着狭眸几分玩味的打量着她，嗓音低沉富有磁性，“你觉得我会如何处置你？”
<br/>
<br/>        下颌被他覆着薄茧的指腹捏住，夏夕绾被迫抬眸看他，男人生的颀长挺拔，俊美非凡，气场如同黑夜般强大而薄冷。
<br/>
<br/>        刚才他已经擦了手，但她还是能嗅到了那股腥甜味还有冷厉的戾气。
<br/>
<br/>        看到了不该看的，很难全身而退。
<br/>
<br/>        这男人，相当危险。
<br/>
<br/>        啪！
<br/>
<br/>        夏夕绾直接打落了男人的手，正色道，“放肆，我可是要嫁入幽兰苑的新娘！”
<br/>
<br/>        要嫁入幽兰苑的新娘？
<br/>
<br/>        男人一挑剑眉，有点意思，他的…新娘？
<br/>
<br/>        “你是海城人？那你应该知道夏家的女儿要嫁入幽兰苑，这场婚礼轰动全城，我就是那个新娘，如果我出了什么意外，你觉得你会不会遇上更大的麻烦？放了我，我什么都没有看见，什么都不会说！”
<br/>
<br/>        夏夕绾现在真的要好好感谢她的后妈李玉兰了，李玉兰接她回海城，只让她坐了廉价的火车，可是这场婚礼她办的极其奢华轰动，来博得她的好名声。
<br/>
<br/>        夏家的女儿嫁入幽兰苑冲喜，这可是海城最大的八卦新闻了，夏夕绾在赌，赌这个男人不想惹上麻烦。
<br/>
<br/>        男人饶有兴趣的看着她，今天他被生意对手买凶劫杀，遇到这个女孩是意外。
<br/>
<br/>        看她不过20岁的女孩，虽然脸色发白，衣衫凌乱，但她一双澄眸清亮而聪慧，闪烁着璀璨的光芒。
<br/>
<br/>        关键，还是他的新娘。
<br/>
<br/>        男人收回目光，带人走了。
<br/>
<br/>        夏夕绾拽紧的指尖，缓缓松开。
<br/>
<br/>        这时前方的男人幽幽的回了头，他看着她，用她可以听懂的唇语缓缓道，“我们很快就会见面的。”
<br/>
<br/>        ……
<br/>
<br/>        俪宫庄园，今天夏家的婚礼就在这里举行。
<br/>
<br/>        新娘休息室里，夏小蝶看着自己同父异母的姐姐夏夕绾，“夏夕绾，你九岁死了亲妈，后来又亲手将爷爷从楼梯上推了下来，连算命的都说你是一个灾星，于是被爸爸送去了乡下，这一次如果不是要你回来冲喜，你一辈子只能待在乡下，所以你要识时务，你可不是夏家的千金大小姐，而是夏家养的一条狗！”
<br/>
<br/>        夏夕绾坐在梳妆台前，淡淡道，“你这只狗在叫谁呢？”
<br/>
<br/>        夏小蝶叉着腰，“狗在叫你！”
<br/>
<br/>        夏夕绾勾唇，“我知道了，所以你不必再叫了。”
<br/>
<br/>        夏小蝶这才知道自己被夏夕绾给绕进去了，她看着夏夕绾澄亮的翦瞳，她回来一直戴着面纱，但露出一双眸子，这眸子光一眼就让人觉得她是一位绝丽倾城的美人。
<br/>
<br/>        夏小蝶心里嫉妒极了，恨不得将夏夕绾那双翦瞳给挖下来，这个乡下来的土包子怎么可能是美人呢，故弄玄虚，明明就是一个丑女！
<br/>
<br/>        “夕绾，吉时已到，可以出发了！”这时夏振国李玉兰带着一群贵宾进来了。
