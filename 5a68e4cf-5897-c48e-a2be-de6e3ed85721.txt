这时福伯道，“你们这些人还真是好笑，我家少奶奶的老公自然就是我家少爷了，怎么会有人问出这么愚蠢的问题！”
<br/>
<br/>        被骂愚蠢的李玉兰指尖颤抖，“你…”
<br/>
<br/>        “我不喜欢跟愚蠢的人说话，”说着福伯就没有再看李玉兰，而是恭敬道，“少奶奶，不要让这些人影响你的心情，今天是你的生辰，少爷会在帝豪酒店的顶楼为你庆生，我们现在上去吧。”
<br/>
<br/>        被福伯鄙视了彻底的李玉兰差点没气晕过去，但是很快她心头就一跳，帝豪酒店的顶楼？
<br/>
<br/>        所有人都知道帝豪酒店是海城逼格最高，最豪华的酒店，没有之一，帝豪酒店的顶楼从来没有对外开放过，传说那里伸手就可摘星辰，美的跟人间仙境一样。
<br/>
<br/>        大家都在传说里听说过，但都没有机会上去过。
<br/>
<br/>        福伯这话一落下，引起了轩然大波，
<br/>
<br/>        这位管家，你是不是吹牛不打草稿？
<br/>
<br/>        都说帝豪酒店的顶楼是人间仙境，很多人豪掷千金都没有机会上去，为什么，因为帝豪酒店是陆氏集团旗下的产业，而陆氏集团总裁陆寒霆最不缺的就是钱，他没有发放通行证，任何人都上不去。
<br/>
<br/>        你说你家少爷在帝豪酒店的顶楼给夏夕绾庆生，你当你家少爷是谁啊？
<br/>
<br/>        这位管家，你下一句话该不会是告诉我们你家少爷就是陆寒霆本尊吧哈哈哈，陆寒霆就是夏夕绾的老公哈哈哈。
<br/>
<br/>        这绝对是本年度我听到的最大的牛逼和笑话。
<br/>
<br/>        大厅里的众人笑成了一团。
<br/>
<br/>        夏振国有点不开心，他也以为这个老头是他的女婿，最后搞错了人，他最不喜欢节外生枝了，现在听福伯在吹牛，他迅速不耐烦道，“绾绾，不管你的老公是谁，你们就不要在这里闹笑话，丢人现眼了。”
<br/>
<br/>        李玉兰刚才吃了一亏，现在又露出了得意的神色，她拉住了夏妍妍的小手，“绾绾，这帝豪酒店的顶楼你肯定上不去了，我家妍妍上去还差不多，妍妍，陆总应该快来给你庆生了吧，到时你跟陆总提一下，让陆总带你上去摘星星。”
<br/>
<br/>        夏妍妍一颗心怦怦乱跳，经大家这么一说，她十分的心动，今天零点陆寒霆为了她用天灯点燃整个海城，如果夜晚陆寒霆再带她去顶楼摘星星，那这就是一段佳话了。
<br/>
<br/>        夏妍妍的心终于落定了，虽然从福伯的谈吐举止来看，夏夕绾嫁的那个老公可能也是一个有头有脸的低调人物，不过再怎么样，都不会是陆寒霆啊。
<br/>
<br/>        陆寒霆才是海城最显赫的一代商界大佬，翻手为云覆手为雨。
<br/>
<br/>        夏妍妍看向夏夕绾，“绾绾，如果我能上去的话一定会讲给你听的，免得你在这里…白日做梦。”
<br/>
<br/>        夏夕绾挑了一下柳叶眉，笑盈盈的一点都不气。
<br/>
<br/>        这时大厅的门突然被推开，外面清冽的冷风裹挟了进来，伴随着一道高大英挺的身躯，陆寒霆来了。
<br/>
<br/>        今天的陆寒霆穿着手工版的黑色西装，身形俊拔英挺，额前的刘海全部往上推，露出他精致英俊的容颜。
<br/>
<br/>        他迈着长腿步伐稳健的走了进来，举手投足皆是上位者的睥睨气场，就像是从天而降的王，让人忍不住想要膜拜。
<br/>
<br/>        陆寒霆来了！
<br/>
<br/>        夏妍妍像只花蝴蝶般快速的迎了上去，“陆总，你来啦？我一直在等你。”
