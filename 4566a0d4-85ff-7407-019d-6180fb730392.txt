　　第449章
<br/>　　林少羽绝杀的一枪，终究没有机会刺出。将军府、相府、扬州的三路大军已经到了。
<br/>　　林冲举着大旗冲到阵前，双方卷在一起，好一阵厮杀。
<br/>　　在乱军丛中，更见林少羽的残暴。骑着黑马在队伍中横冲直撞，所到之处鲜血淋漓。
<br/>　　尉迟离压阵观战，他和林少羽的父亲林问天同朝为官，双方关系甚至称得上不错。看着这一幕的他呆呆发愣，想不明白，林问天怎么培养如此一个怪物出来。
<br/>　　眼下不是杀不掉林少羽，而是杀死他要付出的代价太大。就这短短光景，已经有七八员大将被杀，横尸沙场。
<br/>　　尉迟离辛苦培养出来的人才，可经不起这样的消耗。眼见手下已经将林乐天抢了回来，尉迟离下令退兵。
<br/>　　将军府一退，其他诸侯也便退了，起义军在抛下一些尸体，留下一些敌人的尸体后，重新退回了杀虎口。
<br/>　　各路诸侯的兵马依旧守在杀虎口外，并未着急进攻。反正只要围下去，起义军被全歼是迟早的事情。比这更需要考虑的，是杀死林少羽后之后的变化。
<br/>　　今天合力对敌的各路诸侯，会不会互相乱战起来。
<br/>　　“殿下无事吧？”
<br/>　　营帐内，崔相关心的问道。
<br/>　　“喔，我没事，只是没能杀了林少羽，有些可惜。”李行哉并无受什么伤，而他本来胆子就很大，两军阵前的厮杀没能奈何他。
<br/>　　“殿下今日能够出阵，便已经是收获。”崔相道：“谁都知道，当初殿下最得明帝宠爱，这皇位本就该是殿下的。”
<br/>　　“哈，这件事再议不迟，先杀了林少羽为父皇报仇。”李行哉道。
<br/>　　“当然，当然。”崔相道：“林少羽这逆贼已经是瓮中捉鳖，在劫难逃。有些事提前考虑，总是没错的。”
<br/>　　李行哉沉默下来，崔相说得不错，不管林少羽再如何武勇，可沙场之上，终不是靠匹夫之力就能逆转形势的。他，迟早都会被杀死。
<br/>　　想到这里，李行哉难免有些唏嘘。林少羽无论如何也是一个人物，但现在却是英雄气短，穷途末路。
<br/>　　如果有可能的话，李行哉希望亲手杀了林少羽，一位报仇，二也是为了给程大雷一个交待。如果林少羽注定要死，死在自己手中，要比死在无名宵小手中好。
<br/>　　想到这里，李行哉心中忽然一动：“那个程大雷……杀死了没有？”
<br/>　　一副智珠在握的崔相表情忽地一怔，尴尬道：“还没。”
<br/>　　这边的战斗进行得如火如荼，对程大雷的追捕也从未停止，然而结果，总是有几分出乎预料。各路诸侯都派出了高手，草原上遍布杀机。有些是连程大雷的影子都摸不着，有些倒是能找到程大雷，但结果却被他杀了，还不如碰不着呢。
<br/>　　……
<br/>　　夜，杀虎口内。
<br/>　　林少羽坐在一座大石上，包扎着伤口。他左肩被长枪刺穿，一条胳膊算是废了。如今队伍中，缺医少药，也就只是草草用衣服包扎起来。
<br/>　　他面前是一条湍急的河流，而在河流对岸，同样驻扎着重兵。
<br/>　　身后传来脚步声，林少羽没有回头，便也知道是林冲过来了。
<br/>　　“少羽，你在看什么？”
<br/>　　“你看那边……是青牛山么？”林少羽用手指着远处的山峦。
<br/>　　林冲手搭凉棚，被堵在杀虎口后，他一直想着如何脱困，忙得不可开交，倒是没想过向远处看上一眼。
<br/>　　眼见山脉的起伏，林冲怔了怔：“好像就是青牛山。”
<br/>　　“是呐，我就是从那里来的。”
<br/>　　林少羽轻轻叹了一声，想起了往事。当初，他第一次离家，骑骏马，跨长枪，想着不借助家族荫蔽，白马江湖，何其快哉。
<br/>　　然而，现世狠狠给了他一巴掌，哪里有什么江湖，一个个人无非就是活着而已。
<br/>　　在青牛山遇到程大雷，稀里糊涂成了一名山贼，之后打戎族，闹长安，正义教起事，被剿灭，重新起事……然后便走到了今日，曾经翩翩少年，已阅尽沧桑。
<br/>　　他身前大河滔滔，进不得，身后却是三千残兵败将，每个人身上都有大大小小的伤口。这些人曾经都是普通的农夫、猎户、苦力，现在却一个个狠得像狼，惨得像狗。
<br/>　　“为何会如此，为何会如此？”林少羽呐呐叹道：“我……本来是想做几件好事的。”
<br/>　　无语问苍天，天空星河辽阔，却没有给他回应。林少羽想不明白，自己以正义为旗帜，今日为何却成了天下人人皆欲杀的第一恶贼。
<br/>　　“少羽，我们接下来该怎么办？”林冲问。
<br/>　　“你……怕死吗？”
<br/>　　林冲哈哈笑了一声：“怕死也就走不动今日。”
<br/>　　“那便好，你可知道，我从明帝身上，学到了一件事情。”
<br/>　　“什么？”林冲奇怪的问道。
<br/>　　“体面的死法。”林少羽长身而起，手握紧了大枪：“无论我林少羽是对是错，如何留下千古骂名，但既然已经走到今日，其他我便顾不得了，如今我只求一个体面的死法。整顿队伍，让大家吃掉最后的粮食，喝光最后的酒，明日死战，杀至最后一人，让天下人知晓，我林家少羽，不是懦夫。”
<br/>　　“大丈夫行事，原该如此。”林冲道。
<br/>　　正此时，擂鼓声响，外面传来一片喊杀声。
<br/>　　有手下急匆匆跑到林少羽面前：“报！敌人来袭营了。”
<br/>　　“慌什么，与我杀！”林冲口中大喝。
<br/>　　乌压压人马从杀虎口冲了进来，除了震天的战鼓声，还有一声声大叫。
<br/>　　“杀！一个逆贼的脑袋换一锭金，一个头目的脑袋换一个百夫长。”
<br/>　　“谁杀了逆贼首领，封田百顷，加官进爵！”
<br/>　　这绝对不是袭营，袭营用不得如此大的阵仗。这是总攻，在当天夜里，各路诸侯点起人马，携十万大军，冲进杀虎口。
<br/>　　清酒红人面，财帛动人心，在巨大的赏赐下，帝国军展现了他们从未拥有过的勇敢。很多时候，都是几个人哄抢一个起义军的脑袋。
<br/>　　林少羽深吸一口气，胸腔剧烈的起伏，口中惊雷般吐出一个字。
<br/>　　“杀！”
<br/>(本章完)
