　　五万万贯，也就是五亿贯，这是一个很庞大的数字，如果换算成现代等价值的货币，就算是换成美刀，也需要八千亿左右。
<br/>　　可是任何事情你都不可能去纵向比较，在大唐这个时代你就要按照大唐的标准来衡量一件事情。
<br/>　　比如说五万万贯吧，这个数字我们现在看着或许觉得很庞大，但是放到国家的立场上来看其实并没有多少。
<br/>　　此时的大唐已经吞并了整个东西不管是半岛还是其他，北面的国境经已经接近到北级圈附近；而南面更是延伸到了澳洲；至于西面，印度洋已经成了大唐的领海；而东面，好吧倭国还在打仗，但至少有一半也控制在大唐手中。
<br/>　　如此庞大的国土面职，五万万贯算得了什么？
<br/>　　更不要说大唐又不是单单只卖粮食，丝绸这种东西在大唐已经卖到近一贯钱一匹，出口价翻三倍那就是三贯一匹；瓷器，这个东西根本就没有什么标准，就跟后世的饭碗一样，一套五十的也有，一套五百的也有，所以一个瓶子卖五贯不贵吧？
<br/>　　不过来的可都是番邦的贵族，你好意思只买一匹丝绸？只买一个瓶子，一个碗？脸呢？
<br/>　　哦对了，大唐还有好多玻璃制品，比如小镜子，比如精美的灯饰，这东西也是大唐所独有的，一个小镜子卖两百文也不贵吧？灯饰卖五贯不贵吧？
<br/>　　来了大唐要不要买些？小镜子你好意思只买巴掌那么大的？怎么样不得买几个一人来高的？这么大的镜子，翻上几翻，收一百贯不多吧？
<br/>　　灯饰也一样，你总不能只买一盏吧？那些番邦都喜欢住城堡，买灯怎么也要把整个城堡的灯都换一遍吧？
<br/>　　什么你真的只买一个巴掌大的小镜子？灯也只买一个？成，咱也卖，不过，脸呢？
<br/>　　还有，那个推土机和拖拉机，番邦的贵族家里也有地吧？到了大唐不买一台回去？这也太让人看不起了吧？而这东西可是成对的，要买你就不能只买一样。
<br/>　　好了，你一样买一台，这么多铁加在一起不少钱吧？大唐可以限制钢铁出口的，想运出去不加点钱你好意思？收两倍的出口费，一台五十万贯，不贵吧？
<br/>　　什么？你说太贵了不买？可以啊，咱找一个差不多的，白送他一套，再给他来点钱。
<br/>　　现在有人买了吧？那么你买不卖？那些番邦都是彼此敌对的，敌人买了你不买？脸呢？
<br/>　　还有就是香水啦，照相车啦，自行车啦，前四后八豪华大马车啦等等等等，这些可都是异族番邦所没有的，来了大唐你买不买！要不要带一些回去送人！
<br/>　　不买？不带？脸呢？
<br/>　　最后，东西买完了，比赛也比完了，想要回去？买了这么多的东西你总不能自己背回去吧？大唐又不是某宝，某东，在这个物资紧俏的时代那可是卖方市场，免运费这种事情大唐是不会做的，所以……运费你总得给吧？不给？也可以，东西别拿了，留在大唐吧。
<br/>　　咱大唐不小气，还承认东西是你的，绝对不会有人动你的东西，只不过你得把保管费交了。
<br/>　　哦对，异族来了总得吃东西对吧？大唐那么多，那么多名胜，来了总要去看看吧？看了总要买点纪念品吧？这个又是一笔不小的开支吧？
<br/>　　……
<br/>　　说到这里，李承乾突然觉得五万万贯的标准似乎有些低了，看着下面目瞪口呆，几乎要把眼珠子瞪出来的一群老人渣有些不确定的问道：“你们说，我们是不是要把标准再提高一些？五万万贯好像有些少了。”
<br/>　　没人说话，所有人都沉浸在李承乾刚刚说的那些话里面。
<br/>　　如果真如同他刚刚说的那样，别说五万万贯，就是五十万万贯也不成问题。
<br/>　　可，可这还是皇帝陛下么？看李承乾蹲在龙椅上，一边挖鼻屎一边说话的样子，怎么看都是一个强买强卖的奸商好吧。
<br/>　　终于，良久的沉默之后，褚遂良这老家伙走了出来：“陛，陛下，臣，臣有句话不知当讲不当讲！”
<br/>　　“讲，有什么不能讲的，这是早朝嘛，大家议事的地方，有什么话你就说，说！”李承乾还在刚刚臆想当中，表现的有点嗨。
<br/>　　老禇抬眼看了看有些忘乎所以的李承乾，又犹豫了半天才说道：“陛下，您，您这是一个鸡蛋的家当，这，这些事情都是您一个人想出来的，这……”。
<br/>　　“怎么可能，事在人为嘛！”李承乾从龙椅上跳下来，围着桌子转了两圈然后用手戳着自己的脑袋说道：“你们要把自己的脑子动起来，好好想想！”
<br/>　　言罢，目光一扫，看到了正蹲在一边的程咬金，咳了一声说道：“就拿朕的丈人打个比方吧……”。
<br/>　　“呃，咋啦？”老程原本还在发呆，突然被叫到有些懵。
<br/>　　“丈人啊，假如，朕是说假如啊，假如你看他……”李承乾说着，伸手指了指褚遂良：“他把家里的灯全都换成电灯了，你会怎么样？”
<br/>　　“还能怎么样，老夫全给他砸了！”程咬金想都没想，大咧咧的说道。
<br/>　　一时间，大殿上静的可怕，所有人都在偷眼看着李承乾，一副想笑不敢笑的样子。
<br/>　　李承乾也觉得很是尴尬，他突然发现自己就不应该问程咬金这个魔头，这货脑回路和别人不一样，能有这样的回答一点都不奇怪。
<br/>　　而另一个比较尴尬的就是褚遂良了，这老头儿气鼓鼓的瞪着程咬金，心说特么老子怎么得罪你了，娘、、的，老子买几个电灯，你老匹夫竟然要给老夫砸了！扯蛋呢！
<br/>　　良久，房玄龄似乎想通了李承乾所说的是什么意思，走出朝班：“陛下的意思是挑拨离间？让那些互相敌对的国家看对方不顺眼？”
<br/>　　“对，就是这个意思，如果没有人买东西，我们可以白送嘛，比如说送给倭国两台拖拉机，然后对外说是倭人自己买的，你们说，其他国家会怎么想，那些贵族会不会也买几台，毕竟脸面这东西对于他们来说还是很重要的嘛。”
