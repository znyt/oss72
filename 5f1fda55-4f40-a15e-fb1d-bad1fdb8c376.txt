        给童筱颖植入监听定位器后，连华生又将她两只手臂里暗埋的抽血暗管给拆了下来，毕竟以后她不需要再被他父亲抓去抽血了。
<br/>
<br/>        “我、我真的不会伤害大哥……”童筱颖见连华生正弯腰给自己处理两只手臂上的暗管伤口，忍不住信誓旦旦地说道。
<br/>
<br/>        她昨晚只是一时间的身体不受自我支配而已，再说了，她也没向傅夜沉开枪啊！
<br/>
<br/>        连华生却白了童筱颖一眼，不以为然地回答道：“如果你是以前的那个童筱颖，我自然是会相信你不会伤害阿沉。但问题是，你不是。再说了，只是监听定位你的言行举止而已。你怕什么？”
<br/>
<br/>        “你跟那个连老头一样坏！”童筱颖皱起眉头，一脸不悦地说。
<br/>
<br/>        连华生忍俊不禁道：“那是自然，有其父就必有其子。只不过了，我向着阿沉，跟我老爸不是一边的。”
<br/>
<br/>        顿了顿后，连华生忽然间意识到了什么，神情一敛，十分严肃地看着童筱颖，接着问道：“你在我爸爸的实验室里时，他们从你身上提炼出来的‘解药’，用于的实验对象是什么？”
<br/>
<br/>        “小白鼠啊！”童筱颖不假思索地回答。
<br/>
<br/>        连华生微微皱了下眉头，一脸狐疑地反问：“真的只是小白鼠？”
<br/>
<br/>        童筱颖神色一滞，看向连华生，咧嘴一笑：“那你认为，还有什么？”
<br/>
<br/>        “我爸是不是拿活人做了实验？”连华生质问道，“那个人最后怎么呢？”
<br/>
<br/>        他偷看了他爸爸的实验日记，但是日记的记录有几篇是有上文没下文，这让他十分纳闷。
<br/>
<br/>        “……”童筱颖抿了抿唇，沉默了。
<br/>
<br/>        “小白鼠与人类的基因有百分之一的差异，‘解药’的副作用专攻了这百分之一的差异。而你和裴金澈因为是‘解药’的来源，‘解药’再返还到你们身上的时候，所造成的副作用偏小，只是让你们了出现类似于心理疾病上的第二人格。如果用在其他普通人的身上，结果又会是什么？”连华生眯起眼睛，直勾勾地盯着童筱颖。
<br/>
<br/>        童筱颖却极其不耐烦地回答道：“我又不是学医的，我怎么知道结果是什么。”
<br/>
<br/>        “我实验研究表明，‘解药’确实能治好患病的小白鼠，但是小白鼠被治愈后，会出现了狂躁不安的动向。如果是活人呢？会不会比这种后果更夸张？”连华生慢条斯理地说。
<br/>
<br/>        童筱颖本能地转开话题，问道：“大哥他人呢？今天一早起来就没见到他。”
<br/>
<br/>        连华生瞥了童筱颖一眼，替她包扎好伤口后，见她有意隐瞒，便不再继续刚刚的话题，而是一边收拾着医药箱，一边云淡风轻地回答道：“他去哪儿跟你一点关系也没有，从今天开始，你最好学会‘安分守己’与‘老实听话’。不要再做出昨晚那么危险的行为来，不然，就算阿沉不杀你。我也会……”
<br/>
<br/>        “喀”地一声，连华生抬起手来抹了一把自己的脖子，接着说道，“把你给杀了。”
<br/>
<br/>        童筱颖被他吓得身子哆嗦了一下，没好气地说道：“活该你单身。”
<br/>
<br/>        “诶？你这女人，这话什么意思？”连华生顿时吹刘海瞪眼睛。
<br/>
<br/>        童筱颖微微扬起下巴，嘚瑟起来：“你这么喜欢威胁女人，是没有女人会喜欢你的。”
<br/>
<br/>        “呵呵，你眼光真不好。本少有的是女人喜欢！”连华生合上医药箱，对童筱颖翻了个白眼，懒得再跟这女人废话。
<br/>
<br/>        连华生也离开后，这私宅里一下子就安静了。
<br/>
<br/>        其实，她童筱颖也不知道那个连老头有没有拿活人做实验。
<br/>
<br/>        只是，裴姨帮助她离开实验室的时候，裴姨要她发过誓，不对任何人提及有关于实验室里的任何事情。
<br/>
<br/>        童筱颖觉得待在私宅里好无聊，于是跟刘婶打了声招呼，叫司机过来送她出了门。
<br/>
<br/>        或许是因为跟傅夜沉赌气，又或许是因为她发现自己离开了傅夜沉之后确实不知道该何去何从，她便在外游荡了一天。
<br/>
<br/>        一边逛街一边吃吃喝喝，直到晚上凌晨了，她都没有想要回私宅去的意向。
<br/>
<br/>        童筱颖独自一个人坐在公园路灯下的长椅上，寒风凛冽，冻得她双手抱臂直哆嗦。
<br/>
<br/>        她真不知道自己此刻在想什么，就像一个跟丈夫吵架离家出走的女人一样，心里迷茫，很没安全感。
<br/>
<br/>        直到地面上，一双锃亮的皮鞋出现在她视野里是，她才怔怔地抬起头来。
<br/>
<br/>        傅夜沉颀长的身躯，恍若与这灰黑的夜色融为了一体。
<br/>
<br/>        他优雅地脱下自己身上的灰色风衣，轻轻地披在童筱颖的身上，语气不冷不热地问：“为什么不回家？”
