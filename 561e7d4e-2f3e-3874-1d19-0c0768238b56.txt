        或许是两个人相互折磨了这么久，他慢慢地摸清了她的脾气品行。所以，不会再像以前那样一冲动，就会间接地伤害到她。
<br/>
<br/>        果然，彼此不了解彼此的两个人被强行绑在一起后，需要花更多的时间去磨合。
<br/>
<br/>        只是，他心里的苦衷，她为什么就不能体谅？
<br/>
<br/>        她一定要逼他，气他，明明他在努力学着如何去好好地爱护她，结果总是适得其反。
<br/>
<br/>        傅夜沉到此刻都不知道自己到底做错了什么，他没谈过恋爱，不懂女人的心思，可是，他是真心实意地爱着她童筱颖这个女人。
<br/>
<br/>        两人对视了许久……
<br/>
<br/>        童筱颖因为害怕而微微颤抖的身子渐渐地平静了下来。
<br/>
<br/>        她原本迷离的眸子，辗转间有了一丝异样的微光。
<br/>
<br/>        傅夜沉还未明白她的心境突然为何有所转变之时，她那温柔的唇瓣，便主动贴了上来。
<br/>
<br/>        她两只软得像是没有骨头的手，轻轻地勾住他的脖颈，贴身亲吻了他一会儿。
<br/>
<br/>        而他却没有任何反应。
<br/>
<br/>        筱颖突然投怀送抱，十有八九是心里在盘算着什么事情。
<br/>
<br/>        她到底在盘算着什么，他因为揣测不出她的想法，内心而感到无比慌乱。
<br/>
<br/>        “大哥，我想给你生个孩子，以后，你不在我身边的时候，有孩子陪着我。”童筱颖的声音突然间变得格外柔情似水，但是她的眼睛里却依旧扑朔迷离。
<br/>
<br/>        傅夜沉回过神来，看着童筱颖这张娇俏的脸蛋，心里明知道她另有所图，却仍旧对她的话动了恻隐之心。
<br/>
<br/>        “筱颖，你这是想通了吗？”
<br/>
<br/>        想通了，他是为了她好？
<br/>
<br/>        理解了，他不得已的苦衷？
<br/>
<br/>        童筱颖的双手继续环绕在他的脖颈上，她不回答他任何问题，扬起头来，再次用力吻上了他的薄唇，身体也跟着再次往上将他贴得很紧。
<br/>
<br/>        傅夜沉忽然被她弄得有些狼狈，双手钳制她单薄的肩头，这才能使劲把她扯离，将她软弱的身体向下按回床上。
<br/>
<br/>        他心情十分复杂地质问：“筱颖，你葫芦里到底卖的是什么药？”
<br/>
<br/>        “大哥是不愿意吗？大哥不是一直想要我给你生个孩子吗？我现在在做的事情，就是在自己的肚子里种下大哥的孩子。”童筱颖慢条斯理地回答，语气里没有任何意思其余的感情夹杂其中。
<br/>
<br/>        女人的心，海底的针。
<br/>
<br/>        傅夜沉翻了个身，从床上下来，淡淡地说道：“或许，你真的需要一个人静一静。今晚，我去睡书房。”
<br/>
<br/>        他说完，人也离开了房间。
<br/>
<br/>        童筱颖失魂落魄地从床上坐了起来。
<br/>
<br/>        是不是她的心思被他看穿呢？
<br/>
<br/>        如果今晚成了，她就一定会怀上他的孩子，因为这两天不是她的生理安全期。
<br/>
<br/>        然后，她打算拿孩子来让他妥协，把季雅珊给放了。
<br/>
<br/>        看样子……
<br/>
<br/>        这招是行不通了。
<br/>
<br/>        童筱颖双目呆滞地坐在大床上，心里却在祈祷，希望季雅珊不要为了十爷做傻事。
<br/>
<br/>        书房的书柜后，有一间偌大的密室，里面灯光昏暗，却依稀可见置物架上一排又一排的武器和以及各种类似于特工的装备。
<br/>
<br/>        “你疯了！我们的人去救季雅珊，就等于我们是在叛变！”连华生站在光线偏暗的阴影里，疾言厉色地呵斥。

<br/>        傅夜沉换上了一身黑色的紧身束衣，将匕首绑在了小腿上，淡定道：“她是筱颖的朋友，我不能让她死在组织的严刑逼供之下。”
<br/>
<br/>        “季雅珊的事，是她自找的！更何况，你曾经救过她一命。是她自己不明是非，非得跟暗道上冒险者集团的人扯上关系！”连华生试图劝阻。
<br/>
<br/>        傅夜沉接着戴上腕表：“上头那边的人已经审讯了她一天一夜了，不是什么也没问出来吗？再对她用刑审讯下去，她会死。”
<br/>

<br/>
<br/>        傅夜沉戴完腕表，又一边绑着腰身上的麻醉弹药，一边不以为然地回答道：“我没打算拖兄弟们下水，我一个人去就行了。”

<br/>        “你要夜闯组织的暗牢？”连华生惊愕。
<br/>
<br/>        傅夜沉点了点头：“替我照顾好筱颖。”
<br/>
<br/>        “你如果去救季雅珊，上头一定会认为你叛变了，你到时候会是什么下场，不用我说，你也清楚！”连华生伸手阻拦。

<br/>        傅夜沉淡然地笑了笑：“我这一生一直都在为别人卖命，却从未为自己心爱的女人卖过命。这次，不管是对是错，我都要去做。只为了筱颖。”
