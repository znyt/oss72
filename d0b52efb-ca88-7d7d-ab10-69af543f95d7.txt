        十字路口上红绿灯即将转变，抢黄灯的车辆快速行驶，车轮快速压过一个坑洼，高高溅起污水，哗啦一声，无情溅到了童筱颖的身上。
<br/>
<br/>        雨势，非但没有减缓的意思，且愈下愈大。
<br/>
<br/>        童筱颖一想到这段感情全是她一厢情愿，还曾幻想过小卫到头来会真的回到她身边，便觉得自己是多么的可悲又可笑。
<br/>
<br/>        她慢慢蹲下来，抱着膝盖，像个孩子放声大哭，哭出所有委屈与伤心。
<br/>
<br/>        她怎么会错得如此离谱？
<br/>
<br/>        她为自己的愚蠢，为惨遭践踏的自尊，为不被珍惜的真心真意哭得声嘶力竭，难以控制。
<br/>
<br/>        行经的路人来来往往，被她的痛哭失声吸引，缓下脚步，好奇张望。
<br/>
<br/>        童筱颖感受到众人好奇的视线，狼狈的她想躲藏，却怎么也提不起力气，站起来佯装潇洒离开。
<br/>
<br/>        可不可以……
<br/>
<br/>        可不可以不要这样瞧着她？
<br/>
<br/>        可不可以……
<br/>
<br/>        可不可以不要同情她？
<br/>
<br/>        可不可以……
<br/>
<br/>        可不可以不要理她？
<br/>
<br/>        假如此刻她能够隐形，让所有人都看不见凄惨伤心的她该有多好。
<br/>
<br/>        童筱颖呜咽痛哭，心，空荡荡，已失去一部分的自己。
<br/>
<br/>        突地，雨不再兜头朝她落下，不再冰冷亲吻她的发心、脸庞，好奇的视线似乎被庞大的身影遮掩。
<br/>
<br/>        泪水依然不断自眼角滚落，湿淋淋的童筱颖冷到骨子里，全身不住发抖。
<br/>
<br/>        她抬起头，先入眼帘的是皮质良好的黑色意大利手工皮鞋，慢慢向上是剪裁合身的黑色西装裤，主人的双腿微张，再往上望，发现西装外套包里下的精壮胸膛，接着来到方正的下巴，高挺好看的鼻子，然后，一双黑亮深邃的眼眸映入，她愕然看着身前好看、充满男性魅力的男人。
<br/>
<br/>        傅夜沉将伞撑在她上方，为狼狈不堪的她挡雨，而他自己大半身躯则淋着雨，自己却毫不在意。
<br/>
<br/>        这是回傅氏集团大厦的必经之路，如果不是刚刚前方十字路口亮起了红灯，或许，他都发现不了，这个令他朝思暮想的女人，远在天边近在眼前。
<br/>
<br/>        他为她弃车而来，却在看到她时，心疼得一句话也说不上来。
<br/>
<br/>        童筱颖蹲在地上，泪眼迷蒙地看着上方神情错综复杂的男人，抽抽噎噎地唤了声：“大哥。”
<br/>
<br/>        “大哥？筱颖？”与此同时，异口同声的人，还有不知何时出现在婚纱店门口的傅昼景。
<br/>
<br/>        大手紧握成拳，手背上的青筋凸起，傅夜沉将自己的雨伞轻轻地搭在了童筱颖的身上后，挡住了她的视野的下一秒，转身干脆利落又霸道地给了傅昼景一拳。
<br/>
<br/>        傅昼景一个踉跄，后退了一步，整个身子，直接“哐”的一声，撞到了背后的玻璃门上。
<br/>
<br/>        童筱颖从地上站了起来，看到傅昼景嘴角的淤青还流下一绺血丝，她的眼泪再次溃堤。
<br/>
<br/>        豆粒般大的泪珠潸潸滚落，头发因雨水湿黏在颊畔，她盯着傅昼景看了许久，直到她看到他的背后，季思妍牵着小尚轩走到玻璃门口时，她才伤心欲绝地转身，落荒而逃。
<br/>
<br/>        “筱颖……”傅昼景欲要去追，季思妍见状，慌忙拽住了他的手臂。

<br/>
<br/>        傅昼景拧着眉头，只能眼睁睁地看着自己的大哥傅夜沉，冒着大雨去追他的筱颖。

<br/>        有大哥在的话，筱颖应该不会有事……
<br/>
<br/>        傅昼景黯然神伤地垂下眼帘。
<br/>
<br/>        他和筱颖，还能回去吗？

<br/>        明明他计划好了和季思妍假结婚，安抚好季思妍的心情后，就回澳洲去找筱颖。

<br/>        慌乱之中，一只强劲有力的大手将她给拉回了路边。
