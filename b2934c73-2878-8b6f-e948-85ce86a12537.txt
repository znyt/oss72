
<br/>　　<br/>　　    官方出面，意外死亡的三个工人和受伤的伤者，很快就处理完事了，至于怎么处理这方面官方肯定很有手段和方式，三天过后，办公室主任轻车简从，只带了司机去往省城，通过朋友的关系他见到了一位据说是来自龙虎山的道士。<br/>　　<br/>　　    办公室主任和龙虎山道士是在一间茶馆里见的面，一见到对方许主任就“嚯”了一声，心道好一幅仙风道骨啊。<br/>　　<br/>　　    这位道爷，四平八稳的坐在一张椅子上，面红齿白，鹤发童颜，一缕长须飘在胸前，穿着身道袍显得特别飘逸，身后站着个小道童煽着扑扇，吹的他下巴上的白胡子随风而起，显得特别有意境。<br/>　　<br/>　　    人靠衣装马靠鞍，总的来说印象还是得看气质才行，这位道爷给许主任的印象就是打了个九十分左右，心里已经笃定的认为，此人肯定乃是世外高人了。<br/>　　<br/>　　    “贫道凌虚子，现为龙虎山正一堂首座……”凌虚子一见许主任就侃侃而谈，三言两语就把自己的简介给抖了出来。<br/>　　<br/>　　    许主任连忙点头说道：“早就听说过您的大名了，这一次前来，还是因为有事想要和您聊聊”<br/>　　<br/>　　    “但说无妨”<br/>　　<br/>　　    许主任随后就把破庙拆迁的事讲了一遍，凌虚子听闻时一直都没有插话，捋着长须脸上一片淡然，直到许主任话音落下，他才伸出白嫩的右手，来回的掐印了几遍，嘴里轻吐出了一声“咦？”。<br/>　　<br/>　　    许主任吓了一跳，心道这还真是有什么猫腻不成？就连忙询问着：“您这是算出点什么来了？”<br/>　　<br/>　　    许主任为啥对这位凌虚子如此相信呢，首先是介绍的那位朋友言之凿凿的说，这位龙虎山的高人可是不少省城大人物的座上客，那平时出入的可都是侯门啊，往来无白丁，多少富商富豪想请人家都没有档期，谁要是能把这位高人请去指点一番的话，不管啥事都是水到渠成的。<br/>　　<br/>　　    现在，在一看对方的那一副仙风道骨，许主任就更确信不疑了，凌虚子肯定就是传说中的那种世外高人了。<br/>　　<br/>　　    灵虚子皱眉说道：“俗话说宁拆十座庙不毁一桩婚，这都是凡夫俗子的说法，山庙多有灵，那是神仙和佛陀的居所，你把仙家住的地方都要给拆了，那不是惹得人心里有怨？不妥，不妥，再一个是我刚才也掐算了一番，你们最近在庙旁施工，坏了本地的地气，以后就算是把园区盖起来恐怕也会颇有磨难，比如招商，运营都会步步坎坷啊”<br/>　　<br/>　　    许主任顿时急了：“那不成啊，这庙建在开发区里，必须得拆啊，不然经济开发区里四周都是厂房，公司，出现个庙肯定不好看啊，这不是不伦不类了么？”<br/>　　<br/>　　    “要是这样的话，我这里有上中下三策，许先生可供考虑一番”<br/>　　<br/>　　    “您说……”<br/>　　<br/>　　    “上策么，就是将这庙重新翻修一番，给仙家重新塑个金身，以此来压住拆庙的怨气”<br/>　　<br/>　　    许主任想了想，摇头说道：“这可不行，我还是那个意思，开发区里不能有庙宇，再一个规划图纸都做好了，这庙要是在修起来的话，那园区里的整个规划都得重新做了，这根本行不通，而且以后肯定会有上级部门领导来开发区检查，到时看见园区中有这庙一座庙，那得要给我扣上个封建迷信的帽子了，这肯定不行”<br/>　　<br/>　　    “再说中策，就是我开坛讲道，于庙中先人讲述一番，将此庙迁往他处，不过这中策也有麻烦，一座山头一座庙，这个安置的地方肯定很难找，估计要费一些时间了”<br/>　　<br/>　　    许主任叹了口气，说道：“就是时间不够，工程紧啊，我这才着急来找道长商谈呢”<br/>　　<br/>　　    凌虚子眉头忽然一挑，脸上露出一抹意味深长的笑意，放在桌子下的右手忍不住得意的敲了几下大腿，表面却云淡风轻的说道：“那就只能用下策了，就是我亲自前去，施法将庙中的怨气压下……”<br/>　　<br/>　　    许主任顿时连连点头，说道：“能压得下自然是好了，省时省力，我们也急啊”<br/>　　<br/>　　    凌虚子又习惯性的挑了下眉头，沉吟着伸手在桌子上挺有频率的敲打着，左手拉着长须淡淡的笑了笑，也没说话。<br/>　　<br/>　　    许主任也是个八面玲珑的人，当即就知道这是啥意思了，于是从口袋里掏出个信封放在了桌子上，说道：“道长这是车马费用，等事成之后解决了问题，我在另外重谢您”<br/>　　<br/>　　    凌虚子眼睛都没往桌子上瞅一眼，很平淡的说道：“我这是代替龙虎山真人所收，稍后还要在布施出去，我们出家为道之人对金钱向来都当成是身外之物的，追求大道者岂能为这等闲事所累？”<br/>　　<br/>　　    “道长说的是，我这主要是不想让您太过操累么”<br/>　　<br/>　　    “好吧，我先收着，稍后寻个地方再捐出去就是了，过后会为许主任画个名”凌虚子伸手就从桌子上把信封拿了起来，一摸之间感觉着厚度心里是相当满意了，他随手交给后面的道童，就说道：“明日我们师徒二人准备一番，这就过去开坛做法”<br/>　　<br/>　　    许主任走了后，凌虚子和道童就忙不迭的把信封给拆了，里面的钱不少，正好整三万。<br/>　　<br/>　　    道童感慨的说道：“这钱来的真容易，师傅啊你上下嘴皮一碰就到手了几万块”<br/>　　<br/>　　    “这才算是开胃，我稍后继续忽悠忽悠，钱还得再翻一翻”凌虚子眼睛放光的说道。<br/>　　<br/>　　    道童顿时诧异的说道：“师傅，你不是说省城不能呆了，再呆下去就露馅了，以前您出手的几次期限可要到了，趁着他们没回过神来前，咱不得换个地方重整旗鼓么？”<br/>　　<br/>　　    “这地方人傻钱多，我还真舍不得走啊，借着龙虎山的名头好办事，换个地方可就没这么容易了，唉”凌虚子长吁短叹的摇了摇头，咬牙说道：“最后一把，明天你和我过去，装模作样的给他们上一课，钱收了之后再走也不迟，到嘴了的鸭子飞了太可惜！”
