　　伏息树蟒与这支猎妖队的战斗，还在持续。
<br/>　　不过，战况早已改变。
<br/>　　这支猎妖队，完全落入了下方，几无还手之力。
<br/>　　一个圣王境七重强者带队，清一色圣王境武者队员，这样一支实力强劲，且经验丰富的猎妖队。
<br/>　　竟在一头伏息树蟒上吃了大亏？
<br/>　　此事若传出去，怕是会让人难以置信。
<br/>　　毕竟，伏息树蟒，顶天了也就圣王境六重左右的实力。
<br/>　　“大家快跑。”
<br/>　　这时，为首的队长忽然急切大喝一声。
<br/>　　“你们先走，我拦下这孽畜。”
<br/>　　“不行。”一位年轻队员愧声道，“若非我一意孤行，非要寻实力强大的妖兽猎杀，岂会让大家陷入这般境地。”
<br/>　　“队长，我来挡，你们走。”
<br/>　　“哈哈哈哈。”为首的队长豪爽一笑。
<br/>　　手中，一把大刀，重重劈出一道巨大刀气。
<br/>　　巨大的刀气，轰在伏息树蟒上，生生逼退了伏息树蟒一段距离。
<br/>　　“金光险地的妖兽，异于别处。”
<br/>　　“即便是经验再丰富的猎妖队，也只敢找比自己实力弱得多的妖兽去猎杀。”
<br/>　　“你们还年轻，此次便当买个教训。”
<br/>　　“队长。”一众队员，面露愧色。
<br/>　　为首的队长，横刀而上，一己之力，强行挡下了伏息树蟒。
<br/>　　“快走。”为首的队长大喝一声。
<br/>　　“我顶多能挡这孽畜十息时间，你们速速退离。”
<br/>　　“之后我会尽快追上你们。”
<br/>　　“是。”一众队员，应答一声，连连退离。
<br/>　　接下来的战斗，属于这位圣王境七重武者，以及伏息树蟒。
<br/>　　只不过，战况比之前更加糟糕。
<br/>　　在伏息树蟒面前，这位圣王境七重武者，几乎是被碾压之势。
<br/>　　观这头伏息树蟒的实力，此刻显然达到圣王境八重以上。
<br/>　　十息之后。
<br/>　　这位猎妖队队长，已然身上带伤，满脸狼狈。
<br/>　　刚要转身逃遁，伏息树蟒巨大的身躯却已环绕四周，封锁了去路。
<br/>　　巨大的蟒身，以极快的速度收拢。
<br/>　　观这般情况，这位猎妖队队长绝对来不及逃出封锁，便会被在巨大蟒身的收缩缠绕下，被缠绕而死。
<br/>　　“不好。”
<br/>　　嗖…
<br/>　　恰在此时，一道剑气凌空而过。
<br/>　　剑气的速度，快到了极点，一闪而过。
<br/>　　待得剑气消散，伏息树蟒那巨大的头颅，已被洞穿，就此生死。
<br/>　　庞大的身躯，重重地跌落地面，扬起大片灰尘。
<br/>　　蟒身的中央，这位猎妖队队长脸色煞白。
<br/>　　刚才，他只来得及惊呼一声，便已知自己再无机会逃出。
<br/>　　不曾想，这股念头才刚出现，巨大的伏息树蟒，已然身死。
<br/>　　“不知是哪位前辈出手相救？”猎妖队长反应过来，对着空气拱了拱手。
<br/>　　然而，空气中，并无回答。
<br/>　　下一秒，一旁不远处，那棵参天大树下，萧逸缓缓起身。
<br/>　　直到他有这般明显动作时，那猎妖队长才发现了过来。
<br/>　　“嗯？前辈…不对，是位小友。”猎妖队长这般说着，还是重重行了一礼。
<br/>　　萧逸脸色淡然，指尖一弹，一粒疗伤丹药弹出。
<br/>　　猎妖队长接过，脸色一惊，“王品巅峰疗伤丹药？”
<br/>　　说罢，猎妖队长连忙手掌摊出，“谢小友好意。”
<br/>　　“不过，在下已欠小友一份救命之恩，再赠这珍贵丹药，在下可没脸收下。”
<br/>　　“服下吧。”萧逸淡淡道，“你是队长，既带他们入了这金光险地，便有责任带他们安全离去。”
<br/>　　“你如今带伤，实力稍逊全盛几分，如何有把握保护他们？”
<br/>　　队长二字，绝不仅仅是一队之长，或者全队实力最强者那么简单。
<br/>　　“这…”猎妖队长闻言，迟疑了一下。
<br/>　　下一秒，点了点头，收下了丹药。
<br/>　　“谢小友馈赠，日后，若有机会，我许三刀必倾囊回报。”
<br/>　　“若有所需，在下绝不推辞半分。”
<br/>　　萧逸摇摇头，淡淡道，“不必，小事一桩罢了。”
<br/>　　他自己也是猎妖师，也认可这份职业。
<br/>　　在外行走，遇到了，能帮便帮罢了。
<br/>　　猎妖队长看了眼萧逸淡然的脸色，再次行了一礼，道，“在下还得追上我的同伴们，便不叨扰小友了。”
<br/>　　“去吧。”萧逸点点头。
<br/>　　“告辞。”猎妖队长闪身而离。
<br/>　　那头伏息树蟒的内丹，他未拿。
<br/>　　毕竟不是他杀的妖兽，且他也没能力杀，自然不会拿那内丹。
<br/>　　猎妖队长离去。
<br/>　　原地，只剩萧逸一人。
<br/>　　以及那头巨大的伏息树蟒尸体。
<br/>　　萧逸并未急着收取妖兽精血与妖兽内丹，而是细细感知起来。
<br/>　　据他所知，伏息树蟒，不过是圣王境六重左右的妖兽。
<br/>　　可现在，在他的感知中，这头伏息树蟒的气息，绝对达到圣王境八重以上，甚至接近圣王境九重。
<br/>　　要知道，一般来说，妖兽的寿命很长，但作为代价，它们的实力增长，会异常缓慢。
<br/>　　故，这般忽然实力大增的情况，对妖兽而言，是几乎不可能之事。
<br/>　　萧逸细细感知着。
<br/>　　半晌，眉头一皱。
<br/>　　这头伏息树蟒，气息与实力飙升的源头，在于体内那颗妖兽内丹。
<br/>　　萧逸指尖一道剑气打出，破开了伏息树蟒的尸体。
<br/>　　一颗散发着金光的妖兽内丹，缓缓飘荡而出。
<br/>　　妖兽的一身实力，来源于肉体、血液，以及内丹。
<br/>　　肉体自不必多说，妖兽本就以肉体强大著称。
<br/>　　血液，则是精血之所在。
<br/>　　内丹，则是蕴含一身妖元的地方。
<br/>　　在萧逸的感知中，这头伏息树蟒的肉体、血液，都是圣王境六重层次。
<br/>　　唯独这颗内丹，达到了圣王境八重巅峰层次。
<br/>　　那金光熠熠的光芒，便是内丹层次上升的原因。
<br/>　　那些金光，自然便是如今飘荡在这金光险地内的一缕缕金色光芒。
<br/>　　“看来，传言是真的。”
<br/>　　萧逸自语一声。
<br/>　　这片俩阔无比的金光险地，虽是险地，却也是一块宝地。
<br/>　　萧逸看了眼飘荡四周的金光，再次感知了一番。
<br/>　　周遭，金光氤氲，让得这片险地的夜晚，散发着柔和而美丽的光芒。
<br/>　　只是，这些金光，显然非凡物。
<br/>　　萧逸伸出手，随手捉了一把飘荡着的金光。
<br/>　　然而，手臂触碰到金光，却仿佛拍在了空气中，一穿而过。
