　　蛇王庙的附近，那是一片斗蛇的现场，人也分外的多，气氛热火朝天，高潮迭起。
<br/>　　一年一度的白镇斗蛇大会开启了。
<br/>　　此时，一大群人围着场中央两条毒蛇，在给他们加油助威。
<br/>　　外围，一位清秀的年轻男子一脸的苦笑，他发现自己根本挤不进去，这里三层外三次的人，简直寸步难进。
<br/>　　“看个斗蛇也那么难”涂小安也是无语的，怪自己来的太晚，没有占到一个好位置。
<br/>　　斗蛇还有一个名字，叫赛蛇神，要是你手中的蛇，能在比赛的过程中拿到第一名，那么这条蛇的身价会暴涨，将会有很多人出高价要买你这条蛇。
<br/>　　只是想拿到白镇蛇王节斗蛇大会的第一名，那可不是简单的事情，不知道有多少捕蛇人暗地早开始准备这一天了。
<br/>　　他们手中的毒蛇都是经过特殊的训练，你要是随随便便拿一条蛇出来跟他们的斗，一个呼吸间，就能被咬死。
<br/>　　白镇曾经有一条“蛇王”赢得了三年的斗蛇大会的冠军，后来被一位土豪，用五百万的天价觅得过去。
<br/>　　这比珍稀蛇种的价格还要高，而那条“蛇王”品种并不高，罕见蛇种都算不上，就是被捕蛇人煞费苦心的培养，培养成了一个蛇中杀手。
<br/>　　一条蛇是不是厉害，也不是光靠着品种来衡量，毕竟都是凡蛇，要是经过专业捕蛇人的培养，那将不同凡响。
<br/>　　会非常的凶煞暴戾，见血封喉。
<br/>　　“也不知道今年那位捕蛇人的毒蛇能够拔得头筹，冠绝全场”
<br/>　　涂小安已经知道了，今年的斗蛇大会第一名奖励可是一百万，是一位白镇的土豪赞助的。
<br/>　　想斗蛇，必须提前几天报名，今天想参加都晚了。
<br/>　　否则涂小安看在一百万的份上，还真的有点意动。
<br/>　　挤不进去，涂小安只能垫着脚尖凑热闹，也不能动用飞行术，人太多，被发现了，就不好收场。
<br/>　　忽然...他似有所感，低头一看手臂，既然有一只白色的蚊子落在上面，那长长的口器，对准皮肤，就刺了进去。
<br/>　　咦...他旋即微微诧异。
<br/>　　这只蚊子既然是白色的，而且看样子比一般的蚊子要大上很多，足足体型一公分。
<br/>　　仅仅一眼，涂小安就断定这蚊子不一般，要知道一般我们所看到的蚊子，都是漆黑色的，白色的蚊子还真是很少见。
<br/>　　甚至压根没见过。
<br/>　　莫非这也是罕见品种的蚊子？
<br/>　　人多的地方，蚊子也会喜欢来凑热闹，但是这只蚊子涂小安一个洞察丢过去，大吃一惊，等级既然为五级。
<br/>　　在早期，涂小安也利用系统收服了很多的蚊子，可一般的蚊子连等级都不存在。
<br/>　　这只蚊子等级既然是五级，有点逆天了，简直就是违背常理的。
<br/>　　要知道一条普通的成年毒蛇，系统的等级划分也才五级。
<br/>　　难道这只蚊子的生命力能比得上一条成年的毒蛇吗。
<br/>　　白蚊口器刺入涂小安的皮肤，忽然，它小小样子微微一愣，好像有点茫然若失，紧接着一道戏谑的声音出现。
<br/>　　“是不是没吸到血”
<br/>　　白蚊一颤抖，瞬息间拔出口器，就骤然起飞。
<br/>　　可一只白皙的手划过，直接将这只白蚊用两指掐在指尖上。
<br/>　　“不好意思，我这具身体是虚假的，没有血让你吸”
<br/>　　涂小安将这只白蚊掐住，淡漠通透的看着，越看，越发现这只蚊子不一般。
<br/>　　小小蚊子挣扎起来，既然还有不少力气，这种力气能让涂小安明显的感受到。
<br/>　　他微微一捏，既然不爆，这蚊子的身体好像得到过特殊强化，非同小可。
<br/>　　白镇怎么会突然出现这种蚊子？
<br/>　　这要是被叮咬吸血，恐怕要被吸走一大管血液吧。
<br/>　　涂小安目露深思之色，忽然...人群中传来一阵骚动，有人开始大喊起来：“不好了，有人突然晕倒了...”
<br/>　　嗯？
<br/>　　涂小安闻言，有种不好的预感产生，手指不禁的微微用力，这只白色的蚊子被捏爆了。
<br/>　　半晌后，街道不少地方都传出有人晕倒的消息，大范围的骚动起来，集体陷入了一阵混乱恐慌之中。
<br/>　　“怎么回事，莫非是有人在本蛇王的地盘捣乱？”
<br/>　　事出反常必有妖。
<br/>　　涂小安顾不上多想，连忙朝着有人晕倒的地方跑去。
<br/>　　临近涂小安最近的一个地方，是一位中年妇女晕倒在地，脸色极度苍白，呼吸很是微弱。
<br/>　　一大帮人围着，都有点束手无策，吵吵嚷嚷的，非常的乱。
<br/>　　“都让开点，我是医生，让我看看”
<br/>　　涂小安只能先给自己找一个借口，去查看晕倒之人的状况。
<br/>　　不少人听到涂小安说自己是医生，也没多想的去怀疑，连忙闪开。
<br/>　　涂小安立马蹲下检查，这妇女的气息很脆弱，好像是长期的缺血所致。
<br/>　　如果没有碰到刚才那只白蚊，涂小安也是一脸懵比，但他现在检查妇女身上有没有什么伤口。
<br/>　　很快，他就看到了妇女的脖子上有一大快红肿，伤口明显溃烂了开来。
<br/>　　这伤口一看，跟毒蛇咬的症状是极为的相似。
<br/>　　“她是不是不小心被人群中的毒蛇给咬到了”
<br/>　　有人看到伤口，立刻发问。
<br/>　　涂小安当机笃定：“这不是毒蛇咬的”
<br/>　　“怎么可能不是毒蛇咬的，伤口溃烂，气息薄弱，分明是中毒了”
<br/>　　“这满街的人与蛇，人挤人的，不小心被谁身上的毒蛇咬到，也不是不可能”
<br/>　　说话之人，并不是白镇的当地人。
<br/>　　“她是中毒，但中的不是蛇毒”
<br/>　　涂小安脸色阴郁了下来，这是不是蛇毒，他不用眼睛去看，光闻都能闻出来。
<br/>　　缺血，伤口红肿，在加溃烂，莫非是那种白色的蚊子所为。
<br/>　　涂小安的脑海出现这种想法，要说蚊子叮咬不可能那么的严重。
<br/>　　但那种白色的蚊子跟普通的蚊子完全不一样，光等级都五级了。
<br/>　　岂能用常理看待。
<br/>　　只是此时有人询问，涂小安也不可能告诉大家，这中的是蚊子的毒，那岂不是贻笑大方。
<br/>　　鬼会相信呀。
<br/>　　最主要他自己现在也不是很确定。
<br/>　　“这里也有人晕倒了”
<br/>　　“这边有三个人倒地不起”
<br/>　　不过区区几分钟的时间里，各地都传出尖叫声，不断有人倒地不起，现象严重化。
